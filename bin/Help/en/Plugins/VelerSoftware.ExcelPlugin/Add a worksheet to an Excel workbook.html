Action : Add a worksheet to an Excel workbook

This action adds a new worksheet to an Excel workbook who has been opened or created previously.

Important: To use this action, it is imperative that the software Microsoft Excel (2007 or 2010 or higher) is installed on the computer of the person using your project.

Parameters

Parameters Descriptions
Variable with the concerned Excel workbook Must be equal to the variable (not array) that has the Excel workbook you want to add a leaf. This variable must be previously assigned by the action Create a new Excel workbook or Open an Excel workbook.
Worksheet name to add Must be equal to the name of the new worksheet that will be added to the workbook.

Mistake not to commit : For parameter"Worksheet name to add", you can not use the resources of the project type and Picture type Other. Single resource of type text can be used.

Mistake not to commit : If you use variables in parameter "Worksheet name to add", make sure that the type of value of each of these variables is text (String), or a numeric value (Integer, Decimal, Double). Indeed, only text can be used in these settings.
Note : To check the type of a value, use the action Get the type of a variable, in General category.

Returned value

No value is returned by this action
