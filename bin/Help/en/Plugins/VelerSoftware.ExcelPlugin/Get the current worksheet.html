Action : Get the current worksheet

This action retrieves the name of the sheet, with which the program currently work, of an Excel workbook who has been opened or created previously.

Important: To use this action, it is imperative that the software Microsoft Excel (2007 or 2010 or higher) is installed on the computer of the person using your project.

Parameters

Parameters Descriptions
Variable with the concerned Excel workbook Must be equal to the variable (not array) that has the Excel workbook that you want to retrieve the current sheet. This variable must be previously assigned by the action Create a new Excel workbook or Open an Excel workbook.
Variable receving the current worksheet's name Must be equal to the variable (not array) to be equal to the worksheet's name with which the program currently work.

Returned value

The return value is determined by the name of the worksheet on which your program work at the time of execution of this action. This value is of type Text (System.String).
