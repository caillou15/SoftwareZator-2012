Action : Create a new Excel workbook

This action will initialize a new Excel workbook and assign it to a variable.

Important: To use this action, it is imperative that the software Microsoft Excel (2007 or 2010 or higher) is installed on the computer of the person using your project.

Warning: when this action runs, a new instance of the software Microsoft Excel installed on the user's computer is started, even if the parameter "Display the Microsoft Excel software on the screen" is checked, or unchecked.
Before changing the entire value of the variable used in parameter "Variable receving the new Excel workbook", thanks to use the action Close an Excel workbook to close automatically the software Microsoft Excel when your software does not need it anymore. You will need to use this action before this variable is no longer usable with actions to Excel without having to re-use this action.

Parameters

Parameters Descriptions
Variable receving the new Excel workbook Must be equal to the variable (not array) which will equal the new Excel workbook.
Display the Microsoft Excel software on the screen Determines whether the software's window Microsoft Excel will be displayed on the screen or hidden. This parameter does not change the launch of Excel during execution of this action.

Returned value

No value is returned by this action
