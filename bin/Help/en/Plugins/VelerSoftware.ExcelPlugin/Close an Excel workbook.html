Action : Close an Excel workbook

This action closes an Excel workbook stored in a variable and close the instance of the software Microsoft Excel runs to this workbook, without saving changes made. The Excel window associated with this variable will be automatically closed.

Important: To use this action, it is imperative that the software Microsoft Excel (2007 or 2010 or higher) is installed on the computer of the person using your project.

Tip: To save changes before closing an Excel workbook, use the action Save an Excel workbook.

Parameters

Parameters Descriptions
Variable with the Excel workbook to close Must be equal to the variable (not array) that has the Excel workbook that you want close. This variable must be previously assigned by the action Create a new Excel workbook or Open an Excel workbook.

Returned value

No value is returned by this action
