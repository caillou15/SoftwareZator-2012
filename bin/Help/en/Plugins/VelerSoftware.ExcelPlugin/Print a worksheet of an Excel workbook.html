Action : Print a worksheet of an Excel workbook

This action will print the current sheet or all sheets in an Excel workbook has been opened or created previously.

Important: To use this action, it is imperative that the software Microsoft Excel (2007 or 2010 or higher) is installed on the computer of the person using your project.

Parameters

Parameters Descriptions
Variable with the concerned Excel workbook Must be equal to the variable (not array) that has the Excel workbook that you want to print a worksheet or all of its. This variable must be previously assigned by the action Create a new Excel workbook or Open an Excel workbook.
Print all worksheets Determines whether all sheets in the workbook should be printed or if only the current sheet (set by the action Set the current worksheet) should be printed.

Returned value

No value is returned by this action

Code Visual Basic.Net generated by SoftwareZator

If the parameter "Print all worksheets" is unchecked, the generated code will be as follows :

Votre_Variable.PrintWorksheet(1)

If the parameter "Print all worksheets" is checked, the generated code will be as follows :

Votre_Variable.PrintAllWorksheets(1)
