Action : Adding a text to a printing

This action helps to put text into a page of a print stream previously created with the action New printing.

Parameters

Parameters Descriptions
Variable with the stream to print Must be equal to a variable (not array) whose value is a print stream (the stream has been created and obtained by the action New printing).
Text to add Allows you to enter text to add to the printing sheet.
Text color Sets the color the text will take. This color can be stored in a variable or sets in the parameter. Make sure the value you define either a Color (System.Drawing.Color)
Page number on which must it add the text Must be equal to the number of the page flow to which the text will be added.
Set the X position on the page Must be equal to the horizontal position of the text on the page.
Set the Y position on the page Must be equal to the vertical position of the text on the page.

Mistake not to commit : For the parameters "Text to add", "Page number on which must it add the text", "Set the X position on the page" and "Set the Y position on the page", you can not use the resources of the project type and Picture type Other. Single resource of type text can be used.

Mistake not to commit : If you use variables in parameter "Text to add", "Page number on which must it add the text", "Set the X position on the page" or and "Set the Y position on the page", make sure that the type of value of each of these variables is text (String), or a numeric value (Integer, Decimal, Double). Indeed, only text can be used in these settings.
Note : To check the type of a value, use the action Get the type of a variable, in General category.

Return value

No value is returned by this action
