Action : New printing

This action initializes a new print document, that is to say, create a new printable document. Then use the other actions of the category "Printing" to configure the document and then print it.

Parameters

Parameters Descriptions
Variable used to store and work with the print stream Must be equal to a variable (not array) to be used by other actions of the category "Printing" to configure the printable document (margins, header, content ...).
Document name (optional) Enter the name of the document to print.
Orientation This parameter sets the print orientation of the document: Portrait or Landscape. Auto mode automatically determines, based on document content, whether it should be printed in portrait or landscape.

Mistake not to commit : For the parameter "Nom du document", you can not use the resources of the project type and Picture type Other. Single resource of type text can be used.

Mistake not to commit : If you use variables in parameter "Nom du document", make sure that the type of value of each of these variables is text (String), or a numeric value (Integer, Decimal, Double). Indeed, only text can be used in these settings.
Note : To check the type of a value, use the action Get the type of a variable, in General category.

Return value

No value is returned by this action

Code Visual Basic.Net generated by SoftwareZator

If the parameter "Orientation" is equal to "Landscape", the generated code will be as follows :

Your_Variable = New VelerSoftware.PrintLib.PrintSystem.PrintClass(VelerSoftware.PrintLib.PrintSystem.Orientation.Landscape, "Document name", True)
