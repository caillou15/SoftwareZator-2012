Action : Print

This action allows you to print a print stream previously created with the action New printing.

Parameters

Parameters Descriptions
Variable with the stream to print Must be equal to a variable (not array) whose value is a print stream (the stream has been created and obtained by the action New printing).
Display the print settings Determines if a dialog box to configure a printer and print settings to be displayed.
Do a print preview Determines if a dialog box allowing you to preview before printing the document to be opened.
Variable receiving the result of printing (optional) If necessary, select the variable (not array) receiving the print result.

Return value

The return value is determined by the success or otherwise of the document is sent to the printer. This value does not know if printing was done correctly, but only whether there has been no problem before the start of the printing process (which is not supported by your software but by the computer). It will be stored in the variable defined in the parameter "Variable receiving the result of printing". This value is True/False (Boolean).

Result Returned values
No problems were encountered True
An unknown problem has been encountered False

Code Visual Basic.Net generated by SoftwareZator

If the parameter "Show Window print settings" is checked and the parameter "Do a print preview" is unchecked, the generated code will be as follows :

Your_Variable = Your_Stream.Print(True, False)

If the parameter "Show Window print settings" is unchecked and the parameter "Do a print preview" is checked, the generated code will be as follows :

Your_Variable = Your_Stream.Print(False, True)

Runtime errors

Sometimes a computer program encounters a problem during execution. Often, an error message is displayed. The table below allows you to determine why the error you encounter appears and how to correct it :

Errors types Descriptions Resolve
System.MissingMemberException This error occurs when the action New printing has not been executed, the variable defined in the parameter "Variable with the stream to print", before this action, or that the value of the variable has changed in the meantime. To correct this error, make sure the action New printing is executed for the same variable as that specified in the parameter "Variable with the stream to print" and the value thereof has not been changed or reset before the execution of the action "Print".
