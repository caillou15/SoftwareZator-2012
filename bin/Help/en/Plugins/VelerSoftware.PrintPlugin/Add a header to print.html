Action : Add a header to print

This action defines a new part of the header or footer of a print stream previously created with the action New printing.

Note: this action has an impact on all pages of printable document, not only a page.

Parameters

Parameters Descriptions
Variable with the stream to print Must be equal to a variable (not array) whose value is a print stream (the stream has been created and obtained by the action New printing).
Text of the header Enter the text to be displayed in the header.
Note: regularly do on a printable document, there have page numbers. With two tags (usable only in this setting), you can give the page numbers :
  • &NumPage : equals the number of the current page.
  • &PageCount : equals the number total pages in the document to print.
Example : you are on page &NumPage on &PageCount pages.
Horizontal alignement Sets whether the header will be displayed on the left, right or middle of the page.
Vertical alignement Sets whether the header will be displayed at the top (header itself) or base (footer) of the page.
Show the header on the first page Determines whether the header should be displayed on the front page of printable document or not.

Mistake not to commit : For the parameter "Text of the header", you can not use the resources of the project type and Picture type Other. Single resource of type text can be used.

Mistake not to commit : If you use variables in parameter "Text of the header", make sure that the type of value of each of these variables is text (String), or a numeric value (Integer, Decimal, Double). Indeed, only text can be used in these settings.
Note : To check the type of a value, use the action Get the type of a variable, in General category.

Return value

No value is returned by this action
