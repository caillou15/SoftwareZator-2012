Action : Set the margins of a printed sheet

This action will allow you to set the margins to print content on the sheet of the printer, a print stream previously created with the action New printing.

Parameters

Parameters Descriptions
Variable with the stream to print Must be equal to a variable (not array) whose value is a print stream (the stream has been created and obtained by the action New printing).
Margins Sets the mouse what are the leaf margins. Just move and resize the box in the middle of the representation of the blank page and change the margins. These are displayed at the bottom of the window, in centimeters.

Return value

No value is returned by this action

Code Visual Basic.Net generated by SoftwareZator

If margins are the following: 1cm left, 2 right, the top 3, 4 to the bottom; the generated code is as follows :

Your_Stream.setMargin(3, 1, 4, 2)
