Action : Get the current directory on an FTP server

This action retrieves the folder defined as the current working directory on a server like FTP (File Transfer Protocol) which your software was connected with the action Connect to an FTP server.

Parameters

Parameters Descriptions
Variable receiving the result Must be equal to the variable (not array) receiving the path and folder name.

Returned value

The return value is determined by the path and name of the current directory on the server. It will be stored in the variable defined in the parameter "Variable receiving the result". This value is text (String).

Result Returned values
Recovered folder Text equal the path and name of the current folder
Not recovered -1
Note: An error message will appear on the screen.

Code Visual Basic.Net generated by SoftwareZator

If the variable receiving the result is "Result", the generated code will be as follows :

Result = VelerSoftware_FTPPlugin.GetCurrentDirectory()

Runtimes errors

No runtime error is detected here.
