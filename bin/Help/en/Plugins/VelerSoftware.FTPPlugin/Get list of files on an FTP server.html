Action : Get list of files on an FTP server

This action retrieves a list of files in the specified location on a server like FTP (File Transfer Protocol) which your software was connected with the action Connect to an FTP server.

Parameters

Parameters Descriptions
Full path to the folder for which to retrieve the file list Must equal the full path to the folder that must retrieve the list of files.
Variable receiving the result Must be equal to the array variable receiving the list of files.

Mistake not to commit : For parameter "Full path to the folder for which to retrieve the file list", you can not use the resources of the project type and Picture type Other. Single resource of type text can be used.

Mistake not to commit : If you use variables in parameter "Full path to the folder for which to retrieve the file list", make sure that the type of value of each of these variables is text (String), or a numeric value (Integer, Decimal, Double). Indeed, only text can be used in these settings.
Note : To check the type of a value, use the action Get the type of a variable, in General category.

Returned value

The return value is determined by the contents of the specified directory on the server. It will be stored in the variable defined in the parameter "Variable receiving the result". This value is a list of values ​​of type "VelerSoftware.FTPLib.FtpFileInfo".

Code Visual Basic.Net generated by SoftwareZator

If the variable receiving the result is "Result", the generated code will be as follows :

Result = VelerSoftware_FTPPlugin.GetFiles("/path")
