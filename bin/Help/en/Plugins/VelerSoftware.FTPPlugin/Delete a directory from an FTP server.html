Action : Delete a directory from an FTP server

This action deletes a folder on a server like FTP (File Transfer Protocol) which your software was connected with the action Connect to an FTP server.

Parameters

Parameters Descriptions
Full path to the folder to delete Must equal the full path to the folder that will be deleted on the server.
Variable receiving the result (optional) Must be equal to the variable (not array) receiving the result of the suppression.

Mistake not to commit : For parameter "Full path to the folder to delete", you can not use the resources of the project type and Picture type Other. Single resource of type text can be used.

Mistake not to commit : If you use variables in parameter "Full path to the folder to delete", make sure that the type of value of each of these variables is text (String), or a numeric value (Integer, Decimal, Double). Indeed, only text can be used in these settings.
Note : To check the type of a value, use the action Get the type of a variable, in General category.

Returned value

The return value is determined by the success or failure of deleting the folder on the server. It will be stored in the variable defined in the parameter "Variable receiving the result". This value is numeric (Integer).

Result Returned values
Folder deleted 1
Not deleted 0
Note: An error message will appear on the screen.

Code Visual Basic.Net generated by SoftwareZator

If the variable receiving the result is "Result", the generated code will be as follows :

Result = VelerSoftware_FTPPlugin.RemoveDirectory("/path/folder")
