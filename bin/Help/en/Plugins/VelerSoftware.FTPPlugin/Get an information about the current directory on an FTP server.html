Action : Get an information about the current directory on an FTP server

This action retrieves the file information defined as the current working directory, such as name, creation date, attributes and another on a server like FTP (File Transfer Protocol) which your software was connected with the action Connect to an FTP server.

Parameters

Parameters Descriptions
Variable receiving the result Must be equal to the variable (not array) receiving the information on file.

Returned value

The return value is determined by the current folder on the server. It will be stored in the variable defined in the parameter "Variable receiving the result". This value is of type "VelerSoftware.FTPLib.FtpDirectoryInfo".

Information : if no current directory is set or if it could not be recovered, the value returned will be zero.

Code Visual Basic.Net generated by SoftwareZator

If the variable receiving the result is "Result", the generated code will be as follows :

Result = VelerSoftware_FTPPlugin.GetCurrentDirectoryInfo()

Runtimes errors

No runtime error is detected here.
