Action : Send a request to an FTP server

This action will send a command like "FTP Protocol" to run on a server like FTP (File Transfer Protocol) which your software was connected with the action Connect to an FTP server.

Parameters

Parameters Descriptions
Request to run Must be equal to the query to be executed by the server.
Variable receiving the result (optional) Must be equal to the variable (not array) receiving the query result.

Mistake not to commit : For parameter "Request to run", you can not use the resources of the project type and Picture type Other. Single resource of type text can be used.

Mistake not to commit : If you use variables in parameter "Request to run", make sure that the type of value of each of these variables is text (String), or a numeric value (Integer, Decimal, Double). Indeed, only text can be used in these settings.
Note : To check the type of a value, use the action Get the type of a variable, in General category.

Returned value

The return value is determined by the server response depending on the request sent and the server type and configuration. It will be stored in the variable defined in the parameter "Variable receiving the result". This value is text (String).

Result Returned values
Query executed successfully {value unspecified}
Error -1
Note: An error message will appear on the screen.

Code Visual Basic.Net generated by SoftwareZator

If the variable receiving the result is "Result", the generated code will be as follows :

Result = VelerSoftware_FTPPlugin.SendCommand("QUIT")
