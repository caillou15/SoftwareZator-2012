Action : Create a Access database

This action creates a new Microsoft Access database before connecting to it..

Compability

Microsoft Access databases are available only for software running in 32 bit. In addition, your software must be configured so that it is used in 32 bit only.

Information: software compiled for a 32 bit operating system can run on an operating system 64 bit. The difference is that this software will not enjoy the performance offered by the system.

Note so that in compiling your software for a 32-bit, it will run on a Windows 64 bit, so, the use of Microsoft Access databases can be used with your software on a Windows 32-bit and 64 bit, as it is compiled for a targeted operating system 32 bit.

To change the type of target operating system so that your software supports Microsoft Access on any Windows operating system (XP, Vista, 7, 8), go to "Project settings", and tab "Build", and finally set the "Type of target operating system" setting on "Windows 32-bit only".

Parameters

Parameters Descriptions
Full path to the file database to be created (*.Mdb) Must be equal to the full path to Access database to be created.
Password (optional) Allows you to enter a password that will be needed to connect to the database.
Note: This password can not be changed.
Variable receiving the result (optional) Must be equal to the variable (not array) receiving the result of creation.

Mistake not to commit : For the parameter "Full path to the file database to be created" and "Password", "Name of the cell (column) to modify", you can not use the resources of the project type and Picture type Other. Single resource of type text can be used.

Mistake not to commit : If you use variables in parameter "Password", "Id of the concerned line" and "Name of the cell (column) to modify", make sure that the type of value of each of these variables is text (String), or a numeric value (Integer, Decimal, Double). Indeed, only text can be used in these settings.
Note : To check the type of a value, use the action Get the type of a variable, in General category.

Return value

The return value is determined by the success or otherwise of creating the database. It will be stored in the variable defined in the parameter "Variable receiving the result". This value is numeric (Integer).

Result Returned values
Success 1
The database file already exists 0
Fail -1
Note: An error message appears on the screen

Code Visual Basic.Net generated by SoftwareZator

If the variable receiving the result is "Result", the generated code will be as follows :

Result = VelerSoftware_AccessPlugin.CreateNewDataBase_Access("C:\Your file.mdb", "Password")
