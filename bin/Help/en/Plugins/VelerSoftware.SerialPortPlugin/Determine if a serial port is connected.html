Action : Determine if a serial port is connected

This action determines whether a control of type serial port is connected to a device (printers, mobile phones ...). For more information, see the What is a serial port? document.

Warning : Make sure that the window is where your serial port component is shown (on the screen) when this action is run.

Parameters

Parameters Descriptions
Serial port concerned Must be equal to the control of type serial port (in the window designer) that should determine the connection status.
Variable receiving the result Must be equal to the variable (not array) to be equal to the value that determines whether the serial port is connected or not.

Returned value

The return value is determined by the status of the serial port connection to the device. It will be saved in the variable defined in the parameter "Variable receiving the result". This value is True/False (Boolean).

Result Returned values
The serial port is connected to the device True
The serial port is disconnected False

Code Visual Basic.Net generated by SoftwareZator

If the serial port is "MyPort", the generated code will be as follows :

Variable = Your_Windows.MyPort.IsOpen

Runtime errors

No runtime error is detected here.
