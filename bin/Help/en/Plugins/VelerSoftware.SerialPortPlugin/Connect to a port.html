Action : Connect to a port

This action opens a connection to a serial port that is connected to a device (printers, mobile phones ...). For more information, see the What is a serial port? document.

Warning : Make sure that the window is where your serial port component is shown (on the screen) when this action is run.

Warning : Before this action is executed, it is necessary that the property "SerialName" of the control of type serial port (in the window designer) have been assigned with the name of the port to be opened.

Parameters

Parameters Descriptions
Serial port concerned Must be equal to the control of type serial port (in the window designer) which must connect to a device.

Returned value

No value is returned by this action

Code Visual Basic.Net generated by SoftwareZator

If the serial port is "MyPort", the generated code will be as follows :

Your_Window.MyPort.Open()

Runtime errors

Sometimes a computer program encounters a problem during execution. Often, an error message is displayed. The table below allows you to determine why the error you encounter appears and how to correct :

Errors types Descriptions Resolve
InvalidOperationException This error occurs when the control of type serial port is already connected to a device. To correct this error, you must ensure that the port is closed with the action Disconnect from a port before to execution of this action. To determine if a serial port is connected to a device, use the action Determine if a serial port is connected.
ArgumentOutOfRangeException This error occurs when one or more control properties of type serial port specified is not valid. For example, properties "Parity", "DataBits" or "handshake" do not have valid values​​; "BaudRate" is less than or equal to zero; property "ReadTimeout" or "WriteTimeout" is less than zero and has no value "InfiniteTimeout". To correct this error, modify the property values ​​of the serial port.
ArgumentException This error occurs when the port name that needs to connect does not starts with "COM". To correct this error, you must ensure that the port name starts with "COM" and try to list the available ports with the action Get list of ports.
UnauthorizedAccessException This error occurs when the device or the computer denies access to the port. To correct this error, make sure no other software or use this port and restart the computer.
