Action : Get list of ports

This action retrieves the names of serial ports which are connected devices (printers, mobile phones ...). For more information, see the What is a serial port? document.

Parameters

Parameters Descriptions
Variable receiving the list of serial ports connected Must be equal to the array variable receiving the names of serial ports which devices are connected.

Returned value

The return value is determined by devices connected to the computer. It will be saved in the variable defined in the parameter "Variable receiving the list of serial ports connected". This value is a list of text values (System.String).
