Action : Send a text data

This action allows to send a text data to a serial port that is connected to a device (printers, mobile phones ...) with the action Connect to a port. For more information, see the What is a serial port? document.

Warning : Make sure that the window is where your serial port component is shown (on the screen) when this action is run.

Parameters

Parameters Descriptions
Serial port concerned Must be equal to the control of type serial port (in the window designer) must send a data to the device it is connected.
Text data to send Allows you to enter the text or commands to send to the device connected to the computer.

Returned value

No value is returned by this action

Code Visual Basic.Net generated by SoftwareZator

If the serial port is "MyPort", the generated code will be as follows :

Your_Windows.MyPort.Write("A data")

Runtime errors

Sometimes a computer program encounters a problem during execution. Often, an error message is displayed. The table below allows you to determine why the error you encounter appears and how to correct :

Errors types Descriptions Resolve
InvalidOperationException This error occurs when the control of type serial port is not connected to a device. To correct this error, you must ensure that the port is opened with the action Connect to a port before running this action. To Determine if a serial port is connected to a device, use the action Determine if a serial port is connected.
ArgumentNullException This error occurs when the value of the property "Text data to send" is null. To correct this error, you must enter a value in the parameter "Text data to send" of this action.
TimeoutException This error occurs when the operation was not completed before the end of the waiting period (defined by property "WriteTimeout" of the control of type serial port (in the window designer) . To correct this error, you must ensure that the waiting period is long enough (often supperior to 200 milliseconds, or infinity (equal to -1).
System.InvalidCastException This error occurs when the parameter value "Text datat to send" is not text. Often the problem is caused by a variable used in this setting and has a value other than text. To correct this error, check that all the values ​​involved in this setting are of type Text (String) and make the necessary changes so that these values ​​is text. For this, use the action Get the type of a variable, Convert the value of a variable
