Action : Get the list of MySQL databases

This action will retrieve the list of databases on a MySQL.

Parameters

Parameters Descriptions
Variable receiving the result Must be equal to the array variable receiving the list of databases.

Return value

The return value is determined by the databases installed on the server. It will be stored in the variable defined in the parameter "Variable receiving the result". This value is a list of text values (System.String).

Code Visual Basic.Net generated by SoftwareZator

If the variable receiving the result is "Result", the generated code will be as follows :

Result = VelerSoftware_MySQLPlugin.GetDatabases_MySQL()

Runtime errors

No runtime error is detected here.
