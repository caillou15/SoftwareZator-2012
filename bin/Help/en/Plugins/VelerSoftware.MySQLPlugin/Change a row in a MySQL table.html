Action : Change a row in a MySQL table

This action can change a value of a row in a table in a database on a MySQL which your software will be previously connected with the action Connect to MySQL.

Parameters

Parameters Descriptions
Table's name concerned Must be equal to the name of the table in which a line will be changed.
Id of the concerned line Must be equal to the value of the cell "Id" of the line will be changed.
Name of the cell (column) to modify Must be equal to the corresponding column name in the cell to edit (see illustration below).
New value To enter the new value of the cell in the table.
Variable receiving the result (optional) Must be equal to the variable (not array) receiving the result.

Mistake not to commit : For the parameter "Table's name concerned" and "Id of the concerned line", "Name of the cell (column) to modify", you can not use the resources of the project type and Picture type Other. Single resource of type text can be used.

Mistake not to commit : If you use variables in parameter "Table's name concerned", "Id of the concerned line" and "Name of the cell (column) to modify", make sure that the type of value of each of these variables is text (String), or a numeric value (Integer, Decimal, Double). Indeed, only text can be used in these settings.
Note : To check the type of a value, use the action Get the type of a variable, in General category.

Return value

The return value is determined by the success or otherwise of the renaming. It will be stored in the variable defined in the parameter "Variable receiving the result". This value is numeric (Integer).

Result Returned values
Success 1
Fail -1
Note: An error message appears on the screen

Code Visual Basic.Net generated by SoftwareZator

If the variable receiving the result is "Result", the generated code will be as follows :

Result = VelerSoftware_MySQLPlugin.ChangeRowTable_MySQL("Table's name", 1, "Col1", "New value")
