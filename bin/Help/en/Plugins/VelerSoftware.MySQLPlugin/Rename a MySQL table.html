Action : Rename a MySQL table

This action allows you to change the name of a table on a MySQL which your software will be previously connected with the action Connect to MySQL.

Parameters

Parameters Descriptions
Name of the table to rename Must be equal to the name of the table to be renamed.
New name of the table Must be equal to the new name it will give the table.
Variable receiving the result (optional) Must be equal to the variable (not array) receiving the result.

Mistake not to commit : For the parameter "Name of the table to rename" and "New name of the table", you can not use the resources of the project type and Picture type Other. Single resource of type text can be used.

Mistake not to commit : If you use variables in parameter "Name of the table to rename" and "New name of the table", make sure that the type of value of each of these variables is text (String), or a numeric value (Integer, Decimal, Double). Indeed, only text can be used in these settings.
Note : To check the type of a value, use the action Get the type of a variable, in General category.

Return value

The return value is determined by the success or otherwise of the renaming. It will be stored in the variable defined in the parameter "Variable receiving the result". This value is numeric (Integer).

Result Returned values
Success 1
Failed -1
Note: An error message appears on the screen

Code Visual Basic.Net generated by SoftwareZator

If the variable receiving the result is "Result", the generated code will be as follows :

Resul = VelerSoftware_MySQLPlugin.RenameTable_MySQL("Old name", "New name")
