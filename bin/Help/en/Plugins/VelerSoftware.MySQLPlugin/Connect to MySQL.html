Action : Connect to MySQL

This action allows your software to connect to a MySQL in local (on the computer from which your software is installed) or remotely (on another computer, internet).

Parameters

Parameters Descriptions
Local MySQL By checking this setting, your software will connect to a server installed locally (on the computer from which you installed your software).
Remote MySQL By checking this setting, your software will connect to a server installed on a remote computer. The connection will be by internet.
Adress of MySQL Must be equal to the local address, or IP address of the server that your software will need to login.
Port Must be equal to the port to use for connecting to the server.
User name (optional) Must be equal to the username to use if an identifier is required to connect to the server.
Password (optional) Must be equal to the password to use if an identifier is required to connect to the server.
Name of the initial database (optional) Captures the database, by default, with which the server will work when connected, so you do not then use action Change MySQL database.
Encrypt the connection Sets whether the server connection will be secured by an encryption protocol or not.
Variable receiving the result (optional) Must be equal to the variable (not array) receiving the result of the connection.

Mistake not to commit : For the parameters "Adress of MySQL", "Port", "User name", "Password" and "Name of the initial database", you can not use the resources of the project type and Picture type Other. Single resource of type text can be used.

Mistake not to commit : If you use variables in parameters "Adress of MySQL", "Port", "User name", "Password" and "Name of the initial database", make sure that the type of value of each of these variables is text (String), or a numeric value (Integer, Decimal, Double). Indeed, only text can be used in these settings.
Note : To check the type of a value, use the action Get the type of a variable, in General category.

Return value

The return value is determined by the success or failure of the server connection. It will be stored in the variable defined in the parameter "Variable receiving the result". This value is numeric (Integer).

Result of the connection Returned values
Connected 1
Server not found -1
Incorrect username or password -2
Unknow error -3
Note: An error message will appear on the screen.

Code Visual Basic.Net generated by SoftwareZator

If the server is remote, that the parameter "Encrypt the connection" is checked and the variable receiving the result is "Result", the generated code will be as follows :

Result = VelerSoftware_MySQLPlugin.Connect_MySQL(False, "xxx.xxx.xxx.xxx", "1433", "User", "Password", "Initial database", "yes")

Si le serveur est locale et que le paramètre "Encrypt the connection" est décoché, le code généré sera tel que le suivant :

VelerSoftware_MySQLPlugin.Connect_MySQL(True, "xxx.xxx.xxx.xxx", "1433", "User", "Password", "Initial database", "no")
