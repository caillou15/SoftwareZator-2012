Action : Go to one step in a wizard

This action will allow you to have an assistant (Aero Wizard or Wizard 97) to go to the specified step of it, if it exists.

Parameters

Parameters Descriptions
Wizard concerned Must be equal to the wizard (Aero Wizard or Wizard 97) for which to access one step.
Name of the tab representing the targeted step Used to enter the tab name (property "Name" a TabPage in a Aero Wizard or Wizard 97) to which the assistant has to move.

Mistake not to commit : For the parameter "Name of the tab representing the targeted step", you can not use the resources of the project type and Picture type Other. Single resource of type text can be used.

Mistake not to commit : If you use variables in parameter "Name of the tab representing the targeted step", make sure that the type of value of each of these variables is text (String), or a numeric value (Integer, Decimal, Double). Indeed, only text can be used in these settings.
Note : To check the type of a value, use the action Get the type of a variable, in General category.

Return value

No value is returned by this action
