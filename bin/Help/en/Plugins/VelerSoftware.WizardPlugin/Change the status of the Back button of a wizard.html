Action : Change the status of the Back button a wizard

This action will allow you to change the view state of button "Back" a wizard (Aero Wizard or Wizard 97).

Parameters

Parameters Descriptions
Wizard concerned Must be equal to the wizard (Aero Wizard or Wizard 97) for which to change the status of the button.
New status Sets the new status to be given:
  • Shown
  • Hidden
  • Enabled
  • Disabled

Return value

No value is returned by this action

Code Visual Basic.Net generated by SoftwareZator

If the parameter "New status" is equal to "Disabled", the generated code will be as follows:

Your_Wizard.DisableBackButton()

Runtime errors

No runtime error is detected here.
