Action : Show a Context menu strip

This action will display on the screen specified a context menu.

Warning : Make sure that the window is where your Context menu strip is shown (on the screen) when this action is run.

Parameters

Parameters Descriptions
Context menu strip concerned Must be equal to the context menu that will be displayed on the screen.
Control over which should display the menu If you set a control in this setting, the context menu specified above will be displayed above it.
X position from the peak in the upper left of the screen or control Must be equal to the horizontal position of the context menu to display.
If a control has been defined in the above setting, the position of the context menu will be adjusted relative to the position of the control (see illustration below right), otherwise the position will be coordinated on the screen (see picture below left).
Y position from the peak in the upper left of the screen or control Must be equal to the vertical position of the context menu to display.
If a control has been defined in the above setting, the position of the context menu will be adjusted relative to the position of the control (see illustration below right), otherwise the position will be coordinated on the screen (see picture below left).

Mistake not to commit : For the parameters "X position from the peak in the upper left corner of the screen or control" and "Y position from the peak in the upper left corner of the screen or control", you can not use the resources of the project type and Picture type Other. Single resource of type text can be used.

Mistake not to commit : If you use variables in parameters "X position from the peak in the upper left corner of the screen or control" and or "Y position from the peak in the upper left corner of the screen or control", make sure that the type of value of each of these variables is text (String), or a numeric value (Integer, Decimal, Double). Indeed, only text can be used in these settings.
Note : To check the type of a value, use the action Get the type of a variable, in General category.

Returned value

No value is returned by this action

Code Visual Basic.Net generated by SoftwareZator

If the control selected is "A_Control", that the horizontal position is equal to 25 and vertical position is equal to 30, the generated code will be as follows :

Your_Context_Menu.Show(A_Control, 25, 30)
