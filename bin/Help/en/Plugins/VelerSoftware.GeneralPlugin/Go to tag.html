Action : Go to tag

This action allows to call a previously created tag with the action Tag. When a tag is called, the execution point of the program is moved to the tag that has been defined.

Mistake not to commit : Be very attention to the name of the tag. It should never be the same as the name of a control, a resource, a setting or variable. Otherwise, the computer will confuse, for example, the "function" named XYZ with the "tag" named XYZ.

Parameters

Parameters Descriptions
