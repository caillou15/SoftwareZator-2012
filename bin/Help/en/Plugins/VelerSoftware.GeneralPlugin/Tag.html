Action : Tag

This action allows you to create a bookmark that can be called by the action Go to a tag. When a tag is called, the execution point of the program is moved to a tag that has been defined.

Mistake not to commit : Pay attention to the name of the tag. It should never be the same as the name of a control, a resource, a parameter, a different tag or a variable. Otherwise, the computer will confuse, for example, the "function" XYZ with the "tag" named XYZ.

Parameters

Parameters Descriptions
Name of the tag to create Must be equal to the name of the tag you want to create.

No value is returned by this action
