Action : Quit software

This action will exit your software. It almond to any open windows to close, then closes the application.

Warning : This action is only compatible with a Windows Application project type or Console Application.

Attention : Functions related to events Closed and Closing open windows in your software will not be raised when this action is executed. If any of these functions include validation has to be executed, you must use an action Close a window for each window individually before using this action.

No setting is necessary for this action
