Action : Getting the start-up arguments

This action retrieves the launch parameters of the software.

Example : When in your Windows File Explorer, you right-click on a file then "Open with", the software will be launched with in setting the full path to the file selected in the explorer.
Another example, when using the "Run" function in Start menu of Windows, the settings in your software data can be recovered through such action.
