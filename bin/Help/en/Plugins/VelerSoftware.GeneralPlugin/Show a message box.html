Action : Show a MessageBox

This action will allow you to display a message on the screen with the famous Windows message box.

Parameters

Parameters Descriptions
Title Sets the title of the message to display.
Buttons Allows you to choose which buttons to display in the message.
Icon Can choose the icon to be displayed to the left of the message box.
Texte of the message Must be equal to the text you want to appear in the message.
Variable receiving the result (optional) Must be equal to the variable (not array) receiving the result of the message displayed.

Mistake not to commit : For the parameters "Title" and "Text of the message", you can not use the resources of the project type and Picture type Other. Single resource of type text can be used.

Mistake not to commit : If you use variables in parameters "Title" and or "Text of the message", make sure that the type of value of each of these variables is text (String), or a numeric value (Integer, Decimal, Double). Indeed, only text can be used in these settings.
Note : To check the type of a value, use the action Get the type of a variable, in General category.

Valeur retournée

The return value is determined by the button selected by the user. It will be stored in the variable defined in the parameter "Variable receiving the result". This value is text (String).
