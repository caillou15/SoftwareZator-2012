Action : Add a control to a Flow layout panel

This action adds one control after another in a group.

Warning : Make sure that the window is where your Flow layout panel icon is shown (on the screen) when this action is run.

Parameters

Parameters Descriptions
Flow layout panel concerned Must be equal to the Flow layout panel in which one control will be added.
Control to add Allows you to enter the control to add. This can be created in a variable or at the time of the action (dynamic control).

Returned value

No value is returned by this action

Code Visual Basic.Net generated by SoftwareZator

If the Flow layout panel is in the same window from which you use this action, that control is a new button (created during the execution of the action), the generated code will be as follows :

Me.Your_FlowLayoutPanel.Controls.Add(New System.Windows.Forms.Button())

If the Flow layout panel is in a different window than where you use this action, that the control is stored in a variable, the generated code will be as follows :

Variables.Your_Windows.Your_FlowLayoutPanel.Controls.Add(Your_Variable)

Runtime errors

No runtime error is detected here.
