Action : Disconnect from a SQL Server

This action allows your software to disconnect from the Microsoft SQL Server which your software was connected with the action Connect to SQL Server

Parameters

Parameters Descriptions
Variable receiving the result (optional) Must be equal to the variable (not array) receiving the result of the disconnection.

Return value

The return value is determined by the success or failure of disconnection to the server. It will be stored in the variable defined in the parameter "Variable receiving the result". This value is numeric (Integer).

Result of the disconnection Returned values
Disconnected 1
Always connected 0
Unknow error -1
Note: An error message will appear on the screen.

Code Visual Basic.Net generated by SoftwareZator

If the variable receiving the result is "Result", the generated code will be as follows :

Result = VelerSoftware_SQLServerPlugin.Disconnect_SQLServer()

Runtime errors

No runtime error is detected here.
