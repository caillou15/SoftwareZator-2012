Action : Get the status of a connexion to SQL Server

This action retrieves a value to determine what state of the connecting to a Microsoft SQL Server.

Parameters

Parameters Descriptions
Variable receiving the result Must be equal to the variable (not array) receiving the connection status.

Return value

The return value is determined by the state of the connection to the server. It will be stored in the variable defined in the parameter "Variable receiving the result". This value is numeric (Integer).

Etat Returned values
Disconnected 0
Connected 1
Broken 2
While connecting 3
Running 4
While receiving 5

Code Visual Basic.Net generated by SoftwareZator

If the variable receiving the result is "Result", the generated code will be as follows :

Result = VelerSoftware_SQLServerPlugin.GetConnectStatus_SQLServer()

No runtime error is detected here.
