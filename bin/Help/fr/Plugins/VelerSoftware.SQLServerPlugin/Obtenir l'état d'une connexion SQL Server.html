Action : Obtenir l'état d'une connexion SQL Server

Cette action permet de récupérer une valeur permettant de déterminer dans quel état est la connexion à un serveur Microsoft SQL Server.

Paramètres

Paramètres Descriptions
Variable recevant le résultat Doit être égale à la variable (non tableau) recevant le statut de la connexion.

Valeur retournée

La valeur retournée est déterminée par l'état de la connexion au serveur. Elle sera enregistrée dans la variable définit dans le paramètre "Variable recevant le résultat". Cette valeur est de type numérique (Integer).

Etat Valeurs retournées
Déconnecté 0
Connecté 1
Interrompu 2
En cours de connexion 3
En cours d'exécution 4
En cours de réception 5

Code Visual Basic.Net généré par SoftwareZator

Si la variable recevant le résultat est "Resultat", le code généré sera tel que le suivant :

Resultat = VelerSoftware_SQLServerPlugin.GetConnectStatus_SQLServer()

Erreurs d'exécutions

Aucune erreur d'exécution n'est détecté ici.
