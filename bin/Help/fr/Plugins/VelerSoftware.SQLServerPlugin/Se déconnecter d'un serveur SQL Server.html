Action : Se déconnecter d'un serveur SQL Server

Cette action permet à votre logiciel de ce déconnecter d'un serveur de type Microsoft SQL Server auquel votre logiciel était précédemment connecté avec l'action Se connecter à un serveur SQL Server

Paramètres

Paramètres Descriptions
Variable recevant le résultat (facultatif) Doit être égale à la variable (non tableau) recevant le résultat de la déconnexion.

Valeur retournée

La valeur retournée est déterminée par la réussite ou non de la déconnexion au serveur. Elle sera enregistrée dans la variable définit dans le paramètre "Variable recevant le résultat". Cette valeur est de type numérique (Integer).

Résultat de la déconnexion Valeurs retournées
Déconnecté 1
Toujours connecté 0
Erreur inconnue -1
Remarque : Un message d'erreur apparaitra à l'écran.

Code Visual Basic.Net généré par SoftwareZator

Si la variable recevant le résultat est "Resultat", le code généré sera tel que le suivant :

Resultat = VelerSoftware_SQLServerPlugin.Disconnect_SQLServer()

Erreurs d'exécutions

Aucune erreur d'exécution n'est détecté ici.
