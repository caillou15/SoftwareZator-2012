Action : Se connecter à un port

Cette action permet d'ouvrir une connexion à un port série auquel est connecté un appareil (imprimantes, téléphones portables...). Pour plus d'information, consultez le document Qu'est-ce qu'un port série?.

Attention : Assurez-vous que la fenêtre dans laquelle réside votre Port série est ouverte (affiché à l'écran) au moment où cette action intervient.

Attention : Avant que cette action ne s'exécute, il faut que la propriété "SerialName" du contrôle de type Port série (dans le concepteur de fenêtre) ai été assigné et porte le nom du port qui sera ouvert.

Paramètres

Paramètres Descriptions
Port série concerné Doit être égale au contrôle de type Port série (dans le concepteur de fenêtre) qui doit se connecter à un appareil.

Valeur retournée

Aucune valeur n'est retourné par cette action

Code Visual Basic.Net généré par SoftwareZator

Si le port série est "MonPort", le code généré sera tel que le suivant :

Votre_Fenetre.MonPort.Open()

Erreurs d'exécutions

Il arrive qu'un programme informatique rencontre un problème durant son exécution. Souvent, un message d'erreur est alors affiché. Le tableau ci-dessous vous permet de déterminer pourquoi l'erreur que vous rencontrez s'affiche et comment la corriger :

Types d'erreurs Descriptions Resolutions
InvalidOperationException Cette erreur se produit lorsque le contrôle de type Port série est déjà connecté à un appareil. Pour corriger cette erreur, vous devez vous assurer que le port soit fermé avec l'action Se déconnecter d'un port avant l'exécution de celle-ci. Pour déterminer si un port série est connecté à un appareil, utilisez l'action Déterminer si un port série est connecté.
ArgumentOutOfRangeException Cette erreur se produit lorsqu'une ou plusieurs propriétés du contrôle de type Port série spécifié ne sont pas valides. Par exemple, les propriétés "Parity", "DataBits" ou "Handshake" n'ont pas des valeurs valides ; "BaudRate" est inférieure ou égale à zéro ; la propriété "ReadTimeout" ou "WriteTimeout" est inférieure à zéro et n'a pas la valeur "InfiniteTimeout". Pour corriger cette erreur, modifiez les valeur des propriétés du Port série.
ArgumentException Cette erreur se produit lorsque le nom de port auquel il faut se connecter ne commence pas par "COM". Pour corriger cette erreur, vous devez vous assurer que le nom du port commence par "COM" et essayez de lister les ports disponibles avec l'action Obtenir la liste des ports.
UnauthorizedAccessException Cette erreur se produit lorsque l'appareil ou l'ordinateur refuse l'accès au port. Pour corriger cette erreur, assurez-vous qu'aucun autre logiciel n'utilise ce port et ou redémarrez l'ordinateur.
