Action : Recevoir une donnée textuelle

Cette action permet de récupérer une donnée textuelle envoyé par un appareil (imprimantes, téléphones portables...) branché à un port série auquel votre projet est connecté avec l'action Se connecter à un port. Pour plus d'information, consultez le document Qu'est-ce qu'un port série?.

Attention : Assurez-vous que la fenêtre dans laquelle réside votre Port série est ouverte (affiché à l'écran) au moment où cette action intervient.

Paramètres

Paramètres Descriptions
Port série concerné Doit être égale au contrôle de type Port série (dans le concepteur de fenêtre) pour lequel il faut récupérer une donnée.
Variable recevant le résultat Doit être égale à la variable (non tableau) qui sera égale à la donnée textuelle envoyé par l'appareil si elle existe.

Valeur retournée

La valeur retournée est déterminée par la donnée textuelle renvoyé par l'appareil auquel votre contrôle de type Port Série est connecté. Cette valeur est de type texte (String).

Code Visual Basic.Net généré par SoftwareZator

Si le port série est "MonPort", le code généré sera tel que le suivant :

Variable = Votre_Fenetre.MonPort.ReadExisting()

Erreurs d'exécutions

Il arrive qu'un programme informatique rencontre un problème durant son exécution. Souvent, un message d'erreur est alors affiché. Le tableau ci-dessous vous permet de déterminer pourquoi l'erreur que vous rencontrez s'affiche et comment la corriger :

Types d'erreurs Descriptions Resolutions
InvalidOperationException Cette erreur se produit lorsque le contrôle de type Port série n'est pas connecté à un appareil. Pour corriger cette erreur, vous devez vous assurer que le port soit ouvert avec l'action Se connecter à un port avant l'exécution de celle-ci. Pour déterminer si un port série est connecté à un appareil, utilisez l'action Déterminer si un port série est connecté.
TimeoutException Cette erreur se produit lorsque l'opération ne s'est pas terminée avant la fin du délai d'attente (définit par la propriété "ReadTimeout" du contrôle de type Port série (dans le concepteur de fenêtre). Pour corriger cette erreur, vous devez vous assurer que le délai d'attente soit assez long (souvent suppérieur à 200 millisecondes, ou bien infini (égale à -1).
