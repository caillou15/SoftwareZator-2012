Action : Se déconnecter d'un port

Cette action permet de fermer une connexion à un port série auquel est connecté un appareil (imprimantes, téléphones portables...) avec l'action Se connecter à un port. Pour plus d'information, consultez le document Qu'est-ce qu'un port série?.

Attention : Assurez-vous que la fenêtre dans laquelle réside votre Port série est ouverte (affiché à l'écran) au moment où cette action intervient.

Paramètres

Paramètres Descriptions
Port série concerné Doit être égale au contrôle de type Port série (dans le concepteur de fenêtre) qui doit se déconnecter d'un appareil.

Valeur retournée

Aucune valeur n'est retourné par cette action

Code Visual Basic.Net généré par SoftwareZator

Si le port série est "MonPort", le code généré sera tel que le suivant :

Votre_Fenetre.MonPort.Close()

Erreurs d'exécutions

Il arrive qu'un programme informatique rencontre un problème durant son exécution. Souvent, un message d'erreur est alors affiché. Le tableau ci-dessous vous permet de déterminer pourquoi l'erreur que vous rencontrez s'affiche et comment la corriger :

Types d'erreurs Descriptions Resolutions
InvalidOperationException Cette erreur se produit lorsque le contrôle de type Port série n'est pas connecté à un appareil au moment de l'exécution de cette action. Pour corriger cette erreur, vous devez vous assurer que le port soit ouvert avec l'action Se connecter à un port avant l'exécution de celle-ci. Pour déterminer si un port série est connecté à un appareil, utilisez l'action Déterminer si un port série est connecté.
