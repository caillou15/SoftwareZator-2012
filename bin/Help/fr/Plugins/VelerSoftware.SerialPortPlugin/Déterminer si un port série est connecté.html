Action : Déterminer si un port série est connecté

Cette action permet de déterminer si un contrôle de type Port série est connecté à un appareil (imprimantes, téléphones portables...). Pour plus d'information, consultez le document Qu'est-ce qu'un port série?.

Attention : Assurez-vous que la fenêtre dans laquelle réside votre Port série est ouverte (affiché à l'écran) au moment où cette action intervient.

Paramètres

Paramètres Descriptions
Port série concerné Doit être égale au contrôle de type Port série (dans le concepteur de fenêtre) dont on doit déterminer l'état de connexion.
Variable recevant le résultat Doit être égale à la variable (non tableau) qui sera égale à la valeur qui détermine si le Port série est connecté ou non.

Valeur retournée

La valeur retournée est déterminée par le status de connexion du Port série à l'appareil. Elle sera enregistrée dans la variable définit dans le paramètre "Variable recevant le résultat". Cette valeur est de type Vrai/Faux (Boolean).

Résultat du teste Valeurs retournées
Le Port série est connecté à l'appareil True
Le Port série est déconnecté False

Code Visual Basic.Net généré par SoftwareZator

Si le port série est "MonPort", le code généré sera tel que le suivant :

Variable = Votre_Fenetre.MonPort.IsOpen

Erreurs d'exécutions

Aucune erreur d'exécution n'est détecté ici.
