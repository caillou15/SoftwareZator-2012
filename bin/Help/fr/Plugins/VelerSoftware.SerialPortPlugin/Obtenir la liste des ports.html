Action : Obtenir la liste des ports

Cette action permet de récupérer la liste des noms des ports séries auquel sont connectés des appareils (imprimantes, téléphones portables...). Pour plus d'information, consultez le document Qu'est-ce qu'un port série?.

Paramètres

Paramètres Descriptions
Variable recevant la liste des ports séries en état de connexion Doit être égale à la variable tableau recevant la liste des noms des ports série auquel des appareils sont connectés.

Valeur retournée

La valeur retournée est déterminée par les appareils connectés à l'ordinateur. Elle sera enregistrée dans la variable définit dans le paramètre "Variable recevant la liste des ports séries en état de connexion". Cette valeur est une liste de valeurs de type texte (System.String).

Code Visual Basic.Net généré par SoftwareZator

Si la variable recevant le résultat est "Resultat", le code généré sera tel que le suivant :

Resultat = New System.Collections.Generic.List(Of Object)(System.Linq.Enumerable.Cast(Of Object)(System.Linq.Enumerable.AsEnumerable(System.IO.Ports.SerialPort.GetPortNames())))

Erreurs d'exécutions

Il arrive qu'un programme informatique rencontre un problème durant son exécution. Souvent, un message d'erreur est alors affiché. Le tableau ci-dessous vous permet de déterminer pourquoi l'erreur que vous rencontrez s'affiche et comment la corriger :

Types d'erreurs Descriptions Resolutions
System.ComponentModel.Win32Exception Cette erreur se produit lorsque les noms de ports série n'ont pas pu être interrogés. Pour corriger cette erreur, essayez de débrancher puis rebrancher tous les périphériques et ou redémarrer l'ordinateur.
