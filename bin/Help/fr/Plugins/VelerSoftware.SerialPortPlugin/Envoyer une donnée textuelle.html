Action : Envoyer une donnée textuelle

Cette action permet d'envoyer une donnée textuelle à un port série auquel est connecté un appareil (imprimantes, téléphones portables...) avec l'action Se connecter à un port. Pour plus d'information, consultez le document Qu'est-ce qu'un port série?.

Attention : Assurez-vous que la fenêtre dans laquelle réside votre Port série est ouverte (affiché à l'écran) au moment où cette action intervient.

Paramètres

Paramètres Descriptions
Port série concerné Doit être égale au contrôle de type Port série (dans le concepteur de fenêtre) qui doit envoyer une donnée à l'appareil auquel il est connecté.
Donnée textuelle à envoyer Permet de saisir la ou les commandes textuelles à envoyer à l'appareil connecté à l'ordinateur.

Valeur retournée

Aucune valeur n'est retourné par cette action

Code Visual Basic.Net généré par SoftwareZator

Si le port série est "MonPort", le code généré sera tel que le suivant :

Votre_Fenetre.MonPort.Write("Une donnée")

Erreurs d'exécutions

Il arrive qu'un programme informatique rencontre un problème durant son exécution. Souvent, un message d'erreur est alors affiché. Le tableau ci-dessous vous permet de déterminer pourquoi l'erreur que vous rencontrez s'affiche et comment la corriger :

Types d'erreurs Descriptions Resolutions
InvalidOperationException Cette erreur se produit lorsque le contrôle de type Port série n'est pas connecté à un appareil. Pour corriger cette erreur, vous devez vous assurer que le port soit ouvert avec l'action Se connecter à un port avant l'exécution de celle-ci. Pour déterminer si un port série est connecté à un appareil, utilisez l'action Déterminer si un port série est connecté.
ArgumentNullException Cette erreur se produit lorsque la valeur de la propriété "Donnée textuelle à envoyer" est nulle. Pour corriger cette erreur, vous devez saisir une valeur dans le paramètre "Donnée textuelle à envoyer" de cette action.
TimeoutException Cette erreur se produit lorsque l'opération ne s'est pas terminée avant la fin du délai d'attente (définit par la propriété "WriteTimeout" du contrôle de type Port série (dans le concepteur de fenêtre). Pour corriger cette erreur, vous devez vous assurer que le délai d'attente soit assez long (souvent suppérieur à 200 millisecondes, ou bien infini (égale à -1).
System.InvalidCastException Cette erreur se produit lorsque la valeur du paramètre "Donnée textuelle à envoyer" n'est pas du texte. Souvent, le problème est causé par une variable utilisé dans ce paramètre et qui a une valeur autre que du texte. Pour corriger cette erreur, vérifiez que toutes les valeurs impliquées dans ce paramètre sont bien de type Texte (String) et faites les modifications nécessaires pour que ces valeurs soit du texte. Pour cela, utilisez les actions Obtenir le type d'une variable, Convertir la valeur d'une variable
