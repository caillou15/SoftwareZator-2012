Action : Arrêter la reconnaissance vocale

Cette action va vous permettre de stopper le fonctionnement de la reconnaissance vocale précédemment démarré avec l'action Démarrer la reconnaissance vocale.
Note : Si cette action est utilisé alors que la reconnaissance vocale spécifié est déjà arrêté, aucune erreur ne sera levée.

Attention : Assurez-vous que la fenêtre dans laquelle réside votre reconnaissance vocale est ouverte (affiché à l'écran) au moment où cette action intervient.

Attention : La reconnaissance vocale n'est pas compatible avec Microsoft Windows XP.

Paramètres

Paramètres Descriptions
Reconnaissance vocale à stopper Doit être égale à la reconnaissance vocale que vous voulez arrêter.

Valeur retournée

Aucune valeur n'est retourné par cette action

Code Visual Basic.Net généré par SoftwareZator

Si la reconnaissance vocale se trouve dans la même fenêtre que celle d'où vous utilisez cette action, le code généré sera le suivant :

Me.Votre_Reconnaissance_Vocale.Stop_SpeechRecognition()

Si la reconnaissance vocale se trouve dans une autre fenêtre que celle d'où vous utilisez cette action, le code généré sera le suivant :

Variables.Votre_Fenetre.Votre_Reconnaissance_Vocale.Stop_SpeechRecognition()

Erreurs d'exécutions

Aucune erreur d'exécution n'est détecté ici.
