Action : Démarrer la reconnaissance vocale

Cette action va vous permettre de démarrer le fonctionnement de la reconnaissance vocale précédemment ajouté au Concepteur de fenêtre. Une fois la reconnaissance vocale démarré, il exécutera la fonction lié à l'évènement OnSpeechRecognized de celui-ci à chaque fois qu'une phrase prononcée dans le microphone de l'ordinateur sera comprise.

Attention : Assurez-vous que la fenêtre dans laquelle réside votre reconnaissance vocale est ouverte (affiché à l'écran) au moment où cette action intervient.

Attention : La reconnaissance vocale n'est pas compatible avec Microsoft Windows XP.

Paramètres

Paramètres Descriptions
Reconnaissance vocale à démarrer Doit être égale à la reconnaissance vocale que vous voulez démarrer.

Valeur retournée

Aucune valeur n'est retourné par cette action

Code Visual Basic.Net généré par SoftwareZator

Si la reconnaissance vocale se trouve dans la même fenêtre que celle d'où vous utilisez cette action, le code généré sera le suivant :

Me.Votre_Reconnaissance_Vocale.Start_SpeechRecognition()

Si la reconnaissance vocale se trouve dans une autre fenêtre que celle d'où vous utilisez cette action, le code généré sera le suivant :

Variables.Votre_Fenetre.Votre_Reconnaissance_Vocale.Start_SpeechRecognition()

Erreurs d'exécutions

Il arrive qu'un programme informatique rencontre un problème durant son exécution. Souvent, un message d'erreur est alors affiché. Le tableau ci-dessous vous permet de déterminer pourquoi l'erreur que vous rencontrez s'affiche et comment la corriger :

Types d'erreurs Descriptions Resolutions
NotSupportedException Cette erreur se produit lorsque votre projet veut démarrer la reconnaissance vocale alors qu'elle n'est pas pris en charge par le système d'exploitation sur lequel il est exécuté. En effet, la reconnaissance vocale n'est pas compatible avec Microsoft Windows XP. Pour corriger cette erreur, empêchez à l'utilisateur de démarrer la reconnaissance vocale sous Microsoft Windows XP.
