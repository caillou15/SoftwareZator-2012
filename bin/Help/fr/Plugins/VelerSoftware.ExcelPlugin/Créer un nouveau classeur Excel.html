Action : Créer un nouveau classeur Excel

Cette action permet d'initialiser un nouveau classeur Excel et de l'assigner à une variable.

Important : pour utiliser cette action, il faut impérativement que le logiciel Microsoft Excel (2007 ou 2010 ou suppérieur) soit installé sur l'ordinateur de la personne utilisant votre projet.

Attention : lorsque cette action s'exécute, une nouvelle instance du logiciel Microsoft Excel installé sur l'ordinateur de l'utilisateur est lancé, que le paramètre "Afficher la fenêtre du logiciel Microsoft Excel à l'écran" soit coché ou pas.
Avant de changer entièrement la valeur de la variable utilisé dans le paramètre "Variable recevant le nouveau classeur Excel", merci d'utiliser l'action Fermer un classeur Excel pour fermer automatiquement le logiciel Microsoft Excel lorsque votre logiciel n'en a plus besoin. Vous devrez donc utiliser cette action avant que cette variable ne soit plus utilisable avec les actions pour Excel sans être obligé de réutiliser cette action.

Paramètres

Paramètres Descriptions
Variable reçevant le nouveau classeur Excel Doit être égale à la variable (non tableau) qui sera égale au nouveau classeur Excel.
Afficher la fenêtre du logiciel Microsoft Excel à l'écran Détermine si la fenêtre du logiciel Microsoft Excel devra être affiché à l'écran ou bien masqué. Ce paramètre ne change en rien le lancement de Excel durant l'exécution de cette action.

Valeur retournée

Aucune valeur n'est retourné par cette action

Code Visual Basic.Net généré par SoftwareZator
