Action : Fermer un classeur Excel

Cette action permet de fermer un classeur Excel stocké dans une variable afin de libérer cette variable et fermer l'instance du logiciel Microsoft Excel lancé pour ce classeur, sans enregistrer les modifications apportés. La fenêtre Excel lié à cette variable sera alors automatiquement fermé.

Important : pour utiliser cette action, il faut impérativement que le logiciel Microsoft Excel (2007 ou 2010 ou suppérieur) soit installé sur l'ordinateur de la personne utilisant votre projet.

Astuce : pour enregistrer les modifications apportées avant de fermer un classeur Excel, utilisez l'action Enregistrer un classeur Excel.

Paramètres

Paramètres Descriptions
Variable possédant le classeur Excel à fermer Doit être égale à la variable (non tableau) qui possède le classeur Excel que vous voulez fermer. Cette variable doit préalablement avoir été assigné grâce à l'action Créer un nouveau classeur Excel ou Ouvrir un classeur Excel.

Valeur retournée

Aucune valeur n'est retourné par cette action
