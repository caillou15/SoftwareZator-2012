Action : Imprimer une feuille d'un classeur Excel

Cette action permet d'imprimer la feuille actuelle ou toutes les feuilles d'un classeur Excel ayant été ouvert ou créé précédemment.

Important : pour utiliser cette action, il faut impérativement que le logiciel Microsoft Excel (2007 ou 2010 ou suppérieur) soit installé sur l'ordinateur de la personne utilisant votre projet.

Paramètres

Paramètres Descriptions
Variable possédant le classeur Excel concerné Doit être égale à la variable (non tableau) qui possède le classeur Excel pour lequel vous voulez imprimer l'une de ses feuille ou toutes ses feuilles. Cette variable doit préalablement avoir été assigné grâce à l'action Créer un nouveau classeur Excel ou Ouvrir un classeur Excel.
Imprimer toutes les feuilles Détermine si toutes les feuilles du classeur doivent être imprimées ou bien si la feuille actuelle (définie avec l'action "Définir la feuille actuelle) uniquement doit être imprimée.

Valeur retournée

Aucune valeur n'est retourné par cette action

Code Visual Basic.Net généré par SoftwareZator

Si le paramètre "Imprimer toutes les feuilles" est décoché, le code généré sera le suivant :

Votre_Variable.PrintWorksheet(1)

Si le paramètre "Imprimer toutes les feuilles" est coché, le code généré sera le suivant :

Votre_Variable.PrintAllWorksheets(1)

Erreurs d'exécutions

Aucune erreur d'exécution n'est détecté ici.
