Action : Obtenir le nombre de feuilles

Cette action permet de récupérer le nombre de feuilles d'un classeur Excel ayant été ouvert ou créé précédemment.

Important : pour utiliser cette action, il faut impérativement que le logiciel Microsoft Excel (2007 ou 2010 ou suppérieur) soit installé sur l'ordinateur de la personne utilisant votre projet.

Paramètres

Paramètres Descriptions
Variable possédant le classeur Excel concerné Doit être égale à la variable (non tableau) qui possède le classeur Excel auquel vous voulez récupérer le nombre de feuilles. Cette variable doit préalablement avoir été assigné grâce à l'action Créer un nouveau classeur Excel ou Ouvrir un classeur Excel.
Variable recevant le nombre de feuilles Doit être égale à la variable (non tableau) qui sera égale au nombre de feuilles du classeur Excel.

Valeur retournée

La valeur retournée est déterminée par le nombre de feuilles au moment de l'exécution de cette action. Cette valeur est de type Nombre entier (Integer).

Code Visual Basic.Net généré par SoftwareZator

Variable_De_Resultat = Votre_Variable.WorksheetCount
