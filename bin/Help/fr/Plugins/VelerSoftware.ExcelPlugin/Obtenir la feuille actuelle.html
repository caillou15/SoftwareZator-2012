Action : Obtenir la feuille actuelle

Cette action permet de récupérer le nom de la feuille, avec laquelle le programme travail actuellement, d'un classeur Excel ayant été ouvert ou créé précédemment.

Important : pour utiliser cette action, il faut impérativement que le logiciel Microsoft Excel (2007 ou 2010 ou suppérieur) soit installé sur l'ordinateur de la personne utilisant votre projet.

Paramètres

Paramètres Descriptions
Variable possédant le classeur Excel concerné Doit être égale à la variable (non tableau) qui possède le classeur Excel auquel vous voulez récupérer la feuille actuelle. Cette variable doit préalablement avoir été assigné grâce à l'action Créer un nouveau classeur Excel ou Ouvrir un classeur Excel.
Variable recevant le nom de la feuille Doit être égale à la variable (non tableau) qui sera égale au nom de la feuille avec laquelle le programme travail actuellement.

Valeur retournée

La valeur retournée est déterminée par le nom de la feuille sur lequel votre programme travail au moment de l'exécution de cette action. Cette valeur est de type texte (String).

Code Visual Basic.Net généré par SoftwareZator

Variable_De_Resultat = Votre_Variable.CurrentWorksheet
