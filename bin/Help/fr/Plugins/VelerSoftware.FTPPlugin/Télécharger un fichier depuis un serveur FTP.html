Action : Télécharger un fichier depuis un serveur FTP

Cette action permet de télécharger un fichier depuis un serveur de type FTP (File Transfer Protocol) sur le disque dur, auquel votre logiciel était précédemment connecté avec l'action Se connecter à un serveur FTP.

Information : durant l'exécution de cette application, votre logiciel se figera pendant le téléchargement du fichier, avant de continuer la fonction. Pour éviter celà, utiliser un contrôle de type "Processus d'arrière-plan".

Paramètres

Paramètres Descriptions
Chemin complet vers le fichier à télécharger depuis le serveur Doit être égale au chemin distant complet vers le fichier qui devra être téléchargé depuis le serveur FTP.
Emplacement et nom du fichier une fois sur le disque dur Permet de saisir l'emplacement sur le disque dur dans lequel le fichier sera téléchargé et le nom qu'il portera sur ce disque.
Variable recevant le résultat (facultatif) Doit être égale à la variable (non tableau) recevant le résultat du téléchargement.

Erreur à ne pas commettre : Pour les paramètres "Chemin complet vers le fichier à télécharger depuis le serveur" et "Emplacement et nom du fichier une fois sur le disque dur", vous ne pouvez pas utiliser des ressources du projet de type Image et de type Autre. Seule des ressources de type Texte peuvent être utilisées.

Erreur à ne pas commettre : Si vous utilisez dans variables dans les paramètres "Chemin complet vers le fichier à télécharger depuis le serveur" et "Emplacement et nom du fichier une fois sur le disque dur", assurez-vous que le type de la valeur de chacune de ces variables soit du texte (String), ou bien une valeur de type numérique (Integer, Decimal, Double). En effet, seul du texte peut être utilisé dans ces paramètres.
Note : Pour vérifier le type d'une valeur, utilisez l'action Obtenir le type d'une variable, dans la catégorie Général.

Valeur retournée

La valeur retournée est déterminée par la réussite ou non de l'envoie du fichier sur le serveur. Elle sera enregistrée dans la variable définit dans le paramètre "Variable recevant le résultat". Cette valeur est de type numérique (Integer).

Résultat Valeurs retournées
Fichier téléchargé 1
Non téléchargé 0
Remarque : Un message d'erreur apparaitra à l'écran.

Code Visual Basic.Net généré par SoftwareZator

Si la variable recevant le résultat est "Resultat", le code généré sera tel que le suivant :

Resultat = VelerSoftware_FTPPlugin.DownloadFile("/dossier/fichier.txt", "C:\Fichier.txt")

Erreurs d'exécutions

Il arrive qu'un programme informatique rencontre un problème durant son exécution. Souvent, un message d'erreur est alors affiché. Le tableau ci-dessous vous permet de déterminer pourquoi l'erreur que vous rencontrez s'affiche et comment la corriger :

Types d'erreurs Descriptions Resolutions
System.ArgumentException
ou
System.InvalidCastException		 Cette erreur se produit lorsque la valeur des paramètres "Chemin complet vers le fichier à télécharger depuis le serveur" et "Emplacement et nom du fichier une fois sur le disque dur" ne sont pas du texte ou un nombre. Souvent, le problème est causé par une variable utilisé dans ces paramètres et qui a une valeur autre que du texte ou un nombre. Pour corriger cette erreur, vérifiez que toutes les valeurs impliquées dans ces paramètres sont bien de type Texte (String) ou Numérique (Integer) et faites les modifications nécessaires pour que ces valeurs soit du texte. Pour cela, utilisez les actions Obtenir le type d'une variable, Convertir la valeur d'une variable
