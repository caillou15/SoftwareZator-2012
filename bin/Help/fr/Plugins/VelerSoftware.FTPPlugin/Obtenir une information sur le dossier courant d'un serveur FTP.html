Action : Obtenir une information sur le dossier courant d'un serveur FTP

Cette action permet de récupérer une information le dossier définit comme étant le dossier courant de travail, tel que son nom, sa date de création, ses attributs et autre, sur un serveur de type FTP (File Transfer Protocol) auquel votre logiciel était précédemment connecté avec l'action Se connecter à un serveur FTP.

Paramètres

Paramètres Descriptions
Variable recevant le résultat Doit être égale à la variable (non tableau) recevant les informations sur le dossier.

Valeur retournée

La valeur retournée est déterminée par le dossier courant sur le serveur. Elle sera enregistrée dans la variable définit dans le paramètre "Variable recevant le résultat". Cette valeur est de type "VelerSoftware.FTPLib.FtpDirectoryInfo".

Information : si aucun dossier courant n'est définit ou s'il n'a pas put être récupéré, la valeur retourné sera nulle.

Code Visual Basic.Net généré par SoftwareZator

Si la variable recevant le résultat est "Resultat", le code généré sera tel que le suivant :

Resultat = VelerSoftware_FTPPlugin.GetCurrentDirectoryInfo()

Erreurs d'exécutions

Aucune erreur d'exécution n'est détecté ici.
