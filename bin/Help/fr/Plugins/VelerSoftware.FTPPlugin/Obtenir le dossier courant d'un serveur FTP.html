Action : Obtenir le dossier courant d'un serveur FTP

Cette action permet de récupérer le dossier définit comme étant le dossier courant de travail sur un serveur de type FTP (File Transfer Protocol) auquel votre logiciel était précédemment connecté avec l'action Se connecter à un serveur FTP.

Paramètres

Paramètres Descriptions
Variable recevant le résultat Doit être égale à la variable (non tableau) recevant le chemin et le nom du dossier.

Valeur retournée

La valeur retournée est déterminée par le chemin et le nom du dossier courant sur le serveur. Elle sera enregistrée dans la variable définit dans le paramètre "Variable recevant le résultat". Cette valeur est de type texte (String).

Résultat Valeurs retournées
Dossier récupéré Texte égale au chemin et au nom du dossier courant
Non récupéré -1
Remarque : Un message d'erreur apparaitra à l'écran.

Code Visual Basic.Net généré par SoftwareZator

Si la variable recevant le résultat est "Resultat", le code généré sera tel que le suivant :

Resultat = VelerSoftware_FTPPlugin.GetCurrentDirectory()

Erreurs d'exécutions

Aucune erreur d'exécution n'est détecté ici.
