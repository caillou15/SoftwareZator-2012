Action : Se déconnecter d'un serveur FTP

Cette action permet à votre logiciel de ce déconnecter d'un serveur de type FTP (File Transfer Protocol) auquel votre logiciel était précédemment connecté avec l'action Se connecter à un serveur FTP.
