Action : Obtenir la taille d'un fichier sur un serveur FTP

Cette action permet de récupérer la taille d'un fichier à l'emplacement spécifié sur un serveur de type FTP (File Transfer Protocol) auquel votre logiciel était précédemment connecté avec l'action Se connecter à un serveur FTP.

Paramètres

Paramètres Descriptions
Chemin complet vers le fichier concerné Doit être égale au chemin complet vers le fichier dont on doit récupérer la taille.
Variable recevant le résultat Doit être égale à la variable (non tableau) recevant la taille du fichier.

Erreur à ne pas commettre : Pour le paramètre "Chemin complet vers le fichier concerné", vous ne pouvez pas utiliser des ressources du projet de type Image et de type Autre. Seule des ressources de type Texte peuvent être utilisées.

Erreur à ne pas commettre : Si vous utilisez dans variables dans le paramètre "Chemin complet vers le fichier concerné", assurez-vous que le type de la valeur de chacune de ces variables soit du texte (String), ou bien une valeur de type numérique (Integer, Decimal, Double). En effet, seul du texte peut être utilisé dans ce paramètre.
Note : Pour vérifier le type d'une valeur, utilisez l'action Obtenir le type d'une variable, dans la catégorie Général.

Valeur retournée

La valeur retournée est déterminée par la taille du fichier sur le serveur. Elle sera enregistrée dans la variable définit dans le paramètre "Variable recevant le résultat". Cette valeur est de type numérique (Integer).

Résultat Valeurs retournées
Aucune erreur Entier numérique égale à la taille du fichier en Octet
Erreur -1
Remarque : Un message d'erreur apparaitra à l'écran.

Code Visual Basic.Net généré par SoftwareZator

Si la variable recevant le résultat est "Resultat", le code généré sera tel que le suivant :

Resultat = VelerSoftware_FTPPlugin.GetFileSize("/chemin/fichier concerné")

Erreurs d'exécutions

Il arrive qu'un programme informatique rencontre un problème durant son exécution. Souvent, un message d'erreur est alors affiché. Le tableau ci-dessous vous permet de déterminer pourquoi l'erreur que vous rencontrez s'affiche et comment la corriger :

Types d'erreurs Descriptions Resolutions
System.ArgumentException
ou
System.InvalidCastException		 Cette erreur se produit lorsque la valeur du paramètre "Chemin complet vers le fichier concerné" n'est pas du texte ou un nombre. Souvent, le problème est causé par une variable utilisé dans ce paramètre et qui a une valeur autre que du texte ou un nombre. Pour corriger cette erreur, vérifiez que toutes les valeurs impliquées dans ces paramètres sont bien de type Texte (String) ou Numérique (Integer) et faites les modifications nécessaires pour que ces valeurs soit du texte. Pour cela, utilisez les actions Obtenir le type d'une variable, Convertir la valeur d'une variable
