Action : Se connecter à une base de données Access

Cette action permet à votre logiciel de ce connecter à une base de données de type Microsoft Access.
Une base de données Access est souvent représenté par un fichier de type MDB.

Compatibilité

Les bases de données Microsoft Access sont uniquement disponibles pour des logiciels fonctionnant en 32 bit. En outre, votre logiciel doit être configuré de manière à qu'il soit utilisé en 32 bit uniquement.

Information : un logiciel compilé pour un système d'exploitation 32 bit peut fonctionner sur un système d'exploitation en 64 bit. La différence est que ce logiciel ne pourra pas profiter des performances que lui offre le système.

Notez donc qu'en compilant votre logiciel pour un système 32 bit, celui-ci fonctionnera sous un Windows 64 bit, donc, l'utilisation des bases de données Microsoft Access pourront être utilisés avec votre logiciel sous un Windows 32 bit et 64 bit, tant que celui-ci est compilé pour un système d'exploitation ciblé en 32 bit.

Pour changer le type de système d'exploitation cible afin que votre logiciel supporte Microsoft Access sur tout type de système d'exploitation Windows (XP, Vista, 7, 8), allez dans les "Paramètres du projet", puis l'onglet "Générer", et enfin placez le paramètre "Type de système d'exploitation cible" sur la valeur "Windows 32 bit uniquement".

Paramètres

Paramètres Descriptions
Chemin complet vers le fichier de base de données (*.mdb) Doit être égale au chemin complet du fichier de base de données Access auquel votre logiciel doit se connecter.
Mot de passe (facultatif) Doit être égale au mot de passe à utiliser dans le cas où un identifiant est requis pour se connecter à la base de données.
Variable recevant le résultat (facultatif) Doit être égale à la variable (non tableau) recevant le résultat de la connexion.

Erreur à ne pas commettre : Pour les paramètres "Chemin complet vers le fichier de base de données", "Mot de passe", vous ne pouvez pas utiliser des ressources du projet de type Image et de type Autre. Seule des ressources de type Texte peuvent être utilisées.

Erreur à ne pas commettre : Si vous utilisez dans variables dans les paramètres "Chemin complet vers le fichier de base de données", "Mot de passe", assurez-vous que le type de la valeur de chacune de ces variables soit du texte (String), ou bien une valeur de type numérique (Integer, Decimal, Double). En effet, seul du texte peut être utilisé dans ces paramètres.
Note : Pour vérifier le type d'une valeur, utilisez l'action Obtenir le type d'une variable, dans la catégorie Général.

Valeur retournée

La valeur retournée est déterminée par la réussite ou non de la connexion à la base de données. Elle sera enregistrée dans la variable définit dans le paramètre "Variable recevant le résultat". Cette valeur est de type numérique (Integer).

Résultat de la connexion Valeurs retournées
Connecté 1
Le fichier de base de données est introuvable 0
Mot de passe incorrecte -2
Erreur inconnue -1
Remarque : Un message d'erreur apparaitra à l'écran.

Code Visual Basic.Net généré par SoftwareZator

Resultat = VelerSoftware_AccessPlugin.Connect_Access("C:\Votre fichier.mdb", "Mot de passe")

Erreurs d'exécutions

Il arrive qu'un programme informatique rencontre un problème durant son exécution. Souvent, un message d'erreur est alors affiché. Le tableau ci-dessous vous permet de déterminer pourquoi l'erreur que vous rencontrez s'affiche et comment la corriger :

Types d'erreurs Descriptions Resolutions
System.ArgumentException
ou
System.InvalidCastException		 Cette erreur se produit lorsque la valeur du paramètre "Chemin complet vers le fichier de base de données" et ou "Mot de passe" ne sont pas du texte. Souvent, le problème est causé par une variable utilisé dans ces paramètres et qui ont une valeur autre que du texte. Pour corriger cette erreur, vérifiez que toutes les valeurs impliquées dans ces paramètres sont bien de type Texte (String) et faites les modifications nécessaires pour que ces valeurs soit du texte. Pour cela, utilisez les actions Obtenir le type d'une variable, Convertir la valeur d'une variable
