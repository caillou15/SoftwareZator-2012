Action : Obtenir la liste des bases de données MySQL

Cette action permet de récupérer la liste des bases de données présentes sur un serveur MySQL.

Paramètres

Paramètres Descriptions
Variable recevant le résultat Doit être égale à la variable tableau recevant la liste des bases de données.

Valeur retournée

La valeur retournée est déterminée par les bases de données installés sur le serveur. Elle sera enregistrée dans la variable définit dans le paramètre "Variable recevant le résultat". Cette valeur est une liste de valeurs de type texte (System.String).

Code Visual Basic.Net généré par SoftwareZator

Si la variable recevant le résultat est "Resultat", le code généré sera tel que le suivant :

Resultat = VelerSoftware_MySQLPlugin.GetDatabases_MySQL()

Erreurs d'exécutions

Aucune erreur d'exécution n'est détecté ici.
