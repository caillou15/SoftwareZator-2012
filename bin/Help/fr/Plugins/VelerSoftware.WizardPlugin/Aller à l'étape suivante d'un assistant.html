Action : Aller à l'étape suivante d'un assistant

Cette action va vous permettre de demander à un assistant (Assistant Aero ou Assistant 97) d'aller à l'étape suivante de celui-ci, dans le cas où celle-ci existe.

Paramètres

Paramètres Descriptions
Assistant concerné Doit être égale à l'assistant (Assistant Aero ou Assistant 97) pour lequel il faut accéder à l'étape suivante.

Valeur retournée

Aucune valeur n'est retourné par cette action

Code Visual Basic.Net généré par SoftwareZator

Votre_Assistant.GoToNextStep()

Erreurs d'exécutions

Aucune erreur d'exécution n'est détecté ici.
