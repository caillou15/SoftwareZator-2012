Action : Changer le statut du bouton Suivant d'un assistant

Cette action va vous permettre de changer l'état d'affichage du bouton "Suivant" d'un assistant (Assistant Aero ou Assistant 97).

Paramètres

Paramètres Descriptions
Assistant concerné Doit être égale à l'assistant (Assistant Aero ou Assistant 97) pour lequel il faut changer le statut du bouton.
Nouveau statut Permet de définir le nouveau statut à donner :
  • Affiché
  • Masqué
  • Activé
  • Désactivé

Valeur retournée

Aucune valeur n'est retourné par cette action

Code Visual Basic.Net généré par SoftwareZator

Si le paramètre "Nouveau statut" est égale à "Désactivé", le code généré sera le suivant :

Votre_Assistant.DisableNextButton()

Erreurs d'exécutions

Aucune erreur d'exécution n'est détecté ici.
