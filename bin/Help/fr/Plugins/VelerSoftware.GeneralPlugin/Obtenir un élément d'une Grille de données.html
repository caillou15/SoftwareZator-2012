Action : Obtenir un élément d'une Grille de données

Cette action permet de récupérer une ligne existante dans une Grille de données, à l'index spécifié.

Attention : Assurez-vous que la fenêtre dans laquelle réside votre Grille de données est ouverte (affiché à l'écran) au moment où cette action intervient.

Paramètres

Paramètres Descriptions
Grille de données concerné Doit être égale à la Grille de données dont il faut récupérer une ligne.
Index de l'élément Ce paramètre doit être égale à l'index de la ligne que vous voulez récupérer dans la Grille de données. Autrement dit, la place de cet élément dans la liste. Si vous indiquez l'index 0 par exemple, l'élément (ou la ligne) récupéré sera le tout premier de la Grille de données. Si vous indiquez l'index 1 par exemple, l'élément récupéré sera celui juste avant le second élément de la liste. On peut illustrer ce paramètre avec l'illustration ci-dessous.
Variable recevant le résultat Doit être égale à la variable (non tableau) qui sera égale à la ligne récupérée dans la Grille de données.

Valeur retournée

La valeur retournée est déterminée par la position de la ligne spécifiée dans la Grille de données. Cette valeur est enregistrée dans la variable définie dans le paramètre "Variable recevant le résultat". Celle-ci est du DataGridViewRow (System.Windows.Forms.DataGridView)

Code Visual Basic.Net généré par SoftwareZator

Votre_Variable = Votre_Grille.Rows(Index)

Erreurs d'exécutions

Il arrive qu'un programme informatique rencontre un problème durant son exécution. Souvent, un message d'erreur est alors affiché. Le tableau ci-dessous vous permet de déterminer pourquoi l'erreur que vous rencontrez s'affiche et comment la corriger :

Types d'erreurs Descriptions Resolutions
System.ArgumentOutOfRangeException Cette erreur se produit lorsque la valeur du paramètre "Index de l'élément" est inférieur à 0 ou suppérieur au nombre d'élément contenu dans la Grille de données spécifiée. Le problème est donc que vous tentez de récupérer un élément au dessus d'un élément qui n'existe pas. Pour corriger cette erreur, vérifiez que le nombre d'éléments de la Grille de données soit suppérieur ou égale à l'index que vous avez spécifié. Pour cela, utilisez les actions Obtenir le nombre d'éléments d'une Grille de données.
