Action : Vider un Panneau

Cette action permet de supprimer tous les contrôles d'un Panneau spécifiée.

Attention : Assurez-vous que la fenêtre dans laquelle réside votre Panneau est ouverte (affiché à l'écran) au moment où cette action intervient.

Paramètres

Paramètres Descriptions
Panneau à vider Doit être égale au Panneau dont tous les contrôles seront supprimés.

Valeur retournée

Aucune valeur n'est retourné par cette action.

Code Visual Basic.Net généré par SoftwareZator

Si le Panneau se trouve dans la même fenêtre que celle d'où vous utilisez cette action, le code généré sera le suivant :

Me.Votre_Panneau.Controls.Clear()

Si le Panneau se trouve dans une autre fenêtre que celle d'où vous utilisez cette action, le code généré sera le suivant :

Variables.Votre_Fenetre.Votre_Panneau.Controls.Clear()

Erreurs d'exécutions

Aucune erreur d'exécution n'est détecté ici.
