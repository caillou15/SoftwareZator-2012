Action : Obtenir une propriété d'une donnée sérialisable

Cette action permet de récupérer l'une des propriétés d'une variable ayant pour valeur une donnée sérialisable.

Qu'est-ce que la sérialisation : C'est un procéder qui permet d'enregistrer la valeur complète d'une variable, sous une forme crypté, dans un fichier, et de pouvoir la réouvrir tel qu'elle était lors de son enregistrement, après avoir quitté et relancé votre logiciel par exemple.

Paramètres

Paramètres Descriptions
Variable dont une propriété doit être récupérée Doit être égale à la variable dont une propriété de sa valeur sera récupérée. On considère que cette valeur est une donnée sérialisable.
Type de donnée sérialisable contenu dans la variable Doit être égale au type de donnée, spécifique au projet, dont est la valeur de la variable. Pour créer un nouveau type, cliquer sur le bouton "Gestionnaire de types"
Propriété à récupérer Doit être égale à la propriété (change selon le type de donnée sérialisable) que vous voulez récupérer dans la valeur de la variable.
Variable recevant la valeur de la propriété Doit être égale à la variable (non tableau) qui sera égale à la valeur de la propriété spécifiée.

Erreur à ne pas commettre : Faites attention à ce que le type de la valeur de la variable soit égale à celui que vous avez spécifié dans le paramètre "Type de donnée sérialisable contenu dans la variable". Pour vérifier le type d'une valeur, utilisez l'action Obtenir le type d'une variable, dans la catégorie Variables.

Valeur retournée

La valeur retournée est déterminée par la valeur de la propriété sélectionnée dans le paramètre "Propriété à récupérer". Le type de la valeur varie en fonction de la propriété.

Code Visual Basic.Net généré par SoftwareZator

Votre_Recevant_Valeur = CType(Votre_Variable, VelerSoftware_GeneralPlugin_Types.Votre_Type).Votre_Propriete
