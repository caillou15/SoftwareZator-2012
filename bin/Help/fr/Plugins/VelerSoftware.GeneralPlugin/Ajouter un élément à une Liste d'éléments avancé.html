Action : Ajouter un élément à une Liste d'éléments avancé

Cette action permet d'ajouter un nouvel élément à la suite des autres dans une Liste d'éléments avancé.

Attention : Assurez-vous que la fenêtre dans laquelle réside votre Liste d'éléments avancé est ouverte (affiché à l'écran) au moment où cette action intervient.

Paramètres

Paramètres Descriptions
Liste d'éléments avancé concerné Doit être égale à la Liste d'éléments avancé dans laquelle l'élément sera ajouté.
Ajouter l'élément à la suite de la liste En cochant ce paramètre, le nouvel élément sera ajouté à la fin de la Liste d'éléments avancé, tout en bas de celle-ci.
Insérer l'élément à un index spécifié Ce paramètre vous permet de, à la différence du précédent, ajouter l'élément à une place précise dans la liste, autre qu'à la fin. Si vous indiquez l'index 0 par exemple, la valeur sera ajoutée juste avant la toute première valeur de la liste. Si vous indiquez l'index 1 par exemple, la valeur sera ajoutée juste avant la seconde valeur de la liste. On peut illustrer ce paramètre avec l'illustration ci-dessous.
Nom de l'élément à ajouter Saisissez dans ce paramètre le nom de l'élément ajouté à la liste. Ce nom n'est pas visible à l'écran, il permettra avec d'autre action de différencier les éléments entres-eux.
Texte de l'élément Saisissez dans ce paramètre le texte de l'élément qui devra être affiché.
Infobulle de l'élément Saisissez dans ce paramètre le texte de l'élément qui devra être affiché dans l'infobulle de celui-ci.
Numéro de l'image (facultatif) Ce paramètre permet de saisir une valeur numérique définissant le numéro de l'image à afficher dans l'élément. En effet, si une Liste d'images est définie dans la propriété "StateImageList", "LargeImageList" et ou "SmallImageList", et que celle-ci contient des images, elles seront numéroté. Il faut donc indiquer le numéro de l'image que l'on souhaite associé à l'élément.
Ce paramètre ne peut être nul. Pour n'afficher aucune image, indiquez un numéro d'image égale à 0 et ne donnez pas de Liste d'images aux propriétés cité précédemment.
Cocher la case de l'élément Permet de déterminer si la case à cocher de l'élément (si elle est visible) doit être coché ou non.

Erreur à ne pas commettre : Pour les paramètres "Nom de l'élément à ajouter", "Texte de l'élément" et "Infobulle de l'élément", vous ne pouvez pas utiliser des ressources du projet de type Image et de type Autre. Seule des ressources de type Texte peuvent être utilisées.

Erreur à ne pas commettre : Si vous utilisez dans variables dans les paramètres "Nom de l'élément à ajouter", "Texte de l'élément" et ou "Infobulle de l'élément", assurez-vous que le type de la valeur de chacune de ces variables soit du texte (String), ou bien une valeur de type numérique (Integer, Decimal, Double). En effet, seul du texte peut être utilisé dans ces paramètres.
Note : Pour vérifier le type d'une valeur, utilisez l'action Obtenir le type d'une variable, dans la catégorie Variables.

Valeur retournée

Aucune valeur n'est retourné par cette action

Code Visual Basic.Net généré par SoftwareZator

Votre_Liste.Items.Add(VelerSoftware_GeneralPlugin.AddItemListView("Nom de l'élément", "Texte", "Infobulle", Numero_Image, True)

Erreurs d'exécutions

Il arrive qu'un programme informatique rencontre un problème durant son exécution. Souvent, un message d'erreur est alors affiché. Le tableau ci-dessous vous permet de déterminer pourquoi l'erreur que vous rencontrez s'affiche et comment la corriger :

Types d'erreurs Descriptions Resolutions

ou
System.InvalidCastException		 Cette erreur se produit lorsque les valeurs des paramètres "Nom de l'élément à ajouter", "Texte de l'élément" et "Infobulle de l'élément" ne sont pas du texte. Souvent, le problème est causé par une variable utilisé dans ces paramètres et qui ont une valeur autre que du texte. Pour corriger cette erreur, vérifiez que toutes les valeurs impliquées dans ces paramètres sont bien de type Texte (String) et faites les modifications nécessaires pour que ces valeurs soit du texte. Pour cela, utilisez les actions Obtenir le type d'une variable, Convertir la valeur d'une variable
