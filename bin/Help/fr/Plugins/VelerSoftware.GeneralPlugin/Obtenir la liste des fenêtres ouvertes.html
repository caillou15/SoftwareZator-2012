Action : Obtenir la liste des fenêtres ouvertes

Cette action permet de récupérer la liste de toutes les fenêtres ouvertes dans le logiciel.

Information : Sont récupérés : toutes les fenêtres principales du logiciel y compris celle issue de Bibliothèque de fonctions et les fenêtres enfants (qui sont ouverte dans une fenêtre multi-document).

Paramètres

Paramètres Descriptions
Variable tableau reçevant la liste Doit être égale à la variable (tableau) qui sera égale à la liste des fenêtres ouvertes.

Valeur retournée

La valeur retournée est un tableau de valeur. Cette valeur est de type fenêtre (System.Windows.Forms.Form). Elle est égale à l'ensembles des fenêtres ouvertes dans le logiciel.

Remarque importante : A l'exécution de cette action, toutes les anciennes valeurs présentent dans la variable tableau spécifiées seront supprimés.

Information : Sont récupérés : toutes les fenêtres principales du logiciel y compris celle issue de Bibliothèque de fonctions et les fenêtres enfants (qui sont ouverte dans une fenêtre multi-document).

Code Visual Basic.Net généré par SoftwareZator

Votre_Variable_Tableau = VelerSoftware_GeneralPlugin.GetOpenForms()
