Action : Déterminer si un texte se termine par une chaîne spécifiée

Cette action va vous savoir si une chaîne de caractères donnée se termine par une chaîne de caractères spécifiée. En outre, elle permet de savoir si une chaîne X se termine par un texte Y.

Paramètres

Paramètres Descriptions
Texte concerné Doit être égale au texte dont on déterminera s'il se termine par une chaîne spécifiée.
Texte à rechercher Permet de saisir la chaine de caractère qui sera recherchée à la fin du texte concerné.
Variable recevant le résultat Doit être égale à la variable (non tableau) recevant le résultat de la recherche.

Erreur à ne pas commettre : Pour les paramètres "Texte concerné" et "Texte à rechercher", vous ne pouvez pas utiliser des ressources du projet de type Image et de type Autre. Seule des ressources de type Texte peuvent être utilisées.

Erreur à ne pas commettre : Si vous utilisez dans variables dans les paramètres "Texte concerné" et "Texte à rechercher", assurez-vous que le type de la valeur de chacune de ces variables soit du texte (String), ou bien une valeur de type numérique (Integer, Decimal, Double). En effet, seul du texte peut être utilisé dans ces paramètres.
Note : Pour vérifier le type d'une valeur, utilisez l'action Obtenir le type d'une variable, dans la catégorie Variables.

Valeur retournée

La valeur retournée est déterminée par la présence du texte recherché tout à la fin du texte concerné. Elle sera enregistrée dans la variable définit dans le paramètre "Variable recevant le résultat". Cette valeur est de type Vrai/Faux (Boolean).

Résultat du teste Valeurs retournées
Le texte concerné se termine par la chaîne recherchée True
Le texte concerné ne se termine pas par la chaîne recherchée False

Exemple : Soit le texte "Bonjour Mr Dupont". On met en texte recherché "pont". La valeur retourné sera True car la chaîne "Bonjour Mr Dupont" commence par "pont".

Code Visual Basic.Net généré par SoftwareZator

Si le texte recherché est "pont", le code généré sera le suivant :

Votre_Variable = CStr("Bonjour Mr Dupont").EndsWith("pont")

Erreurs d'exécutions

Il arrive qu'un programme informatique rencontre un problème durant son exécution. Souvent, un message d'erreur est alors affiché. Le tableau ci-dessous vous permet de déterminer pourquoi l'erreur que vous rencontrez s'affiche et comment la corriger :

Types d'erreurs Descriptions Resolutions
System.ArgumentException
ou
System.InvalidCastException		 Cette erreur se produit lorsque la valeur du paramètre "Texte concerné" et ou "Texte à rechercher" ne sont pas du texte. Souvent, le problème est causé par une variable utilisé dans ces paramètres et qui ont une valeur autre que du texte. Pour corriger cette erreur, vérifiez que toutes les valeurs impliquées dans ces paramètres sont bien de type Texte (String) et faites les modifications nécessaires pour que ces valeurs soit du texte. Pour cela, utilisez les actions Obtenir le type d'une variable, Convertir la valeur d'une variable
