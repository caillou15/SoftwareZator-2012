Action : Obtenir tous les éléments sélectionnés d'une Liste déroulante

Cette action permet de récupérer tous les éléments sélectionnés existant dans une Liste déroulante.

Attention : Assurez-vous que la fenêtre dans laquelle réside votre Liste déroulante est ouverte (affiché à l'écran) au moment où cette action intervient.

Paramètres

Paramètres Descriptions
Liste déroulante concerné Doit être égale à la Liste déroulante dont il faut récupérer tous les éléments sélectionnés.
Variable tableau recevant le résultat Doit être égale à la variable tableau qui sera égale à la liste des éléments sélectionnés de la Liste déroulante spécifiée dans le paramètre précédent.

Valeur retournée

La valeur retournée est un tableau de valeur. Cette valeur est de type Texte (String). Elle est égale à l'ensembles des éléments contenu la Liste déroulante.
