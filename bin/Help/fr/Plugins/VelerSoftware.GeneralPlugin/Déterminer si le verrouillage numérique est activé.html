Action : Déterminer si le verrouillage numérique est activé

Cette action permet de récupérer une valeur indiquant si le verrouillage numérique (la touche VERR.MAJ) est activé ou non.

Paramètres

Paramètres Descriptions
Variable recevant le résultat Définit la variable recevant la valeur qui indique si le verrouillage est activé ou non.

Valeur retournée

La valeur retournée est déterminée par le status du verrouillage numérique de l'ordinateur. Elle sera enregistrée dans la variable définit dans le paramètre "Variable recevant le résultat". Cette valeur est de type Vrai/Faux (Boolean).

Etat du verrouillage Valeurs retournées
Activé True
Désactivé False

Code Visual Basic.Net généré par SoftwareZator

Votre_Variable = System.Console.CapsLock
