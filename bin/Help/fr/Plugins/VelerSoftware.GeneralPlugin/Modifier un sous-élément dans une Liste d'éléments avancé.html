Action : Modifier un sous-élément dans une Liste d'éléments avancé

Cette action permet de modifier un sous-élément dans un élément existant dans dans une Liste d'éléments avancé.

Attention : Assurez-vous que la fenêtre dans laquelle réside votre Liste d'éléments avancé est ouverte (affiché à l'écran) au moment où cette action intervient.

Paramètres

Paramètres Descriptions
Variable contenant l'élément concerné du Liste d'éléments avancé Doit être égale à la varible (non tableau) dont la valeur est un élément d'une Liste d'éléments avancé. Pour qu'une variable soit égale à un élément d'une Liste d'éléments avancé, utilisez l'action Obtenir un élément d'une Liste d'éléments avancé, dans la catégorie Liste d'éléments avancé.
Index du sous-élément à modifier Ce paramètre doit être égale à l'index du sous-élément que vous voulez modifier. Autrement dit, la place de ce sous-élément dans l'élément. Si vous indiquez l'index 0 par exemple, la sous-élément modifié sera le tout premier, dans la première colonne. Si vous indiquez l'index 1 par exemple, la sous-élément modifié sera le second sous-élément. On peut illustrer ce paramètre avec l'illustration ci-dessous.
Nouveau texte du sous-élément Saisissez dans ce paramètre le texte qui sera affiché dans le sous-élément spécifié.

Erreur à ne pas commettre : Faites attention à ce que le type de la valeur de la variable spécifiée dans le paramètre "Variable contenant l'élément concerné du Liste d'éléments avancé" soit égale à ListViewItem (System.Windows.Forms.ListViewItem). Pour vérifier le type d'une valeur, utilisez l'action Obtenir le type d'une variable, dans la catégorie Variables.

Erreur à ne pas commettre : Pour le paramètre "Nouvea texte du sous-élément à ajouter", vous ne pouvez pas utiliser des ressources du projet de type Image et de type Autre. Seule des ressources de type Texte peuvent être utilisées.

Erreur à ne pas commettre : Si vous utilisez dans variables dans le paramètre "Nouveau texte du sous-élément à ajouter", assurez-vous que le type de la valeur de chacune de ces variables soit du texte (String), ou bien une valeur de type numérique (Integer, Decimal, Double). En effet, seul du texte peut être utilisé dans ce paramètre.
Note : Pour vérifier le type d'une valeur, utilisez l'action Obtenir le type d'une variable, dans la catégorie Variables.

Valeur retournée

Aucune valeur n'est retourné par cette action

Code Visual Basic.Net généré par SoftwareZator

Si le texte est "Bonjour le monde", le code généré sera le suivant :

Votre_Variable.SubItems(Index).Text = "Bonjour le monde"

Erreurs d'exécutions

Il arrive qu'un programme informatique rencontre un problème durant son exécution. Souvent, un message d'erreur est alors affiché. Le tableau ci-dessous vous permet de déterminer pourquoi l'erreur que vous rencontrez s'affiche et comment la corriger :

Types d'erreurs Descriptions Resolutions
System.InvalidCastException Cette erreur se produit lorsque la valeur du paramètre "Nouveau texte du sous-élément" n'est pas du texte, ou que la valeur du paramètre "Variable contenant l'élément concerné du Liste d'éléments avancé" n'est pas ListViewItem (System.Windows.Forms.ListViewItem). Souvent, le problème est causé par une variable utilisé dans ces paramètres et qui ont une valeur incompatible. Pour corriger cette erreur, vérifiez que toutes les valeurs impliquées dans ce paramètre soit bien de type Texte (String) ou ListViewItem (System.Windows.Forms.ListViewItem) (selon le paramètre) et faites les modifications nécessaires pour que ces valeurs soit du compatible. Pour cela, utilisez les actions Obtenir le type d'une variable, Convertir la valeur d'une variable
System.ArgumentOutOfRangeException Cette erreur se produit lorsque la valeur du paramètre "Index du sous-élément à modifier" est inférieur à 0 ou suppérieur au nombre de sous-élément contenu dans l'élément spécifiée. Le problème est donc que vous tentez de modifier un sous-élément au dessus d'un sous-élément qui n'existe pas. Pour corriger cette erreur, vérifiez que le nombre de sous-éléments soit suppérieur ou égale à l'index que vous avez spécifié.
