Action : Ajouter un élément à une Liste d'éléments

Cette action permet d'ajouter un nouvel élément à la suite des autres dans une Liste d'éléments.

Attention : Assurez-vous que la fenêtre dans laquelle réside votre Liste d'éléments est ouverte (affiché à l'écran) au moment où cette action intervient.

Paramètres

Paramètres Descriptions
Liste d'éléments concerné Doit être égale à la Liste d'éléments dans laquelle l'élément sera ajouté.
Ajouter la valeur à la suite de la liste En cochant ce paramètre, la nouvelle valeur sera ajoutée à la fin de la Liste d'éléments, tout en bas de celle-ci.
Insérer la valeur à un index spécifié Ce paramètre vous permet de, à la différence du précédent, ajouter la valeur à une place dans la liste, autre qu'à la fin. Si vous indiquez l'index 0 par exemple, la valeur sera ajoutée juste avant la toute première valeur de la liste. Si vous indiquez l'index 1 par exemple, la valeur sera ajouter juste avant la seconde valeur de la liste. On peut illustrer ce paramètre avec l'illustration ci-dessous.
Texte de l'élément à ajouter Saisissez dans ce paramètre le texte qui sera affiché dans l'élément ajouté à la liste.

Erreur à ne pas commettre : Pour le paramètre "Texte de l'élément à ajouter", vous ne pouvez pas utiliser des ressources du projet de type Image et de type Autre. Seule des ressources de type Texte peuvent être utilisées.

Erreur à ne pas commettre : Si vous utilisez dans variables dans le paramètre "Texte de l'élément à ajouter", assurez-vous que le type de la valeur de chacune de ces variables soit du texte (String), ou bien une valeur de type numérique (Integer, Decimal, Double). En effet, seul du texte peut être utilisé dans ce paramètre.
Note : Pour vérifier le type d'une valeur, utilisez l'action Obtenir le type d'une variable, dans la catégorie Variables.

Valeur retournée

Aucune valeur n'est retourné par cette action

Code Visual Basic.Net généré par SoftwareZator

Si la Liste d'éléments se trouve dans la même fenêtre que celle d'où vous utilisez cette action, que le texte est "Bonjour le monde", le code généré sera le suivant :

Me.Votre_Liste.Items.Add("Bonjour le monde")

Si la Liste d'éléments se trouve dans une autre fenêtre que celle d'où vous utilisez cette action, que le texte est "Bonjour le monde", le code généré sera le suivant :

Variables.Votre_Fenetre.Votre_Liste.Items.Add("Bonjour le monde")

Types d'erreurs Descriptions Resolutions

ou
System.InvalidCastException		 Cette erreur se produit lorsque la valeur du paramètre "Texte de l'élément à ajouter" n'est pas du texte. Souvent, le problème est causé par une variable utilisé dans ces paramètres et qui ont une valeur autre que du texte. Pour corriger cette erreur, vérifiez que toutes les valeurs impliquées dans ce paramètre soit bien de type Texte (String) et faites les modifications nécessaires pour que ces valeurs soit du texte. Pour cela, utilisez les actions Obtenir le type d'une variable, Convertir la valeur d'une variable
