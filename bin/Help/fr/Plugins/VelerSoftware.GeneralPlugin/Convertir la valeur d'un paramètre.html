Action : Convertir la valeur d'un paramètre

Cette action permet de convertir la valeur d'un paramètre en un autre type de valeur.

Exemple : Si la valeur du paramètre est de type numérique (Integer, Decimal, Double), vous pourrez la convertir en texte (String).

Paramètres

Paramètres Descriptions
Paramètre à convertir Doit être égale au paramètre dont la valeur sera convertie.
Convertir en une valeur de type Doit être égale au type dans lequel la valeur du paramètre sera convertie.
Variable recevant le résultat Doit être égale à la variable (non tableau) recevant la nouvelle valeur issue de la convertion.

Erreur à ne pas commettre : Attention à ce que la valeur du paramètre soit bien compatible dans le type spécifié. Il est courant que des erreurs d'exécutions de produises car la valeur ne peut pas être convertie dans le type définit. Par exemple, une valeur de type "Form" ne peut pas être convertie en "Valeur Vrai/Faux", l'ordinateur ne comprendrait pas ce qu'il faut convertir et retournerai une erreur.
Note : Une valeur numérique égale à "0" ou "1" peut être convertie en "Valeur Vrai/Faux". Une valeur numérique peut être transformé en texte, mais pas dans le sens contraire, à moins que la chaîne à convertir ne possède que des chiffres. A vous de faire attention à ce genre de petits détails durant l'élaboration de votre projet et n'hésitez pas à vous aidez du forum de Veler Software.

Valeur retournée

La valeur retournée est déterminée par le type sélectionné dans le paramètre "Convertir en une valeur de typee".

Code Visual Basic.Net généré par SoftwareZator

Votre_Recevant_Valeur = CType(Votre_Paramètre, Votre_Type)

Erreurs d'exécutions

Il arrive qu'un programme informatique rencontre un problème durant son exécution. Souvent, un message d'erreur est alors affiché. Le tableau ci-dessous vous permet de déterminer pourquoi l'erreur que vous rencontrez s'affiche et comment la corriger :

Types d'erreurs Descriptions Resolutions
System.InvalidCastException Cette erreur se produit lorsque la valeur du paramètre spécifié dans le paramètre "Paramètre à convertir" n'est pas convertissable dans le type spécifié dans le paramètre "Convertir en une valeur de type". Pour corriger cette erreur, essayez de convertir votre paramètre dans un autre type.
