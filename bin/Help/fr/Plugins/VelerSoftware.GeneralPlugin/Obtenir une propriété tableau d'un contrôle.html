Action : Obtenir une propriété tableau d'un contrôle

Cette action permet de récupérer la valeur d'une propriété dite "tableau", c'est-à-dire une propriété, d'un contrôle, qui peut avoir plusieurs valeurs à la fois, tel que la liste des différents éléments d'une liste.

Exemple : Si le contrôle est de type Button (System.Windows.Forms.ListBox), vous pourrez récupérer toutes les propriétés tableau issue de ce type, à savoir, Items, SelectedItems, Controls, etc.

Paramètres

Paramètres Descriptions
Contrôle dont il faut récupérer une propriété tableau Doit être égale au contrôle dont il faut récupérer la valeur d'une propriété tableau spécifiée dans les paramètres suivants.
Propriété tableau à récupérer (selon le type) Permet de choisir la propriété tableau du contrôle à récupérer.
Variable tableau recevant le résultat Doit être égale à la variable tableau recevant la valeur de la propriété spécifiée dans le paramètre précédent.

Valeur retournée

La valeur retournée est déterminée par la valeur de la propriété sélectionnée dans le paramètre "Propriété à récupérer (selon le type)". Le type de la valeur varie en fonction de la propriété.

Code Visual Basic.Net généré par SoftwareZator

Si la valeur de la propriété est une collection, le code généré sera le suivant :

Votre_Variable_Tableau = New System.Collections.Generic.List(Of Object)(System.Linq.Enumerable.Cast(Of Object)(Votre_Contrôle.Votre_Propriete))

Si la valeur de la propriété est un tableau, le code généré sera le suivant :

Votre_Variable_Tableau = New System.Collections.Generic.List(Of Object)(System.Linq.Enumerable.Cast(Of Object)(System.Linq.Enumerable.AsEnumerable(Votre_Contrôle.Votre_Propriete)))
