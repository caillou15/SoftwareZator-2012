Action : Déterminer si une valeur existe dans une variable tableau

Cette action permet de déterminer si une valeur spécifié est présente dans une variable tableau définie.

Paramètres

Paramètres Descriptions
Variable tableau à tester Doit être égale à la variable tableau dans laquelle il faudra vérifier si la valeur spécifiée ci-dessous est présente.
Valeur à vérifier Permet de saisir la valeur dont on doit vérifier l'existance dans la variable tableau spécifiée dans le paramètre précédent. Vous pouvez saisir une valeur textuel, numérique, ou encore du Visual Basic.Net, une couleur et des types particulier.
Variable recevant le résultat Doit être égale à la variable (non tableau) recevant un indice spécifiant si la valeur à vérifier existe dans la variable tableau testée.

Valeur retournée

La valeur retournée est déterminée par la présence de la valeur à vérifier dans la variable tableau spécifiée. Elle sera enregistrée dans la variable définit dans le paramètre "Variable recevant le résultat". Cette valeur est de type Vrai/Faux (Boolean).

Existance Valeurs retournées
La valeur existe True
La valeur n'existe pas False

Code Visual Basic.Net généré par SoftwareZator

Votre_Variable = Votre_Variable_Tableau.Contains(Votre_Valeur)
