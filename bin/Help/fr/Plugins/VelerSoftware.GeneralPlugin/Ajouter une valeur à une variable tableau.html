Action : Ajouter une valeur à une variable tableau

Cette action permet d'ajouter ou d'insérer une nouvelle valeur dans une variable déclaré comme étant un tableau de valeurs.

Paramètres

Paramètres Descriptions
Variable tableau à laquelle il faut ajouter une variable Doit être égale à la variable tableau pour laquelle une valeur sera ajoutée.
Ajouter la valeur à la suite du tableau En cochant ce paramètre, la nouvelle valeur sera ajoutée à la fin du tableau, tout en bas de celui-ci.
Insérer la valeur à un index spécifié Ce paramètre vous permet de, à la différence du précédent, ajouter la valeur à une place dans le tableau, autre qu'à la fin. Si vous indiquez l'index 0 par exemple, la valeur sera ajoutée juste avant la toute première valeur du tableau. Si vous indiquez l'index 1 par exemple, la valeur sera ajouter juste avant la seconde valeur du tableau. On peut illustrer ce paramètre avec l'illustration ci-dessous.
Valeur à ajouter Permet de saisir la nouvelle valeur à ajouter dans la variable. Vous pouvez saisir une valeur textuel, numérique, ou encore du Visual Basic.Net, une couleur et des types particulier.

Valeur retournée

Aucune valeur n'est retourné par cette action.

Code Visual Basic.Net généré par SoftwareZator

Si le paramètre "Ajouter la valeur à la suite du tableau" est coché, le code généré sera le suivant :

Votre_Variable.Add(Votre_Valeur)

Si le paramètre "Insérer la valeur à un index spécifié" est coché, le code généré sera le suivant :

Votre_Variable.Insert(Votre_Index, Votre_Valeur)

Erreurs d'exécutions

Il arrive qu'un programme informatique rencontre un problème durant son exécution. Souvent, un message d'erreur est alors affiché. Le tableau ci-dessous vous permet de déterminer pourquoi l'erreur que vous rencontrez s'affiche et comment la corriger :

Types d'erreurs Descriptions Resolutions
System.ArgumentOutOfRangeException Cette erreur se produit lorsque la valeur du paramètre "Insérer la valeur à un index spécifié" est inférieur à 0 ou suppérieur au nombre de valeurs contenu dans la variable tableau spécifié. Le problème est donc que vous tentez d'insérer un élément au dessus d'un élément qui n'existe pas. Pour corriger cette erreur, vérifiez que le nombre de valeur de la variable tableau soit suppérieur ou égale à l'index que vous avez spécifié. Pour cela, utilisez les actions Obtenir le nombre de valeurs d'une variable tableau.
