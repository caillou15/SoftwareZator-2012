Action : Modifier une propriété d'un paramètre

Cette action permet de modifier la valeur d'une propriété d'un paramètre possèdant une valeur de type spécifié.

Exemple : Si la valeur du paramètre est de type Button (System.Windows.Forms.Button), vous pourrez modifier toutes les propriétés issue de ce type, à savoir, Text, BackColor, Size, etc.

Paramètres

Paramètres Descriptions
Paramètre dont il faut modifier une propriété Doit être égale au paramètre dont il faut modifier la valeur d'une propriété spécifiée dans les paramètres suivants.
Type de la valeur du paramètre Doit être égale au type de la valeur du paramètre. Veillez à indiquer la bonne, dans le cas contraire, des erreurs d'exécutions de votre projet se produiront.
Propriété à modifier (selon le type) Permet de choisir la propriété du paramètre à modifier.
Nouvelle valeur de la propriété Permet de saisir la nouvelle valeur de la propriété spécifié. Vous pouvez saisir une valeur textuel, numérique, ou encore du Visual Basic.Net, une couleur et des types particulier. Veillez à ce que votre valeur soit du même type que celui indiqué par le paramètre "Propriété à modifier".

Erreur à ne pas commettre : Attention à ce que le type de la valeur du paramètre soit le bon. Dans le cas contraire, une erreur d'exécution sera rencontré.
Note : Pour vérifier le type d'une valeur, utilisez l'action Obtenir le type d'une variable, dans la catégorie Variables.

Valeur retournée

Aucune valeur n'est retourné par cette action

Code Visual Basic.Net généré par SoftwareZator

CType(Votre_Paramètre, Votre_Type).Votre_Propriete = Nouvelle_Valeur

Erreurs d'exécutions

Il arrive qu'un programme informatique rencontre un problème durant son exécution. Souvent, un message d'erreur est alors affiché. Le tableau ci-dessous vous permet de déterminer pourquoi l'erreur que vous rencontrez s'affiche et comment la corriger :

Types d'erreurs Descriptions Resolutions
System.InvalidCastException Cette erreur se produit lorsque la valeur du paramètre "Nouvelle valeur de la propriété" est une valeur d'un type différent de celui de la propriété spécifié dans le paramètre "Propriété à modifier" et que celui-ci n'est pas compatible et convertible avec votre valeur. Pour corriger cette erreur, vérifiez que toutes les valeurs impliquées dans ce paramètre sont bien du même type que celui du paramètre "Propriété à modifier" et faites les modifications nécessaires pour que ces valeurs soit compatible. Pour cela, utilisez les actions Obtenir le type d'une variable, Convertir la valeur d'une variable
