Action : Obtenir le nombre de contrôles dans un Panneau de configuration de débit

Cette action permet de récupérer le nombre de contrôles qui est contenu par le Panneau de configuration de débit spécifiée.

Attention : Assurez-vous que la fenêtre dans laquelle réside votre Panneau de configuration de débit est ouverte (affiché à l'écran) au moment où cette action intervient.

Paramètres

Paramètres Descriptions
Panneau de configuration de débit concerné Doit être égale à le Panneau de configuration de débit dont il faut récupérer le nombre de contrôles.
Variable recevant le résultat Doit être égale à la variable (non tableau) recevant le nombre de contrôles.

Valeur retournée

La valeur retournée est déterminée par le nombre de contrôles contenus dans le Panneau de configuration de débit. Cette valeur est enregistré dans la variable définie dans le paramètre "Variable recevant le résultat". Celle-ci est un nombre entier (Integer)

Important : Si la valeur retourné est -1, cela signifie qu'il n'y a aucun contrôle dans le Panneau de configuration de débit.

Code Visual Basic.Net généré par SoftwareZator

Votre_Variable = Votre_Panneau.Controls.Count
