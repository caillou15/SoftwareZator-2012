Action : Obtenir le nom complet de Windows

Cette action permet de récupérer une valeur égale au nom complet de la version du système d'exploitation Windows sur lequel est exécuté votre programme.

Exemple : Microsoft Windows 7 Édition Familiale Premium

Paramètres

Paramètres Descriptions
Variable recevant le résultat Définit la variable recevant le nom complet de Windows.

Valeur retournée

La valeur retournée est déterminée par la version et l'édition de Windows. Elle sera enregistrée dans la variable définit dans le paramètre "Variable recevant le résultat". Cette valeur est de type texte (String).
