Action : Transformer une date en un texte

Cette action permet de convertir une valeur de type Date en un texte compréhensible par l'humain et par l'ordinateur (avec l'action Obtenir une date à partir d'un texte).

Paramètres

Paramètres Descriptions
Date à convertir Ce paramètre permet de définir la date qui sera transformée en texte. Assurez-vous que la valeur que vous définirez soit une Date (System.DateTime)
Format du texte Permet de définir le format du texte qui sera récupéré :
  • JJ/MM/AAAA
  • Lundi 24 Juin 1995
Variable recevant le résultat Doit être égale à la variable (non tableau) recevant le texte.

Erreur à ne pas commettre : Si vous utilisez une variable dans le paramètre "Date à convertir", assurez-vous que le type de la valeur de cette variable soit une Date (System.DateTime).
Note : Pour vérifier le type d'une valeur, utilisez l'action Obtenir le type d'une variable, dans la catégorie Variables.

Valeur retournée

La valeur retournée est déterminée le format de texte spécifié et de la date spécifiée. Cette valeur est enregistré dans la variable définie dans le paramètre "Variable recevant le résultat". Celle-ci est de type texte (String)

Code Visual Basic.Net généré par SoftwareZator

Si le format de texte est "JJ/MM/AAAA", le code généré sera le suivant :

Votre_Variable = Votre_Date.ToShortDateString

Dans le cas contraire, le code généré sera le suivant :

Votre_Variable = Votre_Date.ToShortLongString

Erreurs d'exécutions

Il arrive qu'un programme informatique rencontre un problème durant son exécution. Souvent, un message d'erreur est alors affiché. Le tableau ci-dessous vous permet de déterminer pourquoi l'erreur que vous rencontrez s'affiche et comment la corriger :

Types d'erreurs Descriptions Resolutions
System.ArgumentException
ou
System.InvalidCastException		 Cette erreur se produit lorsque la valeur du paramètre "Date à convertir" n'est pas une date. Souvent, le problème est causé par une variable utilisé dans ce paramètre et qui ont une valeur autre qu'une date. Pour corriger cette erreur, vérifiez que toutes les valeurs impliquées dans ce paramètre soit bien une Date (System.DateTime) et faites les modifications nécessaires pour que cette valeur soit une date. Pour cela, utilisez les actions Obtenir le type d'une variable, Convertir la valeur d'une variable
