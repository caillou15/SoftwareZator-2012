Action : Vider une variable tableau

Cette action permet de supprimer toutes les valeurs d'une variable déclaré comme étant un tableau de valeurs.

Paramètres

Paramètres Descriptions
Variable tableau à vider Doit être égale à la variable tableau pour laquelle il faut supprimer toutes les valeurs.

Valeur retournée

Aucune valeur n'est retourné par cette action.
