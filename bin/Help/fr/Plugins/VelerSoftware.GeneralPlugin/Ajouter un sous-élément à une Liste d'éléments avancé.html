Action : Ajouter un sous-élément à une Liste d'éléments avancé

Cette action permet d'ajouter un nouveau sous-élément à un élément existant dans dans une Liste d'éléments avancé.

Attention : Assurez-vous que la fenêtre dans laquelle réside votre Liste d'éléments avancé est ouverte (affiché à l'écran) au moment où cette action intervient.

Paramètres

Paramètres Descriptions
Liste d'éléments avancé concerné Doit être égale à la Liste d'éléments avancé dans laquelle un sous-élément sera ajouté.
Index de l'élément auquel il faut ajouter un sous-élément Ce paramètre doit être égale à l'index de l'élément auquel vous voulez ajouter un sous-élément. Autrement dit, la place de cet élément dans la liste. Si vous indiquez l'index 0 par exemple, le sous-élément sera ajouté juste avant le tout premier de la liste. Si vous indiquez l'index 1 par exemple, le sous-élément ajouté sera placé juste avant le second élément de la liste. On peut illustrer ce paramètre avec l'illustration ci-dessous.
Ajouter la valeur à la suite des autres sous-éléments En cochant ce paramètre, la nouveau sous-élément sera ajouté à la fin de la liste des autre sous-éléments, tout à droite des autres.
Insérer la valeur à un index spécifié Ce paramètre vous permet de, à la différence du précédent, ajouter le sous-élément à une place dans la liste, autre qu'à la fin. Si vous indiquez l'index 0 par exemple, la valeur sera ajoutée juste avant la toute première de la liste. Si vous indiquez l'index 1 par exemple, la valeur sera ajoutée juste avant la seconde de la liste. On peut illustrer ce paramètre avec l'illustration ci-dessous.
Texte du sous-élément à ajouter Saisissez dans ce paramètre le texte qui sera affiché dans le sous-élément ajouté à l'élément spécifié.

Erreur à ne pas commettre : Pour le paramètre "Texte du sous-élément à ajouter", vous ne pouvez pas utiliser des ressources du projet de type Image et de type Autre. Seule des ressources de type Texte peuvent être utilisées.

Erreur à ne pas commettre : Si vous utilisez dans variables dans le paramètre "Texte du sous-élément à ajouter", assurez-vous que le type de la valeur de chacune de ces variables soit du texte (String), ou bien une valeur de type numérique (Integer, Decimal, Double). En effet, seul du texte peut être utilisé dans ce paramètre.
Note : Pour vérifier le type d'une valeur, utilisez l'action Obtenir le type d'une variable, dans la catégorie Variables.

Valeur retournée

Aucune valeur n'est retourné par cette action

Code Visual Basic.Net généré par SoftwareZator

Si la Liste d'éléments avancé se trouve dans la même fenêtre que celle d'où vous utilisez cette action, que le texte est "Bonjour le monde", le code généré sera le suivant :

Me.Votre_Liste.Items(Index).SubItems.Add("Bonjour le monde")

Si la Liste d'éléments avancé se trouve dans une autre fenêtre que celle d'où vous utilisez cette action, que le texte est "Bonjour le monde", le code généré sera le suivant :

Variables.Votre_Fenetre.Votre_Liste.Items(Index).SubItems.Add("Bonjour le monde")

Erreurs d'exécutions

Il arrive qu'un programme informatique rencontre un problème durant son exécution. Souvent, un message d'erreur est alors affiché. Le tableau ci-dessous vous permet de déterminer pourquoi l'erreur que vous rencontrez s'affiche et comment la corriger :

Types d'erreurs Descriptions Resolutions
System.InvalidCastException Cette erreur se produit lorsque la valeur du paramètre "Texte du sous-élément à ajouter" n'est pas du texte. Souvent, le problème est causé par une variable utilisé dans ces paramètres et qui ont une valeur autre que du texte. Pour corriger cette erreur, vérifiez que toutes les valeurs impliquées dans ce paramètre soit bien de type Texte (String) et faites les modifications nécessaires pour que ces valeurs soit du texte. Pour cela, utilisez les actions Obtenir le type d'une variable, Convertir la valeur d'une variable
System.ArgumentOutOfRangeException Cette erreur se produit lorsque la valeur du paramètre "Index de l'élément auquel il faut ajouter un sous-élément" est inférieur à 0 ou suppérieur au nombre d'élément contenu dans la Liste d'éléments avancé spécifiée. Le problème est donc que vous tentez de récupérer un élément au dessus d'un élément qui n'existe pas. Pour corriger cette erreur, vérifiez que le nombre d'éléments de la Liste d'éléments avancé soit suppérieur ou égale à l'index que vous avez spécifié. Pour cela, utilisez les actions Obtenir le nombre d'éléments d'une Liste d'éléments avancé.
