Action : Modifier une variable

Cette action permet de modifier la valeur d'une variable.

Information : Pour modifier une variable tableau, utilisez les actions Ajouter une valeur à une variable tableau, Vider une variable tableau, Supprimer une valeur à une variable tableau.

Paramètres

Paramètres Descriptions
Variable à modifier Doit être égale à la variable (non tableau) pour laquelle la valeur sera modifiée.
Nouvelle valeur Permet de saisir la nouvelle valeur de la variable. Vous pouvez saisir une valeur textuel, numérique, ou encore du Visual Basic.Net, une couleur et des types particulier.

Valeur retournée

La valeur retournée est déterminée par la nouvelle valeur donnée. Elle sera enregistrée dans la variable définit dans le paramètre "Variable à modifier". Cette valeur est de type texte (String), numérique (Integer, Decimal, Double), couleur (System.Drawing.Color) ou autre selon le contenu.

Code Visual Basic.Net généré par SoftwareZator

Votre_Variable = Votre valeur
