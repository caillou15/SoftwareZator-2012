Action : Modifier une variable

Cette action permet de modifier la valeur d'une propriété (créée avec l'action Déclarer une propriété, dans la zone de déclaration (premier onglet de l'éditeur de fonctions)).

Paramètres

Paramètres Descriptions
Propriété à modifier Doit être égale à la propriété pour laquelle la valeur sera modifiée.
Nouvelle valeur Permet de saisir la nouvelle valeur de la propriété. Vous pouvez saisir une valeur textuel, numérique, ou encore du Visual Basic.Net, une couleur et des types particulier.

Valeur retournée

La valeur retournée est déterminée par la nouvelle valeur donnée. Elle sera enregistrée dans la propriété définit dans le paramètre "Propriété à modifier". Cette valeur est de type texte (String), numérique (Integer, Decimal, Double), couleur (System.Drawing.Color) ou autre selon le contenu.

Code Visual Basic.Net généré par SoftwareZator

Votre_Propriété = Votre valeur
