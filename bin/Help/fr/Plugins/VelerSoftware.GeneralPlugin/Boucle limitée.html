Action : Boucle limitée

Cette action permet de créer une boucle limitée. Une boucle limitée permet d'exécuter, un nombre de fois définit, un ensemble d'actions. Elle exécute cet ensemble N fois, à moins qu'on ne lui a pas demandé de s'ârreter avant. On peut illustrer une boucle avec le schéma ci-contre.

Conseil : Pour arrêter (sortir) une boucle limitée avant la fin de celle-ci, utilisez l'action Sortir d'une boucle.

Quel est la différence avec la Boucle : Une boucle (simple) s'exécute jusqu'à ce qu'on lui donne l'ordre de s'arrêter. Une boucle limitée, elle, s'exécute un nombre de fois donné. On peut ainsi, par exemple, exécuter la boucle 10 fois avant qu'elle ne s'arrête toute seule.

Comment fonctionne une boucle limitée?

Cette action peut paraitre difficile d'utilisation au premier regard mais une fois que l'on a compris le principe cela devient un jeu d'enfant.

Tout d'abord, la différence entre l'action Boucle et la Boucle limitée est qu'une boucle simple s'exécute indéfiniment. Au contraire, une boucle limitée s'exécute un nombre de fois bien précit. En effet, vous pouvez demander à une boucle limitée de s'exécuter 3 fois, 10 fois, 100 fois [...] N fois.

Une boucle limitée repose donc sur le principe suivant : on part du nombre X, et on va jusqu'au nombre N.

Notez que dans une boucle limitée, la valeur X, qui est la valeur de départ, s'incrément à chaque fois que la boucle est exécuté.

Exemple : On part de X = 0 jusqu'à N = 3.
Première exécution : X = 0
Seconde exécution : X = 1
Troisième exécution : X = 2

Vous pourrez remarquer que X n'atteind pas 3, car X est partie de 0.
On peut donc en déduire la formule suivante : Nombre d'exécution de la boucle limitée = (N - X)

Exemple : On part de X = 0 jusqu'à N = 10.
(N - X) = (10 - 0) = 10
La boucle va donc s'exécuter 10 fois.

Exemple : On part de X = 25 jusqu'à N = 30.
(N - X) = (30 - 25) = 5
La boucle va donc s'exécuter 5 fois.

Enfin, la dernière connaissance à acquérir est le pas d'avancement. Jusqu'ici dans les exemples précédents, vous avez pu constater que X s'incrémentait de 1 à chaque exécution de la boucle. Le pas d'avancement d'une boucle est l'incrémentation à effectué à chaque exécution. On en peut ainsi imaginer l'exemple suivant :

Exemple : On part de X = 0 jusqu'à N = 6 avec un pas d'avancement égale à 2.
Première exécution : X = 0
Seconde exécution : X = 2
Troisième exécution : X = 4
Note : il n'y a pas de quatrième exécution, car dès que X atteind 6, la boucle se termine.

Paramètres

Paramètres Descriptions
Commencer la boucle à partir de la variable Définit la variable (non tableau) possédant la valeur de départ de la boucle. Cette variable sera incrémenté du pas d'avancement définit dans les paramètres "Pas (facultatif)" (voir les exemples ci-dessus).
Ayant pour valeur Permet de choisir la valeur de la variable définit dans le paramètre précédent. Cette valeur peut être égale à un nombre constant, ou bien à la valeur d'une autre variable (non tableau).
Jusqu'à Permet de choisir la valeur de fin de la boucle (voir les exemples ci-dessus). Cette valeur peut être égale à un nombre constant, ou bien à la valeur d'une autre variable (non tableau), ou encore au nombre de valeurs contenu dans une variable tableau.
Pas (facultatif) Permet de choisir la valeur du pas d'avancement (par défaut, 1) de la boucle limitée (voir les exemples ci-dessus). Cette valeur peut être égale à un nombre constant, ou bien à la valeur d'une autre variable (non tableau).

Erreur à ne pas commettre : Pour les paramètres "Ayant pour valeur", "Jusqu'à" et "Pas (facultatif)", faites attention si vous utilisez des variables à ce que celles-ci possèdent une valeur de type numérique (Integer, Decimal, Double) ou bien que la variable soit de type tableau.
Note : Pour vérifier le type d'une valeur, utilisez l'action Obtenir le type d'une variable, dans la catégorie Variables.

Valeur retournée

Aucune valeur n'est retourné par cette action

Code Visual Basic.Net généré par SoftwareZator

Code généré pour une boucle limitée qui part de X = 0 jusqu'à N avec un pas égale à 1 :

X = 0
Do While (X < N)

	X = (X + 1)
Loop
