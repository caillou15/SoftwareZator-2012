Action : Modifier une valeur dans une variable tableau

Cette action permet de modifier une valeur existante dans une variable déclaré comme étant un tableau de valeurs.

Paramètres

Paramètres Descriptions
Variable tableau à laquelle il faut modifier une variable Doit être égale à la variable tableau pour laquelle une valeur sera modifiée.
Index de la valeur à modifier Ce paramètre doit être égale à l'index de la valeur que vous voulez modifier dans la variable tableau. Autrement dit, la place de cette valeur dans la variable. Si vous indiquez l'index 0 par exemple, la valeur modifiée sera la toute première valeur du tableau. Si vous indiquez l'index 1 par exemple, la valeur modifiée sera celle juste avant la seconde valeur du tableau. On peut illustrer ce paramètre avec l'illustration ci-dessous.
Nouvelle valeur Permet de saisir la nouvelle valeur dans la variable. Vous pouvez saisir une valeur textuel, numérique, ou encore du Visual Basic.Net, une couleur et des types particulier.

Valeur retournée

Aucune valeur n'est retourné par cette action.

Code Visual Basic.Net généré par SoftwareZator

Votre_Variable(Index) = Nouvelle_Valeur

Erreurs d'exécutions

Il arrive qu'un programme informatique rencontre un problème durant son exécution. Souvent, un message d'erreur est alors affiché. Le tableau ci-dessous vous permet de déterminer pourquoi l'erreur que vous rencontrez s'affiche et comment la corriger :

Types d'erreurs Descriptions Resolutions
System.ArgumentOutOfRangeException Cette erreur se produit lorsque la valeur du paramètre "Index de la valeur à modifier" est inférieur à 0 ou suppérieur au nombre de valeurs contenu dans la variable tableau spécifié. Le problème est donc que vous tentez de modifier un élément au dessus d'un élément qui n'existe pas. Pour corriger cette erreur, vérifiez que le nombre de valeur de la variable tableau soit suppérieur ou égale à l'index que vous avez spécifié. Pour cela, utilisez les actions Obtenir le nombre de valeurs d'une variable tableau.
