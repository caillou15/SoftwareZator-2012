Action : Obtenir tous les éléments sélectionnés d'une Liste d'éléments avancé

Cette action permet de récupérer tous les éléments sélectionnés existant dans une Liste d'éléments avancé.

Attention : Assurez-vous que la fenêtre dans laquelle réside votre Liste d'éléments avancé est ouverte (affiché à l'écran) au moment où cette action intervient.

Paramètres

Paramètres Descriptions
Liste d'éléments avancé concerné Doit être égale à la Liste d'éléments avancé dont il faut récupérer tous les éléments sélectionnés.
Variable tableau recevant le résultat Doit être égale à la variable tableau qui sera égale à la liste des éléments sélectionnés de la Liste d'éléments avancé spécifiée dans le paramètre précédent.

Valeur retournée

La valeur retournée est un tableau de valeur. Cette valeur est de type ListViewItem (System.Windows.Forms.ListViewItem). Elle est égale à l'ensembles des éléments contenu la Liste d'éléments avancé.

Code Visual Basic.Net généré par SoftwareZator

Votre_Variable_Tableau = New System.Collections.Generic.List(Of Object)(System.Linq.Enumerable.Cast(Of Object)(Votre_Liste.SelectedItems))
