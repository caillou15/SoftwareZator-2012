Action : Obtenir une propriété d'une propriété

Cette action permet de récupérer la valeur d'une propriété d'une propriété (créée avec l'action Déclarer une propriété, dans la zone de déclaration (premier onglet de l'éditeur de fonctions)) possèdant une valeur de type spécifié.

Exemple : Si la valeur de la propriété est de type Button (System.Windows.Forms.Button), vous pourrez récupérer toutes les propriétés issue de ce type, à savoir, Text, BackColor, Size, etc.

Paramètres

Paramètres Descriptions
Propriété dont il faut récupérer une propriété Doit être égale à la propriété dont il faut récupérer la valeur d'une propriété spécifiée dans les paramètres suivants.
Type de la valeur de la propriété Doit être égale au type de la valeur de la propriété. Veillez à indiquer la bonne, dans le cas contraire, des erreurs d'exécutions de votre projet se produiront.
Propriété à récupérer (selon le type) Permet de choisir la propriété de la propriété à récupérer.
Variable recevant le résultat Doit être égale à la variable (non tableau) recevant la valeur de la propriété spécifiée dans le paramètre précédent.

Erreur à ne pas commettre : Attention à ce que le type de la valeur de la propriété soit le bon. Dans le cas contraire, une erreur d'exécution sera rencontré.
Note : Pour vérifier le type d'une valeur, utilisez l'action Obtenir le type d'une propriété, dans la catégorie Propriétés.

Valeur retournée

La valeur retournée est déterminée par la valeur de la propriété sélectionnée dans le paramètre "Propriété à récupérer (selon le type)". Le type de la valeur varie en fonction de la propriété.

Code Visual Basic.Net généré par SoftwareZator

Votre_Recevant_Valeur = CType(Me.Votre_Propriété, Votre_Type).Votre_Propriete2

Erreurs d'exécutions

Aucune erreur d'exécution n'est détecté ici.
