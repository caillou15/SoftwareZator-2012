Action : Fermer une fenêtre enfant

Cette action va vous permettre de fermer une fenêtre précédemment ouverte dans une fenêtre parent spécifiée.

Paramètres

Paramètres Descriptions
Fenêtre parent dont il faut fermer une fenêtre enfant Permet de définir la fenêtre qui contient celle qui sera fermée.
Nom de la fenêtre à fermer Doit être égale au nom de la fenêtre à fermer. Ce nom équivaut à la valeur de la propriété "Name" de la fenêtre à fermer.

Astuce : Pour récupérer la propriété Name d'une fenêtre, utilisez l'action Obtenir une propriété d'un contrôle ou Obtenir une propriété d'une variable.

Erreur à ne pas commettre : Pour le paramètre "Nom de la fenêtre à fermer", vous ne pouvez pas utiliser des ressources du projet de type Image et de type Autre. Seule des ressources de type Texte peuvent être utilisées.

Erreur à ne pas commettre : Si vous utilisez des variables dans le paramètre "Nom de la fenêtre à fermer", assurez-vous que le type de la valeur de chacune de ces variables soit du texte (String), ou bien une valeur de type numérique (Integer, Decimal, Double). En effet, seul du texte peut être utilisé dans ces paramètres.
Note : Pour vérifier le type d'une valeur, utilisez l'action Obtenir le type d'une variable, dans la catégorie Variables.

Valeur retournée

Aucune valeur n'est retourné par cette action

Code Visual Basic.Net généré par SoftwareZator

For Each frm As System.Windows.Forms.Form In Votre_Fenêtre.MdiChildren : If frm.Name = "Nom de la fenêtre" Then : frm.Close() : Exit For : End If : Next
