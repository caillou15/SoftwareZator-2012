Action : Sortir d'une fonction pouvant retourner une valeur

Cette action permet de sortir d'une fonction avant la fin de son exécution. En outre, toutes les actions précédent celle-ci ne seront pas exécutées.

Erreur à ne pas commettre : N'utilisez pas cette action pour sortir d'une fonction pouvant être liée à un déclencheur (voir les paramètres de l'action Fonction). Utilisez-la uniquement pour une fonction de type "Fonction pouvant retourner une valeur".

Paramètres

Aucune paramètre n'est nécessaire pour cette action

Valeur retournée

Aucune valeur n'est retourné par cette action
