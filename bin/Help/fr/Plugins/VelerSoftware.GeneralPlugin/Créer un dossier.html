Action : Créer un dossier

Cette action permet de créer un nouveau dossier sur le disque dur.

Information : Pour vérifier que le dossier n'existe pas déjà, utilisez l'action Déterminer si un dossier existe.

Paramètres

Paramètres Descriptions
Nom complet du dossier à créer Doit être égale au chemin complet où votre dossier devra être créé.

Erreur à ne pas commettre : Pour le paramètre "Nom complet du dossier à créer", vous ne pouvez pas utiliser des ressources du projet de type Image et de type Autre. Seule des ressources de type Texte peuvent être utilisées.

Erreur à ne pas commettre : Si vous utilisez dans variables dans les paramètres "Nom complet du dossier à créer", assurez-vous que le type de la valeur de chacune de ces variables soit du texte (String), ou bien une valeur de type numérique (Integer, Decimal, Double). En effet, seul du texte peut être utilisé dans ces paramètres.
Note : Pour vérifier le type d'une valeur, utilisez l'action Obtenir le type d'une variable, dans la catégorie Variables.

Valeur retournée

Aucune valeur n'est retourné par cette action

My.Computer.FileSystem.CreateDirectory("C:\Votre dossier")

Erreurs d'exécutions

Il arrive qu'un programme informatique rencontre un problème durant son exécution. Souvent, un message d'erreur est alors affiché. Le tableau ci-dessous vous permet de déterminer pourquoi l'erreur que vous rencontrez s'affiche et comment la corriger :

Types d'erreurs Descriptions Resolutions
System.InvalidCastException Cette erreur se produit lorsque la valeur du paramètre "Nom complet du dossier à créer" n'est pas du texte. Souvent, le problème est causé par une variable utilisé dans ce paramètre et qui ont une valeur autre que du texte. Pour corriger cette erreur, vérifiez que toutes les valeurs impliquées dans ce paramètre sont bien de type Texte (String) et faites les modifications nécessaires pour que ces valeurs soit du texte. Pour cela, utilisez les actions Obtenir le type d'une variable, Convertir la valeur d'une variable
System.ArgumentException
ou
System.NotSupportedException		 Cette erreur se produit lorsque la valeur du paramètre "Nom complet du dossier à créer" est nulle, ou contient le signe deux-points (:) ou une barre oblique (\ ou /). C'est caractères ne peuvent être utilisés que pour spécifier un disque dur et un dossier (C:\Un dossier\), mais en aucun cas pour le nom d'un fichier ou dossier. Pour corriger cette erreur, vérifiez que le nom du dossier à créer ne soit pas vide et ne contient pas le signe deux-points (:) ou une barre oblique (\ ou /).
System.ArgumentNullException Cette erreur se produit lorsque la valeur du paramètre "Nom complet du dossier à créer" est vide. Pour créer un dossier, vous devez impérativement spécifier le chemin complet vers l'emplacement où il devra être créé suivit du nom du dossier à créer. Pour corriger cette erreur, vérifiez que ce paramètre ne soit pas vide.
System.IO.PathTooLongException Cette erreur se produit lorsque le chemin vers dossier spécifié dans le paramètre "Nom complet du dossier à créer" dépasse la capacité système (256 caractères). Pour corriger cette erreur, vérifiez que le chemin vers le dossier à créer ne dépasse pas les 256 caractères (y compris pour les valeurs des variables).
System.SecurityException
ou
System.UnauthorizedAccessException		 Cette erreur se produit lorsque la session Windows actuel n'a pas les droits d'utilisateurs nécessaire pour créer le dossier spécifié. Pour corriger cette erreur, démarrez votre logiciel en mode Administrateur.
