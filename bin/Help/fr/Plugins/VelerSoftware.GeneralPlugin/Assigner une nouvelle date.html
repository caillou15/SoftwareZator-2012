Action : Assigner une nouvelle date

Cette action permet de définir une nouvelle date en mémoire et de la stocker dans la variable spécifiée.

Paramètres

Paramètres Descriptions
Année Doit être égale à une valeur numérique définissant l'année de la date.
Mois Doit être égale à une valeur numérique définissant le mois de la date. Cette valeur doit être comprise entre 1 (Janvier) et 12 (Décembre).
Jour Doit être égale à une valeur numérique définissant le jour de la date. Cette valeur doit être comprise entre 1 et 31.
Heure Doit être égale à une valeur numérique définissant l'heure de la date. Cette valeur doit être comprise entre 0 et 23.
Minute Doit être égale à une valeur numérique définissant la minute de la date. Cette valeur doit être comprise entre 0 et 59.
Seconde Doit être égale à une valeur numérique définissant la seconde de la date. Cette valeur doit être comprise entre 0 et 59.
Variable recevant le résultat Doit être égale à la variable (non tableau) qui sera égale à la nouvelle date.

Erreur à ne pas commettre : Faites attention, pour tous les paramètres, si vous utilisez des variables, que celles-ci possèdent une valeur de type numérique (Integer, Decimal, Double).
Note : Pour vérifier le type d'une valeur, utilisez l'action Obtenir le type d'une variable, dans la catégorie Variables.

Valeur retournée

La valeur retournée est déterminée par les valeur spécifées dans les paramètres. Cette valeur est enregistré dans la variable définie dans le paramètre "Variable recevant le résultat". Cette valeur est de type Date (System.DateTime).

Code Visual Basic.Net généré par SoftwareZator

Si la date spécifiée est le 25 Janvier 2012 à 4h 8min et 12sec, le code généré sera le suivant :

Votre_Variable = New System.DateTime(2012, 1, 25, 4, 8, 12)
