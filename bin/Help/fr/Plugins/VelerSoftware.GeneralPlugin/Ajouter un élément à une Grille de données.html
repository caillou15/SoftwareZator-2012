Action : Ajouter un élément à une Grille de données

Cette action permet d'ajouter une nouvelle ligne à la suite des autres dans une Grille de données.

Attention : Assurez-vous que la fenêtre dans laquelle réside votre Grille de données est ouverte (affiché à l'écran) au moment où cette action intervient.

Paramètres

Paramètres Descriptions
Grille de données concerné Doit être égale à la Grille de données dans laquelle l'élément sera ajouté.
Donnée à ajouter à chaque cellule de la ligne ajoutée Vous permet de saisir les valeurs textuelles qui apparaitrons dans la ligne ajoutée. Vous pouvez utiliser des variables (par exemple) avec la syntaxe habituelle, à savoir : %(VARIABLE=Votre_Variable)%.

Valeur retournée

Aucune valeur n'est retourné par cette action

Code Visual Basic.Net généré par SoftwareZator

Si la Grille de données se trouve dans la même fenêtre que celle d'où vous utilisez cette action, le code généré sera le suivant :

Me.Votre_Grille.Rows.Add(New String() {"Donnée1", "Donnée2", "Donnée3"})

Si la Grille de données se trouve dans une autre fenêtre que celle d'où vous utilisez cette action, le code généré sera le suivant :

Variables.Votre_Fenetre.Votre_Grille.Rows.Add(New String() {"Donnée1", "Donnée2", "Donnée3"})

Erreurs d'exécutions

Aucune erreur d'exécution n'est détecté ici.
