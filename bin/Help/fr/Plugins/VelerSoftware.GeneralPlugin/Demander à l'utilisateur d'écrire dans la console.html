Action : Demander à l'utilisateur d'écrire dans la console

Cette action vous permet, dans un projet de type Application Console, d'attendre que l'utilisateur écrive une ligne de texte dans la console pour continuer. Le texte écrit est alors récupéré dans une variable.

Paramètres

Paramètres Descriptions
Variable reçevant le texte écrit par l'utilisateur Doit être égale à la variable (non tableau) qui sera égale à la description du projet.

Valeur retournée

La valeur retournée est déterminée par la donnée entrée par l'utilisateur au moment de l'exécution de cette action. Cette valeur est de type texte (String).

Code Visual Basic.Net généré par SoftwareZator

Votre_Variable = System.Console.ReadLine()
