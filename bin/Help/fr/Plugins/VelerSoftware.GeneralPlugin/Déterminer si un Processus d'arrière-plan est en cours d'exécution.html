Action : Déterminer si un Processus d'arrière-plan est en cours d'exécution

Cette action permet de récupérer une valeur qui indique si le Processus d'arrière-plan spécifié est en cours de fonctionnement ou pas.

Paramètres

Paramètres Descriptions
Processus d'arrière-plan concerné Doit être égale au Processus d'arrière-plan dont on doit déterminer s'il est en cours d'exécution.
Variable recevant le résultat Doit être égale à une variable (non tableau) qui reçoit le résultat.

Valeur retournée

La valeur retournée est déterminée par le status de fonctionnement du Processus d'arrière-plan. Elle sera enregistrée dans la variable définit dans le paramètre "Variable recevant le résultat". Cette valeur est de type Vrai/Faux (Boolean).

Résultat du teste Valeurs retournées
Le Processus d'arrière-plan est en cours d'exécution True
Le Processus d'arrière-plan est arrêté False

Code Visual Basic.Net généré par SoftwareZator

Si le Processus d'arrière-plan se trouve dans la même fenêtre que celle d'où vous utilisez cette action, le code généré sera le suivant :

Votre_Variable = Me.Votre_Processus_Arrière_Plan.IsBusy

Si le Processus d'arrière-plan se trouve dans une autre fenêtre que celle d'où vous utilisez cette action, le code généré sera le suivant :

Votre_Variables = Variables.Votre_Fenetre.Votre_Processus_Arrière_Plan.IsBusy

Erreurs d'exécutions

Aucune erreur d'exécution n'est détecté ici.
