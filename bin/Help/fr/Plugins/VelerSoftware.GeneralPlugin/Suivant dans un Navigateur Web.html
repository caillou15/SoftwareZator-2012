Action : Suivant dans un Navigateur Web

L'action "Suivant dans un Navigateur Web", permet d'aller à la page affiché juste avant d'avoir exécuté l'action "Précédent dans un Navigateur Web".

Attention : Assurez-vous que la fenêtre dans laquelle réside votre Navigateur Web est ouverte (affiché à l'écran) au moment où cette action intervient.

Paramètres

Paramètres Descriptions
Navigateur Web concerné Doit être égale au Navigateur Web qui retournera à la page suivante.

Valeur retournée

Aucune valeur n'est retourné par cette action

Code Visual Basic.Net généré par SoftwareZator

Si le Navigateur Web se trouve dans la même fenêtre que celle d'où vous utilisez cette action, le code généré sera le suivant :

Me.Votre_Navigateur.GoForeward()

Si le Navigateur Web se trouve dans une autre fenêtre que celle d'où vous utilisez cette action, le code généré sera le suivant :

Variables.Votre_Fenetre.Votre_Navigateur.GoForeward()

Erreurs d'exécutions

Aucune erreur d'exécution n'est détecté ici.
