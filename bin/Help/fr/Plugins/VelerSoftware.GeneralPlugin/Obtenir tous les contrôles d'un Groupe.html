Action : Obtenir tous les contrôles d'un Groupe

Cette action permet de récupérer tous les contrôles existant dans un Groupe.

Attention : Assurez-vous que la fenêtre dans laquelle réside votre Groupe est ouverte (affiché à l'écran) au moment où cette action intervient.

Paramètres

Paramètres Descriptions
Groupe concerné Doit être égale au Groupe dont il faut récupérer tous les contrôles.
Variable tableau recevant le résultat Doit être égale à la variable tableau qui sera égale à la liste des contrôles du Groupe spécifiée dans le paramètre précédent.

Valeur retournée

La valeur retournée est un tableau de valeur. Cette valeur est de type Control (System.Windows.Forms.Control). Elle est égale à l'ensembles des contrôles contenu dans le Groupe.

Code Visual Basic.Net généré par SoftwareZator

Votre_Variable_Tableau = New System.Collections.Generic.List(Of Object)(System.Linq.Enumerable.Cast(Of Object)(Votre_Groupe.Controls))
