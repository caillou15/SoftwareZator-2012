Action : Changer la couleur de fond de la console

Cette action vous permet, dans un projet de type Application Console, de changer la couleur de fond de la fenêtre de la console.

Paramètres

Paramètres Descriptions
Nouvelle couleur Définit la nouvelle couleur de fond de la console. Exceptionnellement, vous êtes limité à 15 couleurs.

Valeur retournée

Aucune valeur n'est retourné par cette action

Code Visual Basic.Net généré par SoftwareZator

Code généré pour la couleur noir.

System.Console.BackgroundColor = 0

Erreurs d'exécutions

Aucune erreur d'exécution n'est détecté ici.
