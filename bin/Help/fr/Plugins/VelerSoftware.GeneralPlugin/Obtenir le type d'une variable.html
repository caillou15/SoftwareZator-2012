Action : Obtenir le type d'une variable

Cette action permet de récupérer le type de la ou des valeurs d'une variable, sous la forme d'un texte

Paramètres

Paramètres Descriptions
Variable pour lequel il faut récupérer le type Doit être égale à la variable (tableau ou non) dont il faut récupérer la type de sa valeur.
Variable recevant le nom du type Doit être égale à la variable (non tableau) recevant le nom du type de la valeur.

Valeur retournée

La valeur retournée est déterminée par le type de la valeur de la variable spécifiée dans le paramètre Variable pour lequel il faut récupérer le type. Cette valeur est de type texte (String).

Exemple : Si la valeur de la variable est du texte (String), la valeur retourné sera "System.String".

Votre_Recevant_Type = Votre_Variable.GetType.FullName
