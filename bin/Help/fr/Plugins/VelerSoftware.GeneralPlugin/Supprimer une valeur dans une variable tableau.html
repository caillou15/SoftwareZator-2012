Action : Supprimer une valeur dans une variable tableau

Cette action permet de supprimer valeur dans une variable déclaré comme étant un tableau de valeurs. Vous devez soit connaitre l'index de cette valeur, soit connaitre la valeur en elle-même.

Paramètres

Paramètres Descriptions
Variable tableau à laquelle il faut supprimer une variable Doit être égale à la variable tableau pour laquelle une valeur sera supprimé.
L'index de la valeur à supprimer est connue Ce paramètre vous permet de supprimer la valeur à une place précise dans le tableau. Si vous indiquez l'index 0 par exemple, la valeur supprimée sera la toute première. Si vous indiquez l'index 1 par exemple, la valeur supprimée sera la seconde valeur du tableau.
La valeur à supprimer est connue En cochant ce paramètre, vous pourrez préciser quel valeur vous voulez supprimer, sans connaitre sa position dans le tableau de valeur.
Valeur à supprimer Permet de saisir la nouvelle valeur à ajouter dans la variable. Vous pouvez saisir une valeur textuel, numérique, ou encore du Visual Basic.Net, une couleur et des types particulier.

Erreur à ne pas commettre : Pour le paramètre "L'index de la valeur à supprimer est connue", faites attention si vous utilisez des variables à ce que celles-ci possèdent une valeur de type numérique (Integer, Decimal, Double).

Erreur à ne pas commettre : Pour le paramètre "L'index de la valeur à supprimer est connue", faites attention à ce que l'index indiqué soit suppérieur ou égale à 0, et qu'il soit inférieur au nombre de valeurs que possède la variable tableau. Dans le cas contraire, une erreur d'exécution sera levée.

Erreur à ne pas commettre : Pour le paramètre "Valeur à supprimer", faites attention à ce que la valeur indiqué existe dans la variable tableau. Dans le cas contraire, une erreur d'exécution sera levée.
Note : Pour vérifier si une valeur existe dans une variable tableau, utilisez l'action Déterminer si une valeur existe dans une variable tableau, dans la catégorie Général.

Valeur retournée

Aucune valeur n'est retourné par cette action.

Code Visual Basic.Net généré par SoftwareZator

Si le paramètre "L'index de la valeur à supprimer est connue" est coché, le code généré sera le suivant :

Votre_Variable.RemoveAt(Votre_Index)

Si le paramètre "La valeur à supprimer est connue" est coché, le code généré sera le suivant :

Votre_Variable.Remove(Votre_Valeur)

Erreurs d'exécutions

Il arrive qu'un programme informatique rencontre un problème durant son exécution. Souvent, un message d'erreur est alors affiché. Le tableau ci-dessous vous permet de déterminer pourquoi l'erreur que vous rencontrez s'affiche et comment la corriger :

Types d'erreurs Descriptions Resolutions
System.ArgumentOutOfRangeException Cette erreur se produit lorsque le paramètre "L'index de la valeur à supprimer est connue" est utilisé et que sa valeur est inférieur à 0 ou suppérieur au nombre de valeurs contenu dans la variable tableau spécifié. Le problème est donc que vous tentez de supprimer un élément à un index qui n'existe pas. Pour corriger cette erreur, vérifiez que le nombre de valeur de la variable tableau soit suppérieur ou égale à l'index que vous avez spécifié. Pour cela, utilisez les actions Obtenir le nombre de valeurs d'une variable tableau.
