Action : Supprimer un fichier

Cette action permet de supprimer un fichier du disque dur.

Information : Pour vérifier si fichier existe ou pas, utilisez l'action Déterminer si un fichier existe.

Paramètres

Paramètres Descriptions
Nom complet du fichier à supprimer Doit être égale au chemin complet du fichier qui sera supprimé du disque dur.
Afficher toutes les boîtes de dialogues En cochant ce paramètre, Windows affichera une boîte de dialogue demandant à l'utilisateur si le fichier sera supprimé, ainsi qu'une boîte affichant la progression de la suppression.
Afficher uniquement une boîte de dialogue en cas d'erreur En cochant ce paramètre, si une erreur se produit durant la suppression, une boîte de dialogue s'affichera pour montrer l'erreur. Autrement, la suppression est invisible pour l'utilisateur.
Déplacer le fichier dans la corbeille En cochant ce paramètre, le fichier à supprimer sera en fait déplacé dans la Corbeille de Windows. L'utilisateur aura alors la possibilité de restaurer cette suppression.
Supprimer le fichier définitivement En cochant ce paramètre, le fichier sera directement supprimer du disque dur. Cette action est irréversible.

Erreur à ne pas commettre : Pour le paramètre "Nom complet du fichier à supprimer", vous ne pouvez pas utiliser des ressources du projet de type Image et de type Autre. Seule des ressources de type Texte peuvent être utilisées.

Erreur à ne pas commettre : Si vous utilisez dans variables dans le paramètre "Nom complet du fichier à supprimer", assurez-vous que le type de la valeur de chacune de ces variables soit du texte (String), ou bien une valeur de type numérique (Integer, Decimal, Double). En effet, seul du texte peut être utilisé dans ce paramètre.
Note : Pour vérifier le type d'une valeur, utilisez l'action Obtenir le type d'une variable, dans la catégorie Variables.

Valeur retournée

Aucune valeur n'est retourné par cette action

Code Visual Basic.Net généré par SoftwareZator

Code généré si les paramètres "Afficher toutes les boîtes de dialogues" et "Déplacer le fichier dans la corbeille" sont cochés :

My.Computer.FileSystem.DeleteFile("C:\fichier à supprimer", Microsoft.VisualBasic.FileIO.UIOption.AllDialogs, Microsoft.VisualBasic.FileIO.RecycleOption.SendToRecycleBin)

Code généré si les paramètres "Afficher uniquement une boîte de dialogue en cas d'erreur" et "Supprimer le fichier définitivement" sont cochés :

My.Computer.FileSystem.DeleteFile("C:\fichier à supprimer", Microsoft.VisualBasic.FileIO.UIOption.OnlyErrorDialogs, Microsoft.VisualBasic.FileIO.RecycleOption.DeletePermanently)

Erreurs d'exécutions

Il arrive qu'un programme informatique rencontre un problème durant son exécution. Souvent, un message d'erreur est alors affiché. Le tableau ci-dessous vous permet de déterminer pourquoi l'erreur que vous rencontrez s'affiche et comment la corriger :

Types d'erreurs Descriptions Resolutions
System.InvalidCastException Cette erreur se produit lorsque la valeur du paramètre "Nom complet du fichier à supprimer" n'est pas du texte. Souvent, le problème est causé par une variable utilisé dans ce paramètre et qui a une valeur autre que du texte. Pour corriger cette erreur, vérifiez que toutes les valeurs impliquées dans ce paramètre sont bien de type Texte (String) et faites les modifications nécessaires pour que ces valeurs soit du texte. Pour cela, utilisez les actions Obtenir le type d'une variable, Convertir la valeur d'une variable
System.ArgumentException
ou
System.NotSupportedException		 Cette erreur se produit lorsque la valeur du paramètre "Nom complet du fichier à supprimer" est nulle, ou contient le signe deux-points (:) ou une barre oblique (\ ou /). C'est caractères ne peuvent être utilisés que pour spécifier un disque dur et un fichier (C:\Un fichier\), mais en aucun cas pour le nom d'un fichier ou dossier. Pour corriger cette erreur, vérifiez que le nom du fichier à supprimer ne soit pas vide et ne contient pas le signe deux-points (:) ou une barre oblique (\ ou /).
System.ArgumentNullException Cette erreur se produit lorsque la valeur du paramètre "Nom complet du fichier à supprimer" est vide. Pour supprimer un fichier, vous devez impérativement spécifier le chemin complet du fichier à supprimer. Pour corriger cette erreur, vérifiez que ce paramètre ne soit pas vide.
System.IO.FileNotFoundException Cette erreur se produit lorsque le chemin spécifié dans le paramètre "Nom complet du fichier à supprimer" est introuvable sur le disque dur. C'est-à-dire, que le fichier à supprimer n'existe pas. Pour corriger cette erreur, vérifiez que le chemin vers le fichier à supprimer soit correcte, et vérifiez que celui-ci existe avec l'action Déterminer si un fichier existe.
System.SecurityException
ou
System.UnauthorizedAccessException		 Cette erreur se produit lorsque la session Windows actuel n'a pas les droits d'utilisateurs nécessaire pour supprimer le fichier spécifié. Pour corriger cette erreur, démarrez votre logiciel en mode Administrateur.
System.IO.PathTooLongException Cette erreur se produit lorsque le chemin vers fichier spécifié dans le paramètre "Nom complet du fichier à supprimer" dépasse la capacité système (256 caractères). Pour corriger cette erreur, vérifiez que le chemin vers le fichier à supprimer ne dépasse pas les 256 caractères (y compris pour les valeurs des variables).
