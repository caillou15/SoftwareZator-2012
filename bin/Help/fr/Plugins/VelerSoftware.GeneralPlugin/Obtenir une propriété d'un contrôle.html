Action : Obtenir une propriété d'un contrôle

Cette action permet de récupérer la valeur d'une propriété d'un contrôle.

Exemple : Si le contrôle est de type Button (System.Windows.Forms.Button), vous pourrez récupérer toutes les propriétés issue de ce type, à savoir, Text, BackColor, Size, etc.

Paramètres

Paramètres Descriptions
Contrôle dont il faut récupérer une propriété Doit être égale au contrôle dont il faut récupérer la valeur d'une propriété spécifiée dans les paramètres suivants.
Propriété à récupérer (selon le type) Permet de choisir la propriété du contrôle à récupérer.
Variable recevant le résultat Doit être égale à la variable (non tableau) recevant la valeur de la propriété spécifiée dans le paramètre précédent.

Valeur retournée

La valeur retournée est déterminée par la valeur de la propriété sélectionnée dans le paramètre "Propriété à récupérer (selon le type)". Le type de la valeur varie en fonction de la propriété.

Code Visual Basic.Net généré par SoftwareZator

Votre_Variable = Votre_Contrôle.Votre_Propriete
