Action : Obtenir l'index d'une valeur dans une variable tableau

Cette action permet de récupérer le numéro correspondant à la valeur spécifiée dans une variable tableau définie.

Paramètres

Paramètres Descriptions
Variable tableau à laquelle il faut récupérer un index Doit être égale à la variable tableau pour laquelle on doit récupérer l'index d'une valeur spécifiée dans le paramètre suivant.
Valeur dont l'index doit être récupéré Permet de saisir la valeur dont on doit récupérer l'index. Vous pouvez saisir une valeur textuel, numérique, ou encore du Visual Basic.Net, une couleur et des types particulier.
Variable recevant le résultat Doit être égale à la variable (non tableau) recevant l'index de la valeur spécifiée dans le parmaètre précédent.

Valeur retournée

La valeur retournée est déterminée par la position de la valeur spécifiée dans la variable tableau. Cette valeur est enregistré dans la variable définie dans le paramètre "Variable recevant le résultat". Celle-ci est un nombre entier (Integer)

Important : Si la valeur retourné est -1, cela signifie que la valeur n'existe pas dans la variable tableau.

Code Visual Basic.Net généré par SoftwareZator

Votre_Variable = Votre_Variable_Tableau.IndexOf(Votre_Valeur)
