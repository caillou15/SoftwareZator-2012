Action : Supprimer un caractère à la fin d'un texte

Cette action va vous permettre de supprimer un caractère qui se trouve tout à la fin d'un texte. Ce caractère doit-être le tout dernier du texte.

Paramètres

Paramètres Descriptions
Texte concerné Doit être égale au texte dont un caractère sera supprimé.
Caractère à supprimer à la fin du texte Permet de saisir le caractère qui devra être supprimé à la fin du texte.
Variable recevant le résultat Doit être égale à la variable (non tableau) recevant le résultat du traitement sur le texte.

Erreur à ne pas commettre : Pour les paramètres "Texte concerné" et "Caractère à supprimer au début du texte", vous ne pouvez pas utiliser des ressources du projet de type Image et de type Autre. Seule des ressources de type Texte peuvent être utilisées.

Erreur à ne pas commettre : Si vous utilisez dans variables dans les paramètres "Texte concerné" et "Caractère à supprimer au début du texte", assurez-vous que le type de la valeur de chacune de ces variables soit du texte (String), ou bien une valeur de type numérique (Integer, Decimal, Double). En effet, seul du texte peut être utilisé dans ces paramètres.
Note : Pour vérifier le type d'une valeur, utilisez l'action Obtenir le type d'une variable, dans la catégorie Variables.

Valeur retournée

La valeur retournée est déterminée par le caractère à supprimer à la fin du texte. Elle sera enregistrée dans la variable définit dans le paramètre "Variable recevant le résultat". Cette valeur est de type texte (String).

Exemple : Soit le texte "Bonjour". On demande de supprimer le caractère "r". La valeur retourné sera "Bonjou".

Code Visual Basic.Net généré par SoftwareZator

Votre_Variable = CStr("Bonjour").TrimEnd("r")
