Action : Fermer une fenêtre

Cette action va vous permettre de fermer une fenêtre précédemment ouverte, d'un projet de votre solution, à l'écran.

Erreur à ne pas commettre : Une fenêtre déjà fermé ne peut pas être fermé une seconde fois.

Erreur à ne pas commettre : N'utilisez pas cette action à partir d'une fonction lié à l'événement Load ou Activated d'une fenêtre. Les deux conditions peuvent provoquer des fuites de mémoire.

Paramètres

Paramètres Descriptions
Fenêtre à fermer Permet de définir la fenêtre qui devra être fermée.
Valeur à retourner Doit être égale à la valeur qui sera retourné par la fenêtre.

Valeur retournée

La valeur retournée est déterminée par la valeur du paramètre "Valeur à retourner". Elle sera enregistrée dans la variable définit dans le paramètre "Variable recevant le résultat" de l'action Ouvrir une fenêtre. Cette valeur est de type texte (String).

Valeur à retourner Valeurs retournées
OK OK
Annuler Cancel
Oui Yes
Non No
Réessayer Retry
Abandonner Abort
Ignorer Ignore

Erreurs d'exécutions

Il arrive qu'un programme informatique rencontre un problème durant son exécution. Souvent, un message d'erreur est alors affiché. Le tableau ci-dessous vous permet de déterminer pourquoi l'erreur que vous rencontrez s'affiche et comment la corriger :

Types d'erreurs Descriptions Resolutions
System.InvalidOperationException Cette erreur se produit lorsque vous tentez de fermer la fenêtre spécifié alors que celle-ci n'est pas encore créé ou bien déjà fermé. Pour corriger cette erreur, vérifiez que la fenêtre spécifié est ouverte. Pour cela, utilisez les actions Déterminer si une fenêtre est ouverte.
