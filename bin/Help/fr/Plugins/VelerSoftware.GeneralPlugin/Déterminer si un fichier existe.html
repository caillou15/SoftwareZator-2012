Action : Déterminer si un fichier existe

Cette action permet de récupérer une valeur qui indique si un fichier spécifié existe ou non sur le disque dur.

Paramètres

Paramètres Descriptions
Nom complet du fichier à tester Doit être égale au chemin complet du fichier dont il faut tester l'existance.
variable recevant le résultat Doit être égale à une variable (non tableau) qui reçoit la résultat du teste.

Erreur à ne pas commettre : Pour le paramètre "Nom complet du fichier à tester", vous ne pouvez pas utiliser des ressources du projet de type Image et de type Autre. Seule des ressources de type Texte peuvent être utilisées.

Erreur à ne pas commettre : Si vous utilisez des variables dans le paramètre "Nom complet du fichier à tester", assurez-vous que le type de la valeur de chacune de ces variables soit du texte (String), ou bien une valeur de type numérique (Integer, Decimal, Double). En effet, seul du texte peut être utilisé dans ce paramètre.
Note : Pour vérifier le type d'une valeur, utilisez l'action Obtenir le type d'une variable, dans la catégorie Variables.

Valeur retournée

La valeur retournée est déterminée par la présence du fichier sur le disque dur. Elle sera enregistrée dans la variable définit dans le paramètre "Variable recevant le résultat". Cette valeur est de type Vrai/Faux (Boolean).

Résultat du teste Valeurs retournées
Le fichier existe True
Le fichier n'existe pas False

Code Visual Basic.Net généré par SoftwareZator

Votre_Variable = My.Computer.FileSystem.FileExists("C:\Votre fichier.txt")

Erreurs d'exécutions

Il arrive qu'un programme informatique rencontre un problème durant son exécution. Souvent, un message d'erreur est alors affiché. Le tableau ci-dessous vous permet de déterminer pourquoi l'erreur que vous rencontrez s'affiche et comment la corriger :

Types d'erreurs Descriptions Resolutions
System.InvalidCastException Cette erreur se produit lorsque la valeur du paramètre "Nom complet du fichier à tester" n'est pas du texte. Souvent, le problème est causé par une variable utilisé dans ce paramètre et qui a une valeur autre que du texte. Pour corriger cette erreur, vérifiez que toutes les valeurs impliquées dans ce paramètre sont bien de type Texte (String) et faites les modifications nécessaires pour que ces valeurs soit du texte. Pour cela, utilisez les actions Obtenir le type d'une variable, Convertir la valeur d'une variable
