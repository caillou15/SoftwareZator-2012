Action : Afficher l'infobulle d'une Icône d'avertissement

Cette action permet d'afficher à l'écran l'infobulle de l'Icône d'avertissement spécifiée pendant un laps de temps défini.

Attention : Assurez-vous que la fenêtre dans laquelle réside votre Icône d'avertissement est ouverte (affiché à l'écran) au moment où cette action intervient.

Paramètres

Paramètres Descriptions
Icône d'avertissement concerné Doit être égale à l'Icône d'avertissement dont il faut afficher l'infobulle.
Temps d'affichage (en millisecondes) Saisissez dans ce paramètre une valeur numérique définissant le temps d'affichage (en milliseconde, donc 1000 = 1sec) de l'infobulle.

Erreur à ne pas commettre : Pour le paramètre "Temps d'affichage (en millisecondes)", vous ne pouvez pas utiliser des ressources du projet de type Image, Texte et de type Autre.

Erreur à ne pas commettre : Si vous utilisez dans variables dans le paramètre "Temps d'affichage (en millisecondes)", assurez-vous que le type de la valeur de chacune de ces variables soit de type numérique (Integer, Decimal, Double). En effet, seul une valeur numérique peut être utilisé dans ce paramètre.
Note : Pour vérifier le type d'une valeur, utilisez l'action Obtenir le type d'une variable, dans la catégorie Variables.

Valeur retournée

Aucune valeur n'est retourné par cette action
