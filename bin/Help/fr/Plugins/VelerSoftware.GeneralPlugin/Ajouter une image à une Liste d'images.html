Action : Ajouter une image à une Liste d'images

Cette action donne la possibilité d'ajouter une image à la suite dans une Liste d'images. Souvent, les Listes d'images sont utilisés pour définir les icônes à utiliser dans une Liste d'éléments avancés ou autre.

Attention : Assurez-vous que la fenêtre dans laquelle réside votre Liste d'images est ouverte (affiché à l'écran) au moment où cette action intervient.

Paramètres

Paramètres Descriptions
Liste d'images concerné Doit être égale à la Liste d'images à laquelle vous voulez ajouter une image.
Image à ajouter Doit être égale à une variable, ressource, ou autre, comportant une image qui sera celle ajoutée.

Valeur retournée

Aucune valeur n'est retourné par cette action

Code Visual Basic.Net généré par SoftwareZator

Si la Liste d'images se trouve dans la même fenêtre que celle d'où vous utilisez cette action, et que l'image à ajouter est égale à la variable "Une_Variable", le code généré sera le suivant :

Me.Votre_Liste.Images.Add(Une_Variable)

Si la Liste d'images se trouve dans une autre fenêtre que celle d'où vous utilisez cette action, et que l'image à ajouter est égale à la ressource "Une_Image", le code généré sera le suivant :

Variables.Votre_Fenetre.Votre_Liste.Images.Add(My.Resources.Une_Image)

Types d'erreurs Descriptions Resolutions

ou
System.InvalidCastException		 Cette erreur se produit lorsque la valeur du paramètre "Image à ajouter" n'est pas une image. Souvent, le problème est causé par une variable ou ressource utilisée dans ce paramètre et qui a une valeur autre qu'une image. Pour corriger cette erreur, vérifiez que toutes le valeur impliquée dans ce paramètre soit bien de type Image (System.Drawing.Image) et faites les modifications nécessaires pour que cette valeur soit une image. Pour cela, utilisez les actions Obtenir le type d'une variable, Convertir la valeur d'une variable
