Action : Supprimer une valeur dans le registre

Cette action permet de supprimer une valeur existante dans une clé du registre Windows.

Information : Pour explorer le registre Windows, démarrer "Exécuter" (touche Windows + R de votre clavier), puis tapez "regedit" et valiez.

Information : Aucune erreur d'exécution ne sera levée si la valeur n'existe pas, mais une erreur de type "System.NullReferenceException" sera levée si la clé est introuvable.

Paramètres

Paramètres Descriptions
Emplacement et nom de la clé Doit être égale au chemin complet de la clé contenant la valeur à supprimer. Indiquez dans la liste déroulante la base de registre concerné puis le chemin.
Exemple : Liste déroulante : LocalMachine ; Zone de texte : SOFTWARE\Votre Société
L'exemple ci-dessus supprimera alors une valeur dans la clé à l'emplacement "HKEY_LOCAL_MACHINE\SOFTWARE\Votre Société".
Nom de la valeur à supprimer Doit être égale au nom de la valeur à supprimer dans la clé définie dans le paramètre précédent.
Remarque : Pour supprimer la valeur par défaut de la clé, laissez ce champs vide.

Erreur à ne pas commettre : Pour les paramètres "Emplacement et nom de la clé" et "Nom de la valeur à supprimer", vous ne pouvez pas utiliser des ressources du projet de type Image et de type Autre. Seule des ressources de type Texte peuvent être utilisées.

Erreur à ne pas commettre : Si vous utilisez dans variables dans les paramètres "Emplacement et nom de la clé" et "Nom de la valeur à supprimer", assurez-vous que le type de la valeur de chacune de ces variables soit du texte (String), ou bien une valeur de type numérique (Integer, Decimal, Double). En effet, seul du texte peut être utilisé dans ce paramètre.
Note : Pour vérifier le type d'une valeur, utilisez l'action Obtenir le type d'une variable, dans la catégorie Variables.

Valeur retournée

Aucune valeur n'est retourné par cette action

Code Visual Basic.Net généré par SoftwareZator

My.Computer.Registry.LocalMachine.OpenSubKey("Clé\Sous clé", True).DeleteValue("Valeur")

Erreurs d'exécutions

Il arrive qu'un programme informatique rencontre un problème durant son exécution. Souvent, un message d'erreur est alors affiché. Le tableau ci-dessous vous permet de déterminer pourquoi l'erreur que vous rencontrez s'affiche et comment la corriger :

Types d'erreurs Descriptions Resolutions
System.InvalidCastException Cette erreur se produit lorsque la valeur du paramètre "Emplacement et nom de la clé" et ou "Nom de la valeur à supprimer" ne sont pas du texte. Souvent, le problème est causé par une variable utilisé dans ce paramètre et qui a une valeur autre que du texte. Pour corriger cette erreur, vérifiez que toutes les valeurs impliquées dans ces paramètres sont bien de type Texte (String) et faites les modifications nécessaires pour que ces valeurs soit du texte. Pour cela, utilisez les actions Obtenir le type d'une variable, Convertir la valeur d'une variable
System.ArgumentNullException Cette erreur se produit lorsque la valeur du paramètre "Emplacement et nom de la clé" et ou "Nom de la valeur à supprimer" sont vides. Pour supprimer une clé, vous devez impérativement spécifier le chemin complet de la clé à supprimer. Pour corriger cette erreur, vérifiez que ce paramètre ne soit pas vide.
System.Security.SecurityException
ou
UnauthorizedAccessException		 Cette erreur se produit lorsque l'utilisateur n'a pas les autorisations requises pour supprimer la clé ou ne dispose pas des droits d'accès à la base de registres appropriés. Pour corriger cette erreur, vérifiez que l'utilisateur ai les droits d'accès au registre et ai les autorisations nécessaires pour supprimer la clé. Pour Windows Vista, 7 et 8, vérifiez le niveau de l'UAC.
System.NullReferenceException Cette erreur se produit lorsque la clé définie dans le paramètre "Emplacement et nom de la clé" est introuvable dans le registre Windows. Pour corriger cette erreur, vérifiez que la clé existe bien à l'emplacement désigné sur l'ordinateur de l'utilisateur rencontrant cette erreur.
