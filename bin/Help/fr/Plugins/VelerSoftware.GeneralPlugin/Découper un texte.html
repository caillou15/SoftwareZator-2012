Action : Découper un texte

Cette action va vous permettre de transformer un texte en un tableau de valeurs. Le découpage du texte sera fait à partir d'un caractère donnée.

Paramètres

Paramètres Descriptions
Texte concerné Doit être égale au texte qui sera découpé.
Caractère de découpage Permet de saisir le caractère qui définira le découpage du texte (voir la partie "Valeur retournée" ci-dessous).
Variable tableau recevant le résultat Doit être égale à la variable tableau recevant le résultat du traitement sur le texte.

Erreur à ne pas commettre : Pour les paramètres "Texte concerné" et "Caractère de découpage", vous ne pouvez pas utiliser des ressources du projet de type Image et de type Autre. Seule des ressources de type Texte peuvent être utilisées.

Erreur à ne pas commettre : Si vous utilisez dans variables dans les paramètres "Texte concerné" et "Caractère de découpage", assurez-vous que le type de la valeur de chacune de ces variables soit du texte (String), ou bien une valeur de type numérique (Integer, Decimal, Double). En effet, seul du texte peut être utilisé dans ces paramètres.
Note : Pour vérifier le type d'une valeur, utilisez l'action Obtenir le type d'une variable, dans la catégorie Variables.

Valeur retournée

La valeur retournée est déterminée par le caractère de découpage. Elle sera enregistrée dans la variable définit dans le paramètre "Variable tableau recevant le résultat". Cette valeur est un tableau de valeurs texte (String).

Exemple : Soit le texte "Bonjour|Monsieur|Dupont". On demande de découper selon le caractère "|". La valeur retourné sera :

  • Bonjour
  • Monsieur
  • Dupont

Votre_Variable_Tableau = New System.Collections.Generic.List(Of Object)(System.Linq.Enumerable.Cast(Of Object)(CStr("Bonjour|Monsieur|Dupont").Split("|")))

Erreurs d'exécutions

Il arrive qu'un programme informatique rencontre un problème durant son exécution. Souvent, un message d'erreur est alors affiché. Le tableau ci-dessous vous permet de déterminer pourquoi l'erreur que vous rencontrez s'affiche et comment la corriger :

Types d'erreurs Descriptions Resolutions
System.ArgumentException
ou
System.InvalidCastException		 Cette erreur se produit lorsque la valeur du paramètre "Texte concerné" et ou "Caractère de découpage" ne sont pas du texte. Souvent, le problème est causé par une variable utilisé dans ces paramètres et qui ont une valeur autre que du texte. Pour corriger cette erreur, vérifiez que toutes les valeurs impliquées dans ces paramètres sont bien de type Texte (String) et faites les modifications nécessaires pour que ces valeurs soit du texte. Pour cela, utilisez les actions Obtenir le type d'une variable, Convertir la valeur d'une variable
