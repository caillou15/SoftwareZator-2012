Action : Exécuter une fonction d'un paramètre

Cette action permet de faire exécuter une fonction d'un paramètre possèdant une valeur de type spécifié.

Paramètres

Paramètres Descriptions
Paramètre concerné Doit être égale au paramètre pour lequel une fonction sera exécutée.
Type de la valeur du paramètre Doit être égale au type de la valeur du paramètre. Veillez à indiquer la bonne, dans le cas contraire, des erreurs d'exécutions de votre projet se produiront.
Fonction à exécuter (selon le type) Doit être égale à la fonction que vous voulez faire exécuter.
Paramètres de la fonction Cette liste permet de définir les paramètres à donner à la fonction.
  • Si vous voulez indiquez une valeur nul, utilisez le mot clé suivant : Nothing
  • Si vous voulez utilisez du texte en paramètres, utilisez des guillets : "Votre texte"
  • Si vous voulez utilisez dans variables dans un paramètres, introduisez les de la manière suivante : "Votre texte" & Votre_Variable & "la fin du texte"
    Ou bien :
    1234 + Votre_Variable * 678
Variable recevant le résultat de la fonction Si votre fonction peut retourner une valeur, vous pourrez sélectionner une variable (non tableau) qui sera assigné une fois la fonction exécutée.

Valeur retournée

La valeur retournée dépend du type de valeur retourné par la fonction que vous aurez choisi d'exécuter, ainsi que les paramètres de celle-ci, si elle en possède. Elle sera enregistrée dans la variable définit dans le paramètre "Variable recevant le résultat de la fonction". Cette valeur est de type texte (String), numérique (Integer, Decimal, Double), couleur (System.Drawing.Color) ou autre selon les fonctions.

Code Visual Basic.Net généré par SoftwareZator

Si la fonction à exécuté ne peut pas retourner de valeur, le code généré sera le suivant :

Votre_Paramètre.Votre_Fonction(Paramètre1, Paramètre2)

Si la fonction à exécuté peut retourner une valeur, le code généré sera le suivant :

                                                
Votre_Variable_De_Resultat = Votre_Paramètre.Votre_Fonction(Paramètre1, Paramètre2)

Erreurs d'exécutions

Aucune erreur d'exécution n'est détecté ici.
