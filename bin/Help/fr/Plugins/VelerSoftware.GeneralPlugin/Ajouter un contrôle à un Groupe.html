Action : Ajouter un contrôle à un Groupe

Cette action permet d'ajouter un contrôle à la suite des autres dans un Groupe.

Attention : Assurez-vous que la fenêtre dans laquelle réside votre Groupe est ouverte (affiché à l'écran) au moment où cette action intervient.

Paramètres

Paramètres Descriptions
Groupe concerné Doit être égale au Groupe dans lequel un contrôle sera ajouté.
Contrôle à ajouter Permet de saisir le contrôle à ajouter. Celui-ci peut être dans une variable ou créé au moment de l'action (contrôle dynamique).

Valeur retournée

Aucune valeur n'est retourné par cette action

Code Visual Basic.Net généré par SoftwareZator

Si la Groupe se trouve dans la même fenêtre que celle d'où vous utilisez cette action, que le contrôle est un nouveau bouton (créé au moment de l'exécution de l'action), le code généré sera le suivant :

Me.Votre_Groupe.Controls.Add(New System.Windows.Forms.Button())

Si la Groupe se trouve dans une autre fenêtre que celle d'où vous utilisez cette action, que le contrôle est enregistré dans une variable, le code généré sera le suivant :

Variables.Votre_Fenetre.Votre_Groupe.Controls.Add(Votre_Variable)

Erreurs d'exécutions

Aucune erreur d'exécution n'est détecté ici.
