Action : Assigner une nouvelle donnée sérialisable

Cette action permet de créer un nouvelle donnée sérialisable en mémoire et de l'enregistrer dans une variable.

Qu'est-ce que la sérialisation : C'est un procéder qui permet d'enregistrer la valeur complète d'une variable, sous une forme crypté, dans un fichier, et de pouvoir la réouvrir tel qu'elle était lors de son enregistrement, après avoir quitté et relancé votre logiciel par exemple.

Information : Pour modifier la donnée sérialisable une fois qu'elle a été initialisé avec cette action, utilisez l'action Modifier une propriété d'une donnée sérialisable.

Paramètres

Paramètres Descriptions
Type de donnée sérialisable à initialiser Doit être égale au type de donnée, spécifique au projet, devant être initialisé. Pour créer un nouveau type, cliquer sur le bouton "Gestionnaire de types"
Variable recevant la nouvelle donnée sérialisable Doit être égale à la variable (non tableau) recevant la nouvelle donnée qui a été créée.

Valeur retournée

La valeur retournée est déterminée par le type de donnée sérialisable initialisé. Elle sera enregistrée dans la variable définit dans le paramètre "Variable recevant la nouvelle donnée sérialisable".
