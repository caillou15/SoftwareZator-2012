Action : Obtenir une valeur dans une variable tableau

Cette action permet de récupérer une valeur existante dans une variable déclaré comme étant un tableau de valeurs, à l'index spécifié.

Paramètres

Paramètres Descriptions
Variable tableau dont il faut récupérer une variable Doit être égale à la variable tableau pour laquelle une valeur sera récupérée.
Index de la valeur à récupérer Ce paramètre doit être égale à l'index de la valeur que vous voulez récupérer dans la variable tableau. Autrement dit, la place de cette valeur dans la variable. Si vous indiquez l'index 0 par exemple, la valeur récupérée sera la toute première valeur du tableau. Si vous indiquez l'index 1 par exemple, la valeur récupérée sera celle juste avant la seconde valeur du tableau. On peut illustrer ce paramètre avec l'illustration ci-dessous.
Variable recevant la valeur récupérée Doit être égale à la variable (non tableau) qui sera égale à la valeur récupérée dans la variable tableau.

Valeur retournée

La valeur retournée est déterminée par la position de la valeur spécifiée dans la variable tableau. Cette valeur est enregistré dans la variable définie dans le paramètre "Variable recevant la valeur récupérée". Le type de cette valeur varie en fonction de celle-ci. Pour obtenir le type d'une valeur, utilisez l'action Obtenir le type d'une variable, dans la catégorie Variables.

Code Visual Basic.Net généré par SoftwareZator

Votre_Variable = Votre_Variable_Tableau(Index)

Erreurs d'exécutions

Il arrive qu'un programme informatique rencontre un problème durant son exécution. Souvent, un message d'erreur est alors affiché. Le tableau ci-dessous vous permet de déterminer pourquoi l'erreur que vous rencontrez s'affiche et comment la corriger :

Types d'erreurs Descriptions Resolutions
System.ArgumentOutOfRangeException Cette erreur se produit lorsque la valeur du paramètre "Index de la valeur à récupérer" est inférieur à 0 ou suppérieur au nombre de valeurs contenu dans la variable tableau spécifié. Le problème est donc que vous tentez de récupérer un élément au dessus d'un élément qui n'existe pas. Pour corriger cette erreur, vérifiez que le nombre de valeur de la variable tableau soit suppérieur ou égale à l'index que vous avez spécifié. Pour cela, utilisez les actions Obtenir le nombre de valeurs d'une variable tableau.
