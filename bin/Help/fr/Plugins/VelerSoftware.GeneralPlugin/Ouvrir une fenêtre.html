Action : Ouvrir une fenêtre

Cette action va vous permettre d'afficher une fenêtre, d'un projet de votre solution, à l'écran.

Information : Pour fermer une fenêtre de votre application, utilisez l'action Fermer une fenêtre.

Erreur à ne pas commettre : Une fenêtre déjà affiché à l'écran ne peut pas être ouverte une seconde fois en même temps.

Paramètres

Paramètres Descriptions
Fenêtre à ouvrir Permet de définir la fenêtre qui devra être ouverte.
Bloquer les fenêtres derrière pendant que celle-ci est ouverte Si ce paramètre est coché, lorsque la fenêtre spécifiée s'ouvrira, celle-ci empêchera l'utilisateur d'avoir accès aux autres fenêtres ouvertes jusqu'à ce qu'elle soit fermé. En activant ce paramètre, vous aurez aussi la possibilité de récupérer une valeur de retour de la fenêtre.
Remarque : si ce paramètre est utilisé, les actions après celle-ci ne seront exécuté qu'une fois la fenêtre fermé.
Variable recevant le résultat (facultatif) Doit être égale à la variable (non tableau) recevant le résultat de la fermeture de la fenêtre.

Valeur retournée

La valeur retournée est déterminée par le mode de fermeture de la fenêtre défini dans l'action Fermer une fenêtre, utilisé pour fermer la fenêtre spécifiée. Elle sera enregistrée dans la variable définit dans le paramètre "Variable recevant le résultat". Cette valeur est de type texte (String).

Mode de fermeture Valeurs retournées
OK OK
Annuler (ou bien le bouton rouge de fermeture en haut à droite de la fenêtre) Cancel
Oui Yes
Non No
Réessayer Retry
Abandonner Abort
Ignorer Ignore

Code Visual Basic.Net généré par SoftwareZator

Si la fenêtre ne bloque pas les fenêtres enfants, la code généré est le suivant :

Votre_Fenêtre.Show()

Si la fenêtre bloque les fenêtres enfants et qu'une variable reçoit le résultat, le code généré sera le suivant :

Votre_Variable = Votre_Fenêtre.ShowDialog.ToString()

Erreurs d'exécutions

Il arrive qu'un programme informatique rencontre un problème durant son exécution. Souvent, un message d'erreur est alors affiché. Le tableau ci-dessous vous permet de déterminer pourquoi l'erreur que vous rencontrez s'affiche et comment la corriger :

Types d'erreurs Descriptions Resolutions
System.InvalidOperationException Cette erreur se produit lorsque l'une de ces conditions est rencontré :
  • La fenêtre à ouvrir est déjà affiché à l'écran.
  • La fenêtre à ouvrir est la même que celle depuis laquelle vous utilisez cette action.
  • La fenêtre à ouvrir est désactiver.
 Pour corriger cette erreur, vérifiez que la fenêtre n'est pas déjà affiché (il faut que la propriété Visible de la fenêtre est égale à False) et que vous ne tentez pas d'ouvrir la fenêtre depuis laquelle vous exécutez cette action. Pour cela, utilisez l'action Obtenir une propriété d'un contrôle.
