Action : Obtenir la version du projet

Cette action permet de récupérer le paramètre "Version" du projet actif.

Paramètres

Paramètres Descriptions
Variable reçevant la version du projet Doit être égale à la variable (non tableau) qui sera égale à la version du projet.

Valeur retournée

La valeur retournée est déterminée par le paramètre "Version" de votre projet. Cette valeur est de type texte (String).

Exemple : 1.0.0.0

Code Visual Basic.Net généré par SoftwareZator

Votre_Variable = My.Application.Info.Version.ToString()
