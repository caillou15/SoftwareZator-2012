Action : Obtenir l'adresse IP

Cette action permet de récupérer une valeur indiquant l'adresse IP de l'ordinateur sur le réseau internet.

Paramètres

Paramètres Descriptions
Variable recevant le résultat Définit la variable recevant la valeur qui indique l'adresse.

Valeur retournée

La valeur retournée est déterminée par le status de la connexion internet de l'ordinateur. Elle sera enregistrée dans la variable définit dans le paramètre "Variable recevant le résultat". Cette valeur est de type texte (String).

Code Visual Basic.Net généré par SoftwareZator

Votre_Variable = VelerSoftware_GeneralPlugin.GetIp
