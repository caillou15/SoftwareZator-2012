Action : Obtenir la version de Windows

Cette action permet de récupérer une valeur numérique égale à la version complète du système d'exploitation Windows sur lequel est exécuté votre programme.

Paramètres

Paramètres Descriptions
Variable recevant le résultat Définit la variable recevant la version de Windows.

Valeur retournée

La valeur retournée est déterminée par la version de Windows. Elle sera enregistrée dans la variable définit dans le paramètre "Variable recevant le résultat". Cette valeur est de type texte (String).
