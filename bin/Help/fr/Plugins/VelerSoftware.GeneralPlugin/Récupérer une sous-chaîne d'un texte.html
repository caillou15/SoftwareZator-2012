Action : Récupérer une sous-chaîne d'un texte

Cette action permet de récupérer une partie définie d'un texte. Vous pourrez ainsi récupérer par exemple un mot ou une suite de mot.

Paramètres

Paramètres Descriptions
Texte concerné Doit être égale au texte dont une partie sera récupérée.
Numéro de caractère auquel il faut commencer à récupérer le texte Doit être égale au numéro du caractère à partir duquel la récupération d'une sous-chaîne sera faite.
Longueur de la chaîne à récupérer Doit être égale au nombre de caractère sur lequel la chaîne doit être récupérée. Vous ne pouvez pas définir un nombre suppérieur au nombre de caractères restant après le numéro indiqué dans le paramètre précédant. Si vous voulez récupérer tous les caractères après celui de départ, laissez ce paramètre vide.
Variable recevant le résultat Doit être égale à la variable (non tableau) recevant la sous-chaîne.

Erreur à ne pas commettre : Pour les paramètres "Texte concerné", vous ne pouvez pas utiliser des ressources du projet de type Image et de type Autre. Seule des ressources de type Texte peuvent être utilisées.

Erreur à ne pas commettre : Si vous utilisez dans variables dans le paramètre "Texte concerné", assurez-vous que le type de la valeur de chacune de ces variables soit du texte (String), ou bien une valeur de type numérique (Integer, Decimal, Double). En effet, seul du texte peut être utilisé dans ce paramètre.
Note : Pour vérifier le type d'une valeur, utilisez l'action Obtenir le type d'une variable, dans la catégorie Variables.

Erreur à ne pas commettre : Si vous utilisez dans variables dans les paramètres "Numéro de caractère auquel il faut commencer à récupérer le texte" et ou "Longueur de la chaîne à récupérer", assurez-vous que le type de la valeur de chacune de ces variables soit un nombre entier (Integer). En effet, seul un nombre peut être utilisé dans ces paramètres.
Note : Pour vérifier le type d'une valeur, utilisez l'action Obtenir le type d'une variable, dans la catégorie Variables.

Valeur retournée

La valeur retournée est déterminée par le caractère de début et la longueur de la chaîne. Elle sera enregistrée dans la variable définit dans le paramètre "Variable recevant le résultat". Cette valeur est de type texte (String).

Exemple : Soit le texte "Bonjour". On demande de récupérer une sous-chaîne à partir du caractère numéro 1 sur une longueur de 3 caractères. La valeur retourné sera "onj".

Code Visual Basic.Net généré par SoftwareZator

Si le numéro du caractère de départ est 0 et la longueur est 3, le code généré sera le suivant :

Votre_Variable = CStr("Bonjour").SubString(0, 3)

Si la longueur de la chaîne n'est pas définie, le code généré sera le suivant :

Votre_Variable = CStr("Bonjour").SubString(0)
