Action : Modifier une propriété d'un contrôle

Cette action permet de modifier la valeur d'une propriété d'un contrôle.

Exemple : Si le contrôle est de type Button (System.Windows.Forms.Button), vous pourrez modifier toutes les propriétés issue de ce type, à savoir, Text, BackColor, Size, etc.

Paramètres

Paramètres Descriptions
Contrôle dont il faut modifier une propriété Doit être égale au contrôle dont il faut modifier la valeur d'une propriété spécifiée dans les paramètres suivants.
Propriété à modifier (selon le type) Permet de choisir la propriété de la variable à modifier.
Nouvelle valeur de la propriété Permet de saisir la nouvelle valeur de la propriété spécifié. Vous pouvez saisir une valeur textuel, numérique, ou encore du Visual Basic.Net, une couleur et des types particulier. Veillez à ce que votre valeur soit du même type que celui indiqué par le paramètre "Propriété à modifier".

Valeur retournée

Aucune valeur n'est retourné par cette action

Code Visual Basic.Net généré par SoftwareZator

Votre_Contrôle.Votre_Propriete = Nouvelle_Valeur

Erreurs d'exécutions

Il arrive qu'un programme informatique rencontre un problème durant son exécution. Souvent, un message d'erreur est alors affiché. Le tableau ci-dessous vous permet de déterminer pourquoi l'erreur que vous rencontrez s'affiche et comment la corriger :

Types d'erreurs Descriptions Resolutions
System.InvalidCastException Cette erreur se produit lorsque la valeur du paramètre "Nouvelle valeur du paramètre" est une valeur d'un type différent de celui de la propriété spécifié dans le paramètre "Propriété à modifier" et que celui-ci n'est pas compatible et convertible avec votre valeur. Pour corriger cette erreur, vérifiez que toutes les valeurs impliquées dans ce paramètre sont bien du même type que celui du paramètre "Propriété à modifier" et faites les modifications nécessaires pour que ces valeurs soit compatible. Pour cela, utilisez les actions Obtenir le type d'une variable, Convertir la valeur d'une variable
