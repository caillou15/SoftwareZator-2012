Action : Obtenir les arguments de démarrage

Cette action permet de récupérer les paramètres de lancement du logiciel.

Exemple : Lorsque, dans votre explorateur de fichiers Windows, vous effectuer un clic droit sur un fichier puis "Ouvrir avec", le logiciel lancé aura en paramètre le chemin complet vers le fichier sélectionné dans l'explorateur.
Autre exemple, lorsque vous utilisez la fonction "Exécuter", dans Windows, les paramètres données à votre logiciel peuvent être récupérés grâce à cette action.

Paramètres

Paramètres Descriptions
Variable tableau reçevant les arguments Doit être égale à la variable (tableau) qui sera égale aux arguments de lancement du logiciel.

Valeur retournée

La valeur retournée est un tableau de valeur. Cette valeur est de type texte (String). Elle est égale à l'ensembles des paramètres ayant été utilisés pour lancer le logiciel.

Code Visual Basic.Net généré par SoftwareZator

Votre_Variable_Tableau = New System.Collections.Generic.List(Of System.Object)(System.Environment.GetCommandLineArgs())

Si cette action est utilisée dans la fonction "Startup", dans les évènements d'application, le code généré sera le suivant :

Votre_Variable_Tableau = New System.Collections.Generic.List(Of System.Object)(Ctype(_e, Microsoft.VisualBasic.ApplicationServices.StartupEventArgs).CommandLine)

Si cette action est utilisée dans la fonction "StartupNextInstance", dans les évènements d'application, le code généré sera le suivant :

Votre_Variable_Tableau = New System.Collections.Generic.List(Of System.Object)(Ctype(_e, Microsoft.VisualBasic.ApplicationServices.StartupNextInstanceEventArgs).CommandLine)
