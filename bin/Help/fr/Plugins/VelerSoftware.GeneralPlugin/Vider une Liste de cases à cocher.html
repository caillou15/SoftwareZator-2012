Action : Vider une Liste de cases à cocher

Cette action permet de supprimer tous les éléments d'une Liste de cases à cocher spécifiée.

Attention : Assurez-vous que la fenêtre dans laquelle réside votre Liste de cases à cocher est ouverte (affiché à l'écran) au moment où cette action intervient.

Paramètres

Paramètres Descriptions
Liste de cases à cocher, à vider Doit être égale à la Liste de cases à cocher dont tous les éléments seront supprimés.

Valeur retournée

Aucune valeur n'est retourné par cette action.

Code Visual Basic.Net généré par SoftwareZator

Si la Liste de cases à cocher se trouve dans la même fenêtre que celle d'où vous utilisez cette action, le code généré sera le suivant :

Me.Votre_Liste.Items.Clear()

Si la Liste de cases à cocher se trouve dans une autre fenêtre que celle d'où vous utilisez cette action, le code généré sera le suivant :

Variables.Votre_Fenetre.Votre_Liste.Items.Clear()

Erreurs d'exécutions

Aucune erreur d'exécution n'est détecté ici.
