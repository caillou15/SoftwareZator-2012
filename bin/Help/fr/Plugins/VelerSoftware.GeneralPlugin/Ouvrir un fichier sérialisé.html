Action : Ouvrir un fichier sérialisé

Cette action permet d'ouvrir un fichier précédemment enregistré avec l'action Enregistrer une donnée sérialisable.

Qu'est-ce que la sérialisation : C'est un procéder qui permet d'enregistrer la valeur complète d'une variable, sous une forme crypté, dans un fichier, et de pouvoir la réouvrir tel qu'elle était lors de son enregistrement, après avoir quitté et relancé votre logiciel par exemple.

Information : Pour vérifier que le fichier existe, utilisez l'action Déterminer si un fichier existe.

Paramètres

Paramètres Descriptions
Nom complet du fichier à ouvrir Doit être égale au chemin complet du fichier à charger.
Variable recevant la donnée désérialisé Doit être égale à la variable (non tableau) recevant la donnée du fichier qui avait été sérialisé.

Erreur à ne pas commettre : Pour le paramètre "Nom complet du fichier à créer", vous ne pouvez pas utiliser des ressources du projet de type Image et de type Autre. Seule des ressources de type Texte peuvent être utilisées.

Erreur à ne pas commettre : Si vous utilisez dans variables dans le paramètre "Nom complet du dossier à créer", assurez-vous que le type de la valeur de chacune de ces variables soit du texte (String), ou bien une valeur de type numérique (Integer, Decimal, Double). En effet, seul du texte peut être utilisé dans ce paramètre.
Note : Pour vérifier le type d'une valeur, utilisez l'action Obtenir le type d'une variable, dans la catégorie Variables.

Astuce : Une fois que votre fichier a été chargé, vous ne savez pas quel était le type de celui-ci. Pour le déterminer, utilisez l'action Obtenir le type d'une variable, dans la catégorie Variables.

Valeur retournée

La valeur retournée est déterminée par la réussite du chargement ou non, et du contenu du fichier. Elle sera enregistrée dans la variable définit dans le paramètre "Variable recevant la donnée désérialisé". Cette valeur est de type Objet (System.Object). Si le chargement du fichier a échoué, la valeur retourné sera nul.
