Action : Obtenir l'index d'un élément d'une Liste d'éléments

Cette action permet de récupérer l'index d'un élément (dont on connait le nom) existant dans une Liste d'éléments.

Attention : Assurez-vous que la fenêtre dans laquelle réside votre Liste d'éléments est ouverte (affiché à l'écran) au moment où cette action intervient.

Paramètres

Paramètres Descriptions
Liste d'éléments concerné Doit être égale à la Liste d'éléments dont il faut récupérer l'index d'un élément.
Nom de l'élément Ce paramètre doit être égale au nom de l'élément dont vous voulez récupérer l'index dans la Liste d'éléments.
Variable recevant le résultat Doit être égale à la variable (non tableau) qui sera égale à l'index de l'élément dans la Liste d'éléments.

Valeur retournée

La valeur retournée est déterminée par la position de l'élément spécifié dans la Liste d'éléments. Cette valeur est enregistré dans la variable définie dans le paramètre "Variable recevant le résultat". Celle-ci est un nombre entier (Integer)

Important : Si la valeur retourné est -1, cela signifie que l'élément spécifié n'existe pas dans la Liste d'éléments.

Code Visual Basic.Net généré par SoftwareZator

Votre_Variable = Votre_Liste.Items.IndexOf("Nom de l'élément")
