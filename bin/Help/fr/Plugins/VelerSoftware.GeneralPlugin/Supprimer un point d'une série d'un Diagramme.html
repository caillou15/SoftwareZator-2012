Action : Supprimer un point d'une série d'un Diagramme

Cette action permet de supprimer un point dans une série affiché dans le Diagramme spécifié.

Attention : Assurez-vous que la fenêtre dans laquelle réside votre Diagramme est ouverte (affiché à l'écran) au moment où cette action intervient.

Paramètres

Paramètres Descriptions
Diagramme concerné Doit être égale au Diagramme dans lequel le point sera supprimé.
Nom de la série concerné Saisissez dans ce paramètre le nom de la série dans laquelle le point sera supprimé.
Index du point à supprimer Doit être égale à l'index du point sur l'axe des abscisse, indépendamment de la valeur affiché dans les légendes, qui sera supprimé.
Exemple : si veut supprimer la valeur du point d'abscisse "Janvier", qui est le tout premier point du graphique, l'index sera 0. Si c'est le second point, l'index sera 1 et ainsi de suite.

Erreur à ne pas commettre : Pour le paramètre "Nom de la série concerné", vous ne pouvez pas utiliser des ressources du projet de type Image et de type Autre. Seule des ressources de type Texte peuvent être utilisées.

Erreur à ne pas commettre : Si vous utilisez dans variables dans le paramètre "Nom de la série concerné", assurez-vous que le type de la valeur de chacune de ces variables soit du texte (String), ou bien une valeur de type numérique (Integer, Decimal, Double). En effet, seul du texte peut être utilisé dans ce paramètre.
Note : Pour vérifier le type d'une valeur, utilisez l'action Obtenir le type d'une variable, dans la catégorie Variables.

Valeur retournée

Aucune valeur n'est retourné par cette action

Code Visual Basic.Net généré par SoftwareZator

Si le Diagramme se trouve dans la même fenêtre que celle d'où vous utilisez cette action, que la série est "Ma Serie", le code généré sera le suivant :

Me.Votre_Diagramme.Series("Ma Serie").Points().RemoveAt(Index)

Si le Diagramme se trouve dans une autre fenêtre que celle d'où vous utilisez cette action, que la série est "Ma Serie", le code généré sera le suivant :

        
Variables.Votre_Fenetre.Votre_Diagramme.Series("Ma Serie").Points().RemoveAt(Index)
