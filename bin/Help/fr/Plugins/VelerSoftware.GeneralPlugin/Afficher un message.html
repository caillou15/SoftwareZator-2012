Action : Afficher un message

Cette action va vous permettre d'afficher un message à l'écran grâce à la très célèbre boîte de message de Windows.

Paramètres

Paramètres Descriptions
Titre Définit le titre du message à afficher.
Boutons Permet de choisir les boutons à afficher dans le message.
Icône Permet de choisir l'icône devant être affiché à gauche de la boîte de message.
Texte du message Doit être égale au texte que vous voulez faire apparaitre en temps que message.
Variable recevant le résultat (facultatif) Doit être égale à la variable (non tableau) recevant le résultat du message affiché.

Erreur à ne pas commettre : Pour les paramètres "Titre" et "Texte du message", vous ne pouvez pas utiliser des ressources du projet de type Image et de type Autre. Seule des ressources de type Texte peuvent être utilisées.

Erreur à ne pas commettre : Si vous utilisez dans variables dans les paramètres "Titre" et "Texte du message", assurez-vous que le type de la valeur de chacune de ces variables soit du texte (String), ou bien une valeur de type numérique (Integer, Decimal, Double). En effet, seul du texte peut être utilisé dans ces paramètres.
Note : Pour vérifier le type d'une valeur, utilisez l'action Obtenir le type d'une variable, dans la catégorie Variables.

Valeur retournée

La valeur retournée est déterminée par le bouton sélectionné par l'utilisateur. Elle sera enregistrée dans la variable définit dans le paramètre "Variable recevant le résultat". Cette valeur est de type texte (String).

Boutons sélectionnés Valeurs retournées
OK OK
Annuler Cancel
Oui Yes
Non No
Réessayer Retry
Abandonner Abort
Ignorer Ignore

Erreurs d'exécutions

Il arrive qu'un programme informatique rencontre un problème durant son exécution. Souvent, un message d'erreur est alors affiché. Le tableau ci-dessous vous permet de déterminer pourquoi l'erreur que vous rencontrez s'affiche et comment la corriger :

Types d'erreurs Descriptions Resolutions
System.ArgumentException
ou
System.InvalidCastException		 Cette erreur se produit lorsque la valeur du paramètre "Titre" et ou "Texte du message" ne sont pas du texte. Souvent, le problème est causé par une variable utilisé dans ces paramètres et qui ont une valeur autre que du texte. Pour corriger cette erreur, vérifiez que toutes les valeurs impliquées dans ces paramètres sont bien de type Texte (String) et faites les modifications nécessaires pour que ces valeurs soit du texte. Pour cela, utilisez les actions Obtenir le type d'une variable, Convertir la valeur d'une variable
