Action : Démarrer un Minuteur

Cette action va vous permettre de démarrer le fonctionnement d'un Minuteur précédemment ajouté au Concepteur de fenêtre. Une fois le Minuteur démarré, il exécutera la fonction lié à l'évènement Tick de celui-ci toute les X millisecondes. Ce laps de temps dépend de la valeur de la propriété "Interval" du Minuteur.

Attention : Assurez-vous que la fenêtre dans laquelle réside votre Minuteur est ouverte (affiché à l'écran) au moment où cette action intervient.

Paramètres

Paramètres Descriptions
Minuteur à démarrer Doit être égale au Minuteur que vous voulez démarrer.

Valeur retournée

Aucune valeur n'est retourné par cette action

Code Visual Basic.Net généré par SoftwareZator

Si le Minuteur se trouve dans la même fenêtre que celle d'où vous utilisez cette action, le code généré sera le suivant :

Me.Votre_Minuteur.Start()

Si le Minuteur se trouve dans une autre fenêtre que celle d'où vous utilisez cette action, le code généré sera le suivant :

Variables.Votre_Fenetre.Votre_Minuteur.Start()

Erreurs d'exécutions

Aucune erreur d'exécution n'est détecté ici.
