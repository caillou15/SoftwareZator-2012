Action : Imprimer

Cette action permet d'imprimer un flux d'impression précédemment créé avec l'action Nouvelle impression.

Paramètres

Paramètres Descriptions
Variable possédant le flux d'impression à imprimer Doit être égale à une variable (non tableau) ayant pour valeur un flux d'impression (ce flux a été créé et obtenu par l'action Nouvelle impression).
Afficher la fenêtre des paramètres d'impression Permet de déterminer si une boîte de dialogue permettant de configurer une imprimante et les paramètres d'impression doit être affichée.
Faire un aperçu avant impression Permet de déterminer si une boîte de dialogue permettant d'afficher un aperçu du document avant l'impression doit être ouvert.
Variable recevant le résultat de l'impression (facultatif) Au besoin, permet de sélectionner la variable (non tableau) recevant le résultat de l'impression.

Valeur retournée

La valeur retournée est déterminée par la réussite ou non de l'envoi du document à l'imprimante. Cette valeur ne permet pas de savoir si l'impression a été effectuée correctement, mais uniquement de savoir s'il n'y a pas eu de problème avant le début de la procédure d'impression (qui n'est pas géré par votre logiciel mais par l'ordinateur). Elle sera enregistrée dans la variable définit dans le paramètre "Variable recevant le résultat de l'impression". Cette valeur est de type Vrai/Faux (Boolean).

Résultat Valeurs retournées
Aucun problème n'a été rencontré True
Un problème inconnu a été rencontré False

Code Visual Basic.Net généré par SoftwareZator

Si le paramètre "Afficher la fenêtre des paramètres d'impression" est coché et que le paramètre "Faire un aperçu avant impression" est décoché, le code généré sera le suivant :

Votre_Variable = Votre_Flux.Print(True, False)

Si le paramètre "Afficher la fenêtre des paramètres d'impression" est décoché et que le paramètre "Faire un aperçu avant impression" est coché, le code généré sera le suivant :

Votre_Variable = Votre_Flux.Print(False, True)

Erreurs d'exécutions

Il arrive qu'un programme informatique rencontre un problème durant son exécution. Souvent, un message d'erreur est alors affiché. Le tableau ci-dessous vous permet de déterminer pourquoi l'erreur que vous rencontrez s'affiche et comment la corriger :

Types d'erreurs Descriptions Resolutions
System.MissingMemberException Cette erreur se produit lorsque l'action Nouvelle impression n'a pas été exécuté, pour la variable définie dans le paramètre "Variable possédant le flux d'impression à imprimer", avant cette action, ou bien que la valeur de la variable a changé entre temps. Pour corriger cette erreur, veillez à ce que l'action Nouvelle impression soit exécuté pour la même variable que celle spécifiée dans le paramètre "Variable possédant le flux d'impression à imprimer" et que la valeur de celle-ci n'a pas été changé ou réinitialisé avant l'exécution de l'action "Imprimer".
