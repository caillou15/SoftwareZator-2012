Action : Nouvelle impression

Cette action permet d'initialiser une nouvelle impression de document, c'est-à-dire, créer un nouveau document imprimable. Il faudra ensuite utiliser les autres actions de la catégorie "Impression" afin de configurer le document puis l'imprimer.

Paramètres

Paramètres Descriptions
Variable utilisée pour mémoriser et travailler avec le flux d'impression Doit être égale à une variable (non tableau) qui sera utilisée par les autres actions de la catégorie "Impression" afin de configurer le document imprimable (marge, en-tête, contenu...).
Nom du document (facultatif) Permet de saisir le nom du document à imprimer.
Orientation Ce paramètre permet de définir l'orientation d'impression du document : Portrait ou Paysage. Le mode Automatique détermine automatiquement, d'après le contenu du document, s'il doit être imprimé en portrait ou en paysage.

Erreur à ne pas commettre : Pour le paramètre "Nom du document", vous ne pouvez pas utiliser des ressources du projet de type Image et de type Autre. Seule des ressources de type Texte peuvent être utilisées.

Erreur à ne pas commettre : Si vous utilisez dans variables dans le paramètre "Nom du document", assurez-vous que le type de la valeur de chacune de ces variables soit du texte (String), ou bien une valeur de type numérique (Integer, Decimal, Double). En effet, seul du texte peut être utilisé dans ces paramètres.
Note : Pour vérifier le type d'une valeur, utilisez l'action Obtenir le type d'une variable, dans la catégorie Général.

Valeur retournée

Aucune valeur n'est retourné par cette action

Code Visual Basic.Net généré par SoftwareZator

Si le paramètre "Orientation" est égale à "Paysage", le code généré sera le suivant :

Votre_Variable = New VelerSoftware.PrintLib.PrintSystem.PrintClass(VelerSoftware.PrintLib.PrintSystem.Orientation.Landscape, "Nom du document", True)

Erreurs d'exécutions

Il arrive qu'un programme informatique rencontre un problème durant son exécution. Souvent, un message d'erreur est alors affiché. Le tableau ci-dessous vous permet de déterminer pourquoi l'erreur que vous rencontrez s'affiche et comment la corriger :

Types d'erreurs Descriptions Resolutions
System.ArgumentException
ou
System.InvalidCastException		 Cette erreur se produit lorsque la valeur du paramètre "Nom du document" n'est pas du texte. Souvent, le problème est causé par une variable utilisé dans ce paramètre et qui ont une valeur autre que du texte. Pour corriger cette erreur, vérifiez que la valeur impliquée dans ce paramètre est bien de type Texte (String) et faites les modifications nécessaires pour que cette valeur soit du texte. Pour cela, utilisez les actions Obtenir le type d'une variable, Convertir la valeur d'une variable
