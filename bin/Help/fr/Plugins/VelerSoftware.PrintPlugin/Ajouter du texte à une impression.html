Action : Ajouter du texte à une impression

Cette action permet de mettre du texte dans une page d'un flux d'impression précédemment créé avec l'action Nouvelle impression.

Paramètres

Paramètres Descriptions
Variable possédant le flux d'impression à imprimer Doit être égale à une variable (non tableau) ayant pour valeur un flux d'impression (ce flux a été créé et obtenu par l'action Nouvelle impression).
Texte à ajouter Permet de saisir le texte à ajouter à la feuille d'impression.
Couleur du texte Permet de définir la couleur que prendra le texte. Cette couleur peut être stocké dans une variable ou définit dans le paramètre. Assurez-vous que la valeur que vous définirez soit une Couleur (System.Drawing.Color)
Numéro de la page à laquelle il faut ajouter le texte Doit être égale au numéro de la page du flux à laquelle le texte sera ajoutée.
Position X dans la page Doit être égale à la position horizontale du texte dans la page.
Position Y dans la page Doit être égale à la position verticale du texte dans la page.

Erreur à ne pas commettre : Pour les paramètres "Texte à ajouter", "Numéro de la page à laquelle il faut ajouter le texte", "Position X dans la page" et "Position Y dans la page", vous ne pouvez pas utiliser des ressources du projet de type Image, Texte et de type Autre.

Erreur à ne pas commettre : Si vous utilisez dans variables dans les paramètres "Texte à ajouter", "Numéro de la page à laquelle il faut ajouter le texte", "Position X dans la page" et ou "Position Y dans la page", assurez-vous que le type de la valeur de chacune de ces variables soit de type numérique (Integer, Decimal, Double). En effet, seul une valeur numérique peut être utilisée dans ces paramètres.
Note : Pour vérifier le type d'une valeur, utilisez l'action Obtenir le type d'une variable, dans la catégorie Général.

Valeur retournée

Aucune valeur n'est retourné par cette action

Code Visual Basic.Net généré par SoftwareZator

Si le paramètre "Variable possédant le flux d'impression à imprimer" est égale à "Votre_Flux", que le paramètre "Texte à ajouter" est égale à "Bonjour le monde", que la couleur du texte est noire, que le numéro de la page est 1, que la "Position X dans la page" est égale à 50, et que la "Position Y dans la page" est égale à 100, le code généré sera le suivant :

Votre_Flux.AddText(New System.Drawing.Point(50, 100), "Bonjour le monde", System.Drawing.Color.FromArgb(255, 0, 0, 0), 1)

Erreurs d'exécutions

Il arrive qu'un programme informatique rencontre un problème durant son exécution. Souvent, un message d'erreur est alors affiché. Le tableau ci-dessous vous permet de déterminer pourquoi l'erreur que vous rencontrez s'affiche et comment la corriger :

Types d'erreurs Descriptions Resolutions
System.ArgumentOutOfRangeException Cette erreur se produit lorsque la valeur du paramètre "Numéro de la page à laquelle il faut ajouter le texte" est inférieur à 1. Le problème est donc que vous tentez d'insérer une image sur une page dont l'index n'existe pas. Pour corriger cette erreur, vérifiez que le numéro de la page soit suppérieur ou égale à 1.
