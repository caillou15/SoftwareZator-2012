Action : Aperçu avant impression d'une zone de texte enrichi

Cette action va vous permettre de faire un aperçu du contenu d'une Zone de texte enrichi dans une feuille de papier avant d'être imprimé.

Attention : Assurez-vous que la fenêtre dans laquelle réside votre éditeur de texte est ouverte (affiché à l'écran) au moment où cette action intervient.

Paramètres

Paramètres Descriptions
Zone de texte concerné Doit être égale à l'éditeur de texte (Zone de texte enrichi uniquement) dont le contenu sera prévisualisé.

Valeur retournée

Aucune valeur n'est retourné par cette action

Code Visual Basic.Net généré par SoftwareZator

Using PrintTextAera_abhdx As New VelerSoftware.PrintLib.PrintSystem.PrintRichTextBox(Votre_Editeur)
PrintTextAera_abhdx.PrintPreview()
End Using
