Méthodes

Nom Description Type
Main() Fonction exécuté lorsque l'action est ajouté à l'éditeur de fonctions. Retourne une valeur Boolean indiquant si l'action sera ajouté à l'éditeur de fonctions (True) ou s'il faut annuler l'ajout (False). System.Boolean
Modify() Fonction exécuté lorsque l'utilisateur clique sur le bouton "Modifier les paramètres de l'action". Retourne une valeur Boolean indiquant si les modifications apportés seront enregistrés (True) ou s'il faut annuler les modifications (False). Boolean
GetVBCode(System.Boolean) Fonction exécuté lorsque SoftwareZator génère les codes Visual Basic.Net avant une génération du projet. Retourne une valeur CodeDom.CodeObject représentant le code VB.Net généré et prêt à être compilé. System.CodeDom.CodeObject
ResolveError(VelerSoftware.SZVB.Projet.Erreur, System.EventArgs) Fonction exécuté lorsque l'utilisateur veut corriger automatiquement une erreur de génération ou d'exécution. Retourne une valeur Boolean indiquant si la correction a été faite (True) ou n'a pas pu être faite (False). System.Boolean
